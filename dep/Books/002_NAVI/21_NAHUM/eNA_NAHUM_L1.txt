<b>משא נינוה .</b> ת''י מטל כס דלווט לאשקא' ית נינוה : <b>ספר חזון נחום .</b> חזון נקוד קמץ לפי שאינו דבוק ואינו דומה לחזון ישעיה שהוא נקוד חטף פתח וכן פי' ספר חזון כבר נכתב עליה והוא נבואת יונה בן אמיתי ועתה חזר ונתנבא עלי' נחום האלקושי את המשא הזה אלקוש היה שם עירו וכן ת''י מלקדמין אתנבי עלה יונה בר אמתי ותבת מחובהא וכדו דאוסיפת למיחטי תב ואיתנבי עלה נחום דמבית אלקוש : 
! {ב}
<b>אל קנוא ונוקם ה' .</b> על זרעו של סנחריב נתנבא נחום ובימי מנשה כמו שמצינו בסדר עולם אל קנא ונוקם ה' מאז שנפרע לישראל מאויביהם וגם עדיין עתה נוקם הוא ובעל חימה ועתיד הוא להיות נוקם לצריו שהחריבו את ארצו והגלו את עמו ונוטר הוא את איבתו לבבליים ור''ש אומר ג' נקימות הללו כנגד ג' גליות שהגלה סנחריב את ישראל כמו שמצינו בסדר עולם : 
! {ג}
<b>ה' ארך אפים וגדל כח .</b> גדל כח הוא והיכולת בידו להנקם ואשר לא מיהר נקמתו לפי שארך אפים הוא ומכל מקום נקה לא ינקה . <b>ה' בסופה ובשערה דרכו</b> . הם שלוחיו להפרע מאויביו שנא' במצרים ( שמות יד ) ברוח קדים עזה ובדור המבול ( איוב ד ) מנשמת אלוה יאבדו בצור נאמר ( יחזקאל כו ) רוח קדים שברך : <b>וענן אבק רגליו .</b> גם הוא דרך נקמתו ( שמות יד ) וישקף אל מחנה מצרים בעמוד אש וענן ( יחזקאל ל ) היא ענן יכסנה : 
! {ד}
<b>גוער בים .</b> כמשמעו ודוגמא הוא כלפי העכו''ם שנמשלו למים כענין שנאמר ( ישעיה יז ) לאומים כשאון מים כבירים ישאון : <b>וכל הנהרות החריב .</b> כאן ניבא שעתיד הקב''ה להמליך את נבוכדנצר ( ירמיה כה ) בימי יהויקים ויתן את אשור ואת כל הארצות לחרב : <b>אומלל בשן וכרמל .</b> מושבות הטובים : <b>ופרח לבנון אומלל .</b> חכמי ישראל פירשו כלפי מיני מגדים שנטע שלמה בבהמ''ק שהיו של זהב וכיון שנכנסו גוים להיכל יבשו : 
! {ה}
<b>הרים רעשו ממנו .</b> המלכים והשרים מהכשדים יפחדו מגזירתו : <b>ותשא הארץ .</b> משאת עשן : 
! {ו}
<b>נתכה .</b> הגיע לארץ כמו ( שמות ט ) ומטר לא נתך ארצה : <b>נתצו ממנו .</b> מאימתו : 
! {ז}
<b>טוב ה' למעוז .</b> ואף בשעה שהוא נפרע מאויביו לא זזו רחמיו מלהטיב ליראיו ולא כמדת בשר ודם שכשהוא עסוק בזו אינו פנוי בזו כענין שנא' ( שם טו ) ה' איש מלחמ' ה' שמו כמו שהוא במכילתין : <b>ויודע .</b> את צרכי חוסי בו : 
! {ח}
<b>ובשטף עובר .</b> ובחמה שוטפת : <b>כלה יעשה מקומה .</b> של ארץ ומוסב זה על ותשא הארץ מפניו כמו שאמור למעלה : 
! {ט}
<b>כלה הוא עושה .</b> לנינוה ומלכי אשור לא ישנה להם עוד פעם אחרת : 
! {י}
<b>כי עד סירים סבוכים .</b> כלומר פתאום תבא עליהם מכתם ולא ירדו מכבודם מעט מעט כי בעוד שהסירים והקוצי' נסבכים נשרשים בחזקה : <b>וכסבאם סבואים .</b> ועודם במשתיהן ובשמחתן ויש פותרין כי עד סירים לשון הסרה : <b>אכלו כקש יבש .</b> אשר נתמלא ונגמר בשולו ויבש : <b>מלא .</b> אשובי''ר בלע''ז וכן מלאתך , המלאה הזרע : 
! {יא}
<b>ממך .</b> את נינוה : <b>יצא .</b> סנחריב החושב רעה דימה להחריב דירת מטה ומעלה כתוב אחד אומר ( מלכים ב יט ) ואבא מלון קיצו וכתוב אחד אומר ( ישעיה לז ) ואבא מרום קיצו בתחילה אחריב דירה של מטה ולבסוף דירה של מעלה כדאי' בחלק ( דף צד ) : 
! {יב}
<b>אם שלמים וכן רבים .</b> יפה תרגום של יונתן מיושב על המקרא אם יהון שלמים בעצה שיהו כולם שוים בהסכמה א' וכן אם יהיו רבי' מאד אנשי נינוה אויביך ירושלים וכן אף אם גזו חדקל ועברו את פרת לצור עליה לא יצליחו שהרי עניתיך ולא אענך עוד , ועוד שמעתי אם שלמים אנשי נינוה בגדולתם וכן אם יהו עוד רבים ונכבדים יותר וכן נגוזו ועבר וכן אגיזם ואעבירם הרי זה כמו ( שמו' א ) וכאשר יענו אותו כן ירבה : <b>ועניתיך לא אענך עוד .</b> אענה אותך בפעם הזאת כלה ונחרצה ולא אזקוק לך עוד ורבותינו דרשו לענין מעשה הצדקה במסכת גיטין אך אינו על סדר שאר הנבואה : 
! {יג}
<b>ועתה אשבור מטהו מעליך .</b> שכינה אומרת כן לנביא שהוא ישראלי אשבור מטהו של אשור מעליך ומעמי : 
! {יד}
<b>וצוה עליך ה' .</b> עתה מוסב הדיבור אל מלך אשור : <b>לא יזרע משמך עוד .</b> לא יעמוד ממך מלך : <b>מבית אלהיך אכרית פסל ומסכה .</b> ואשים את בית אלהיך קברך שם תברח להמלט ושם תהרג כמו שכתוב שנהרג אביך בית נסרוך אלהיו : <b>כי קלות .</b> בעיני : 

~ נחום פרק-ב

! {א}
<b>שלמי נדריך .</b> אשר נדרת להקב''ה אם יצילך מכף סנחריב מלך אשור כי עתה נצלת : <b>כי לא יוסיף עוד לעבר בך .</b> אותו הבליעל : <b>כלה נכרת .</b> הוא וזרעו : 
! {ב}
<b>עלה מפיץ על פניך נצור מצורה .</b> המפיץ אשר עלה על פניך אל ארץ יהודה בימי יחזקיהו עכשיו הוא נצור מצורה נבוכדנצר מלך בבל שהוא צר עליו : <b>צפה דרך .</b> הבט אתה איש יהודה על אם הדרכים ותראה את ההולכים לצור עליו ויתחזקו מתניך ואמץ כח מאד : 
! {ג}
<b>כגאון ישראל .</b> כמו שהיה כבר : <b>בקקום .</b> בזזום והריקום לשון בקיקה נופל על גפן : 
! {ד}
<b>מגן גבוריהו .</b> של נבוכדנצר הצרים על אשור : <b>מאדם .</b> מאודמים הם ואנשי חילם : <b>מתולעים .</b> לבושי זהורית : <b>באש פלדות הרכב ביום הכינו .</b> ביום שהוא מוכן לצאת בצבא מצחצח רכב ברזל שלו ולא ידעתי מהו פלדות ואומר אני שהוא מין המצחצחת ברזל יפה ויש פותרין פלדות לשון לפיד אש בדרך ההפך : <b>והברושים הרעלו .</b> ת''י ורבני משריתהון מסרבלין בצבעונין : <b>הרעלו .</b> מעוטפין ודומה לו ( ישעיה ג ) הנטיפות והשרות והרעלות ובל' משנה שנינו מדיות רעולות מסכת שבת ( דף ס''ה ע''ש ) : 
! {ה}
<b>יתהוללו .</b> ינהגו עצמן בשגעון : <b>ישתקשקון .</b> ת''י קל נקוש זיניהון משתמע והוא מלשון והשיקו היקבים ( יואל ב ) כמשק גבים ( ישעיה לג ) ל' השמעת קול : <b>ירוצצו .</b> ירועעו את השומעים כברקים הללו הטורדים את הבריות כן ת''י : 
! {ו}
<b>יזכור .</b> מלך אשור את אדיריו וגבוריו וידמה בלבו לצאת ולהלחם והם יכשלו בהליכתם : <b>ימהרו חומתה .</b> לנוס לרוץ אל חומת העיר כל תיבה הצריכה למ''ד בתחלתה הטיל לה ה''א בסופה : <b>והוכן הסוכך .</b> ת''י יבנון מגדליא ויש לפרש והוכן הסוכך והמלך נזדיין בכלי זיונו כמו ממשח הסוכך לשון מושל : 
! {ז}
<b>שערי הנהרות .</b> שערי העיר אשר מצד הנהרות : <b>וההיכל נמוג .</b> נרעד ונע מכח אבני בליסטרא הנזרקי' בכתליו הנקרא בומ''א : 
! {ח}
<b>והוצב .</b> היא המלכה הנצבת לימין המלך כד''א ( תהלים מ''ה ) נצבה שגל לימינך : <b>גלתה הועלתה .</b> בגלוי הלכה בשבי והעלתה מן העיר : <b>מנהגות כקול יונים .</b> מנהגות עצמן בקינה כקול יונים המקוננים מנהגות שרימייצנ''ט בלע''ז : <b>מתופפות על לבבהן .</b> ת''י מקישות ידיהן על לבן כהקיש בתופים : 
! {ט}
<b>ונינוה כברכת מים מימי היא .</b> מיום שהיא נעשית ישבה בשלוה ואין מניד כבריכת מים מכונסין שאין זזים ממקומם : <b>והמה .</b> עתה מרוב דוחקם למדו לנוס ונסים : <b>עמדו עמודו .</b> אומרים להם ואין מפנה לבו לשמוע ולעמוד : 
! {י}
<b>בזו כסף .</b> אתם הבאים על נינוה : <b>ואין קצה .</b> אין חשבון : <b>לתכונה .</b> לאוצרותיהם על שם שהכסף והזהב הנתון באוצר נתון בחשבון כמה דאת אמר את הכסף המתוכן ( מ''ב יב ) : <b>כבוד מכל כלי חמדה .</b> לשון הכבדה כמו ( לקמן ג ) התכבד כילק לשון מכבד הבית אשקוונ''ט בלע''ז ולכבד אותם מכל כלי חמדתא וכן ת''י ספו מכל מני חמדתא , כבוד כמו זכור ושמור : 
! {יא}
<b>בוקה .</b> ריקנית : <b>ומבוקה .</b> ומרוקנת על ידי בוקקים : <b>ומבולקה .</b> מפורצת בחומותיה אישברטיר''א בלע''ז : <b>ופיק ברכים .</b> כשלון ברכים כמו ( שמואל א כה ) לפוקה ולמכשול : <b>קבצו פארור .</b> שחרורית כקדירה : 
! {יב}
<b>איה מעון אריות .</b> קינה היא על נינוה שהיתה מדור מלכים גבורים וחזקים כאריות : <b>אשר הלך אריה לביא שם .</b> ת''י אתר דהוו אזלין מלכין ושבקין תמן בניהון הא כאריה דשרי לרוחצן : 
! {יג}
<b>אריה טורף בדי גורותיו .</b> כתרגומו מלכיה היו מביאים שלל רב עד שהיה די לבני ביתם : <b>ומחנק .</b> בהמות לצורך לבאותיו כלומר מרבה אוצרותיו לבניו : <b>וימלא טרף חוריו .</b> ת''י וממלן אוצרותיהן חטוף : 
! {יד}
<b>רכבה .</b> רוב רכב שבה קרטדי''ן בלע''ז ה''א יתירה היפך הלשון לריבוי מרכבות וכן ( יחזקאל כד ) סיר אשר חלאתה בה זוהם רב : <b>ולא ישמע עוד קול מלאככה .</b> קול מלאכיך כמו שעשה כבר רבשקה וחביריו שלוחי אשור ( ישעיה לז ) ויקרא בקול גדול יהודית : 

~ נחום פרק-ג

! {א}
<b>פרק .</b> גזל שמפרקו מיד בעליו : <b>לא ימיש טרף .</b> מתוכה תמיד : 
! {ב}
<b>קול שוט .</b> להכות סוסים היה נשמע בה תמיד : <b>וקול רעש .</b> אופני מרכבות : <b>וסוס דוהר .</b> דולג ומקפץ וכן מדהרות דהרות אביריו ( שופטים ה ) : 
! {ג}
<b>ולהב חרב .</b> רולימל''א בלע''ז כי סכין וחרב קרוי להב כמו ( שם ד ) ויבא הנצב אחר הלהב : <b>וברק חנית .</b> חנית מרוטה ומברקת אור יוצא ממנה פלנדו''ר בלע''ז : <b>וכובד פגר .</b> היו מרבים פגרים ולא היה חשבון לגויות נבילותיה המושלכות בה עד שהיו העוברי' נכשלים בגויות הפגרים : 
! {ד}
<b>מרוב זנוני זונה .</b> מרוב חנופות עיר שהיו יודעי' לפתות לב מלכי הארץ להתחבר עמם וסוף היו כובשי' אותם תחתיה' : 
! {ה}
<b>שוליך .</b> הם שפולי בגדי האשה : <b>מערך .</b> גלויך כמו ( מיכה א ) עריה בשת : 
! {ו}
<b>שקוצים .</b> בגדים צואים המנוולים אותך לפי הענין שדימה אותה לזונה דבר הכתוב : <b>כראי .</b> כזבל לשון והסיר את מוראתו בנוצתה ( ויקרא א ) : 
! {ז}
<b>ידוד ממך .</b> יתרחק ממך : 
! {ח}
<b>התיטבי מנוא אמון .</b> ומה תבטחי בעשרך ובגבורתך וכי גדולה את מנוא היא אלכסנדרי' של מצרים אמון לשון פדגוג מנוא הגדולה שהיתה אומנת מלכי מצרים דמרבי בה מלכו : <b>אשר חיל ים .</b> אשר היה הים לה לחיל ולחומה שורא ובר שורא : 
! {ט}
<b>כוש עצמה .</b> כושים היו לה לחוזק : <b>פוט ולובים היו בעזרתך .</b> והפוטים והלובים אשר היו בעזרתך גם הם היו בחילה : 
! {י}
<b>גם היא לגולה הלכה .</b> אף היא הלכה בגולה אעפ''כ גדולתה לא עמדה לפני נבוכדנצר : <b>רתקו .</b> נכבלו לשון רתוקות כסף ( ישעיה מ ) עשה הרתוק ( יחזקאל ז ) שלשלות : <b>בזיקים .</b> שלשלאות : 
! {יא}
<b>גם את .</b> אף לפי גדולתך תשכרי כמותה בכוס התרעלה ותהיה נעלמה : 
! {יב}
<b>כל מבצריך תאנים עם בכורים .</b> כל ערי מבצריך יהיו נוחים ליכבש ולבוז מה שבתוכה כאילני תאנים וכשהן עם בכורות מבושלת שבהם שכשמניעים האילן נופלים הבכורים לתוך פי האוכלים : <b>תאנים .</b> אילני תאנים : <b>אם ינועו .</b> אשר אם ינועו ביד אדם בחזקה : 
! {יד}
<b>מי מצור שאבי לך .</b> עיר שבאים לצור עליה אוספים להם בני העיר מים רבים לתוכה בתוך חביות לשתות בימי המצור כשאין בתוכה בורות : <b>באי בטיט ורמסי .</b> אותו לעשות ממנו לבינים כדי לחזק בו בדקי החומות : <b>החזיקי מלבן .</b> אחזי בידך המלבן העשוי לדפוס הלבינים ללבון בתוכה : 
! {טו}
<b>התכבד .</b> במטאטי השמיד : <b>כילק .</b> הגומר ומכלה כל ירק השדה ויש פותרין התכבד כילק תתכסי בכובד גייסות כילק והתכבדי לשון עם כבד : 
! {טז}
<b>ילק פשט ויעוף .</b> יהיו קלים לצאת בסחורה כילק שפושט ועוף : 
! {יז}
<b>מנזריך .</b> שריך והמ''ם יסוד ושורש בתיבה כמ''ם של משמרתך ושל משמעתך ושל מנעליך ופעמים שהיא נופלת ממנה כמו נזיר אחיו ( בראשית מט ) , וכן נזיריה שריה ( איכה ד ) : <b>וטפסריך .</b> הממונים שלך : <b>כגוב גובי .</b> כחיילי גובאי גובי , קומלנגשטדי''ץ בלע''ז : <b>החונים בגדרות ביום קרה .</b> כן דרך הארבה והזבוב החסיל והילק בעת הקור נדבקים בכתלים וכשיבא חום הן נודדים משם והולכים להם כך יגלו כל עמך : <b>ונודד .</b> ומטלטל : <b>ולא נודע מקומו אים .</b> איפה הם ואנה הלכו : 
! {יח}
<b>נמו רועיך .</b> ל' תנומה : <b>ישכנו .</b> אין יכולים לקום : <b>נפושו .</b> כמו נפוצו וכל האותיות היוצאת ממוצא אחד מתחלפות זו בזו זי''ן שי''ן סמ''ך צד''י כולן יוצאין מסוף הלשון ושולי השניים : 
! {יט}
<b>אין כהה לשברך .</b> אין מיצר על שברך כי הכל שמחים , כהה ל' דבר עמום : <b>תקעו כף .</b> לשמוח :
