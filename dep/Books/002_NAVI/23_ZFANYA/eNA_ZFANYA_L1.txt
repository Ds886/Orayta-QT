! {ב}
<b>אסוף אסף .</b> לשון כליון כמו ( שופטים יח ) ואספת את נפשך דמיכה ואל''ף חסרה בתיבה שנייה שהרי היה לו לכתוב אאסוף אבל יש כאלה הרבה כמו לא יהל שם ערבי ( ישעיה יג ) שהוא כמו לא יאהל : 
! {ג}
<b>והמכשלות את הרשעים .</b> הם העכו''ם : 
! {ד}
<b>את שאר הבעל .</b> שלא תהא שארית לזכרונו : <b>הכמרים עם הכהנים .</b> פלחיהון עם כומריהם : 
! {ה}
<b>ואת הנשבעים לה' .</b> וחוזרים ונשבעים במלכם בעכו''ם שלהם עושין עכו''ם עיקר לומר אם אין אתה מאמין בשבועת ה' הריני נשבע לך בעכו''ם : 
! {ז}
<b>הקדיש קרואיו .</b> זימן חיילותיו ליום מועד : 
! {ח}
<b>מלבוש נכרי .</b> תכשיטי עכו''ם : 
! {ט}
<b>הדולג על המפתן .</b> ת''י דמהלכין בנימוסי פלשתאי , שלא היו דורכים על מפתן דגון שנא' על כן לא ידרכו כהני דגון וגו' ( שמואל א ה ) : 
! {י}
<b>משער הדגים .</b> ת''י מתרע נוניא ושער היה בירושלים וכך שמו : <b>מן המשנה .</b> משער העופות שהוא שני לו : 
! {יא}
<b>יושבי המכתש .</b> נחלא דקדרון שעמוקה כמכתשת זה ת''י ומדרש אגדה משער הדגים זו עכו הנתונה בחיקו של הים ששם נצודים דגים הרבה : <b>מן המשנה .</b> לוד שהיא משנה לירושלים : <b>מהגבעות .</b> זו צפורי שיושבת בראשי ההרים כצפור : <b>יושבי המכתש .</b> זו טבריא שעמוקה מכל ארץ ישראל : <b>כי נדמה כל עם כנען .</b> ת''י ארי אתר כל עמא דדמי עובדיהון לעובדי עמא דארעא דכנען : <b>כל נטילי כסף .</b> כל עתירי נכסיא טעוני משא כסף וזהב כמו ( משלי כז ) כובד האבן ונטל החול ל' משא : 
! {יב}
<b>אחפש את ירושלים בנרות .</b> אדקדק בעונותיהם : <b>הקופאים .</b> הצפים על שמריהם ויצף הברזל ( מלכים ב ו ) מתרגמינן וקפא פרזלא : <b>האומרים בלבבם לא ייטיב ה' .</b> לעושי רצונו : <b>ולא ירע .</b> לעוברי רצונו : 
! {יד}
<b>מר צורח שם גבור .</b> הגבור יצרח שם בקול מר : 
! {טו}
<b>יום שואה .</b> לשון שממון : <b>ומשואה .</b> שממון יותר : 
! {יז}
<b>והצרותי לאדם .</b> לישראל הקרויים אדם : <b>ולחומם .</b> בשרם ול' ערבי בשר קורין אל לחם : <b>כגללים .</b> רעי : 

~ צפניה פרק-ב

! {א}
<b>התקוששו .</b> התלקטו ואספו יחד כמו לקושש קש ( שמות ה ) : <b>וקושו .</b> מעשיכם השוו מעשיכם לדעת קונכם , ורז''ל דרשו קשוט עצמך ואח''כ קשוט אחרים עשו שניהם לשון היקש אדם המשוה עצמו ומישר דרכו : <b>הגוי לא נכסף .</b> ת''י דלא חמד למיתב לאורייתא : 
! {ב}
<b>בטרם לדת חוק .</b> לפני בא הגזירה : <b>כמוץ עבר .</b> הרי זה מקרא קצר כמוץ העובר מלפני רוח וכעשן העובר מלפני השמש וזהו תרגומו של עבר יום כך ת''י תהוון דמן לכמוצא דנשב ביה רוחא וכתננא דעדי מן קדם יומא , נ''א כטולא דעדו וכו' זהו ל' צל : 
! {ד}
<b>כי עזה עזובה תהיה .</b> כי אם תעשו כן אפקוד על שכיניכם הרעים פלשת ועמון ומואב כמו שהנביא מסדרן והולך : <b>בצהרים יגרשוה .</b> אשדוד שמה ובצהרים אשר ישוד בה קטב יגרשוה ותהי שדודה , ד''א בצהרים יגרשוה מקום מרעה צאן היה וצהרים קשה לצאת בו צאן וכן הוא אומר איכה תרביץ בצהרים ( שיר א ) : 
! {ה}
<b>חבל הים .</b> ספר ימא ול' חבל מדת הקרקע היא שמודדין בחבל כלומר הסמוכים לים : <b>גוי כרתים .</b> ת''י עמא דמחייבין לאישתצאה ומי הן יושבי חבל הים כנען ארץ פלשתים כי פלשתים יושבים על שפת הים האחרון במערב' של א''י בתוך מצרי תחומיה ואף שדורשו יונתן לכליה גוי כרתים מדינה היא בפלשתים ושמה כרתי וכן הוא אומר בצקלג אנחנו פשטנו על נגב הכרתי ועל אשר ליהודה ות''י על דרום כרתי ולמטה כתוב בכל השלל הגדול אשר לקחו מארץ פלשתים ומארץ יהודה : 
! {ו}
<b>נות כרות רועים .</b> נוה עראי שיכרו בה הרועים לחם בבוקר כרות לשון ויכרה להם כירה גדולה ( מלכים ב ו ) : 
! {ז}
<b>והיה חבל לשארית בית יהודה .</b> והיה אותו הגבול גורל לשארית בני יהודה חבל זה ל' גורל הוא וכן ת''י ויהי עדבא לשארא דבית יהודה : 
! {ח}
<b>אשר חרפו את עמי .</b> כשהיו מוליכין ישראל בגולה לצד ארץ כשדים והיו עוברין דרך עמון ומואב ורואים ישראל בוכים וגונחים וצועקין היו מונין אותן ואומרים מה לכם להיות סגופים מה תבכו הלא לבית אביכם אתם הולכים בעבר הנהר ישבו אבותיכם מעולם : <b>ויגדילו על גבולם .</b> ויתגאו על גבולם : 
! {ט}
<b>כי מואב כסדום תהיה .</b> גם אתם לקדמתכ' תשובו לוט אביכם לא מסדום היה : <b>ממשק חרול .</b> השמעת קול כמו משק גבים ( ישעיה לג ) ואף מ''ם הראשונה יסוד היא בתיבה כמו ממשח הסוכך ( יחזקאל כח ) ממשק חרול השמעת קול חרולים הגדילים שם מאין עובר והן נוקשין זו על זו ומשמיעות קול : <b>ומכרה מלח .</b> מקום כריית מלח : 
! {יא}
<b>כי רזה .</b> התיש כחם : <b>וישתחוו לו .</b> כשיראו שנתקלקלו יראתן : 
! {יב}
<b>גם אתם כושים .</b> כשגלו שם ישראל מעבר לנהרי כוש : 
! {יג}
<b>על צפון .</b> אשור בצפון אצל בבל : <b>נינוה .</b> הוא ראש לאשור : 
! {יד}
<b>כל חיתו גוי .</b> כל בהמות הגוים : <b>קאת : וקפוד .</b> עופות : <b>בכפתוריה .</b> פומל''ש שבראש הגג : <b>חורב בסף .</b> החורבן יהיה ניכר בספי הבית : <b>קול ישורר בחלון .</b> קול העופות מצפצפים בחלונות : <b>כי ארזה ערה .</b> כי ארזיה עקר ושרש כמו ערו ערו ( תהלים קלו ) וי''ת וטללה סתרו הוא גג הבית הספון בארז שאפי' בתי אבנים מסככין בנסרים של עץ : 
! {טו}
<b>ואפסי .</b> ואפס זולתי : 

~ צפניה פרק-ג

! {א}
<b>הוי מוראה ונגאלה .</b> עד עכשיו דבר בנינוה ועכשיו חוזר על ירושלים הוי שתהא לבז ולסחי כרואי ותהא מגואלת בצחן עוונה , מוראה כמו מוראתו ( ויקרא א ) ושמתיך כרואי ( נחום ג ) : <b>העיר היונה .</b> כיונה פותה אין לב : 
! {ג}
<b>זאבי ערב .</b> הממהרים לאכול טרפם בלילה : <b>לא גרמו לבוקר .</b> לא השאירו אף העצמות לגרם לבוקר : 
! {ד}
<b>חמסו תורה .</b> לא הורו אותה לשואליהם :

! {ו}
<b>הכרתי גוים .</b> הבאתי פורענו' על העכו''ם כדי שתראו ותיראו כמו שהוא מסיים והולך אמרתי אך תראי אותי : 
! {ז}
<b>ולא יכרת מעונה .</b> ולא יחרבון מדוריה : <b>כל אשר פקדתי עליה .</b> ולא יכרת ממנו כל הטוב אשר צויתי להביא עליה : 
! {ח}
<b>קומי לעד .</b> שאקום להתוועד עליכם : 
! {י}
<b>מעבר לנהרי כוש .</b> יובילון לי מנחה : <b>עתרי .</b> את המתפללין אלי : <b>בת פוצי .</b> כנסיות נפוצותי אשר הפיצותי : 
! {יא}
<b>לא תבושי מכל עלילותיך .</b> כי כבר לקית ונרצה עוונך : 
! {יב}
<b>עם עני ודל .</b> עם ענוותן ומקבל עולבן : 
! {טו}
<b>משפטיך .</b> יסוריך יושטיצ''א בלע''ז :

<b>גבור יושיע .</b> גבור הוא אשר יושיע מן העכו''ם : <b>יחריש באהבתו .</b> יכסה על פשעיך באהבתו וכן ת''י יכבוש על חובך ברחמתיה :

! {יט}
<b>הנני עושה .</b> ת''י הא אנא עביד גמירא ויש לפותרו ל' עישוי כמו ועסות' רשעים ( מלאכי ג ) : <b>בכל הארץ בשתם .</b> בכל מקום אשר היו שם לבשתם שם אתנם לשם ולתהלה :
