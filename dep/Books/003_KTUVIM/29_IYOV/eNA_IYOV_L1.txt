<b>כעוללים .</b> תינוקות ולכך קורא אותם עוללים שכולם שחוק וליכלוך כמו ( לקמן טז ) ועוללתי בעפר וכמו ויתעללו בה כל הלילה ( שופטים יט ) : 
! {יז}
<b>חדלו רוגז .</b> נמנעים מרוגזם שהיו נוהגים ורגילים להרגיז בארץ : <b>ושם ינוחו .</b> ושם נחו דבר ההווה כתבו בלשון עתיד : 
! {יט}
<b>קטן וגדול שם .</b> שם ניכר מי חשוב ומי שפל ולפי שאותה גדולה גדולת עולם הוא ואותו שפלות שפלות עולם היא : 
! {כ}
<b>למה יתן .</b> ( הנותן והוא הקב''ה ) : <b>לעמל אור .</b> למי שגדל בעמל וצרה למה נותן לו אור עולם ולא המיתו בהוולדו :

<b>המחכים למות .</b> למרי נפש התאבים ומתאוים למות וקובלים לאמר איננו : <b>ויחפרוהו .</b> מרגלים אחריו : <b>ממטמונים .</b> יותר משמרגלי' אחר מטמוני אוצרות והיו : 
! {כב}
<b>שמחים אלי גיל .</b> וישישו אם היו מוצאים קבר : <b>אלי גיל .</b> בשביל הגילה והששון של ( מציאות ) קבר :

<b>לגבר אשר דרכו נסתרה .</b> כל טובות אשר פעל נסתרו מאת בעל הגמול ולא הביט בהם : <b>ויסך אלוה בעדו .</b> עשה מחיצה לפניו לסיגרו במסגר כמו ( הושע ב ) הנני שך את דרכך :

<b>תבא .</b> באה : <b>ויתכו .</b> לשון נתך ארצה נגרים ויורדים כמים ויש לפרש נתיכה ל' רבייה : 
! {כה}
<b>פחד פחדתי .</b> כל ימי עמדתי בפחד על זה כמו שאמרנו ( לעיל א ) ויהי כי הקיפו וגו' : <b>ואשר יגורתי .</b> הן עתד הועתד ונגזר שיבא לי : 
! {כו}
<b>לא שלותי .</b> מדאגה זו והנה ויבא רוגז :

<b>מענה אליפז .</b> ( אליפז בן עשו היה ועל ידי שנתגדל בחיקו של יצחק זכה שתשרה עליו שכינה : <b>התימני .</b> מארץ תימן מארצו של עשו ( סא''א ) : 
! {ב}
<b>הנסה .</b> לשון תימה הבעבור שניסה דבר אליך תלאה בנסיון אחד שניסך בוראך ובא אליך אתה נלאה , ומעתה עצור במילין מי יוכל מלהשיבך : 
! {ג}
<b>הנה יסרת רבים .</b> שהרבו דברים הוכחתם ויסרתם : <b>וידים רפות .</b> ויגורות מפורענות הבא עליה' אתה היית רגיל לחזקם ולומר אל תירא שכך הוא מדת הדין : 
! {ד}
<b>יקימון .</b> רגילות הם מליך להקים את הכושל : 
! {ו}
<b>הלא .</b> עתה סופך מוכיח על תחלתך שיראתך שהיית ירא שמים : <b>כסלתך .</b> היא ( ע' תוס' ב''מ נח ) , מחמת כסילות היא ולא מדעת שלימה וכן תקותך ותום דרכיך הכל כסילות : 
! {ז}
<b>נכחדו .</b> נעלמו ונאבדו כמו ותכחד מן הארץ ( שמות ט ) : 
! {ח}
<b>כאשר ראיתי .</b> אשר ( כן ) נכחדו חורשי און ומכינים את האון במחשבתם כחורש זה שמכין לפני הזריעה ואחרי כן זורעים במעשיהם את העמל וראיתי , שהם יקצרוהו כלומר רעתם שבאה עליהם , וראיתי כי : 
! {ט}
<b>מנשמת אלוה יאבדו .</b> מנשיבת סערתו וכעסו : <b>ומרוח אפו יכלו .</b> כלים תמיד ועל דור המבול דיבר : 
! {י}
<b>שאגת אריה וקול שחל וגו' .</b> הארי גדול , והשחל בינוני , והכפיר קטן . כולם נתעו . כלומר המלכים והשרים והעבדים וכל שמות הללו שמות אריה הם : 
! {יא}
<b>ליש אובד מבלי טרף .</b> השרים אבדו כליש זה האובד מבלי מזון כך אבד כחם מעשות חמס , אובד אינו לשון אדם האובד דבר ממנו אלא הוא עצמו הולך לאיבוד כמו תעיתי כשה אובד ( תהלים קיט ) וכמו הייתי ככלי אובד ( שם לא ) : <b>ובני לביא יתפרדו .</b> מדבר בדור הפלגה אשר נפוצו משם על פני כל הארץ :

<b>תסמר .</b> הרוח שערת בשרי : <b>תסמר .</b> הוא ל' אדם שעמדו שערותיו : 
! {טז}
<b>יעמוד .</b> עומד היה בפני ולא הייתי מכיר מראהו : <b>דממה וקול .</b> קול דבר חשאי הייתי שומע , ד''א דממה וקול אשמע קול שמעתי מדממה אבל דממה לא שמעתי דממה אלו המלאכים שאומרים שירה כדמתרגם ואחר הרעש קול דממה דקה ( מלכים א יט ) קל דמשבחין בחשאי , משה דממה שמע מכאן שהראשונים נשתמשו בקול והאחרוני' בבת קול כאדם המכה בפטיש וקול ההברה נשמע למרחוק , כך הוא אומר : 
! {יז}
<b>האנוש מאלוה יצדק .</b> בתמיה : <b>ואם מעושהו יטהר גבר .</b> ואם יותר מעושהו יטהר גבר ומה זה חבירך יצעק : 
! {יח}
<b>הן בעבדיו .</b> הצדיקים לא יאמין שלא יחטאו לו ומסלקם מן העולם לפני זמנם : <b>תהלה .</b> ל' הוללות : 
! {יט}
<b>אף .</b> כל שכן : <b>שוכני בתי חומר אשר בעפר יסודם .</b> קברם : <b>ודכאום לפני עש .</b> מן השמים מתישין את כחם עד שהן שפלים ונדכאים לפני הרימה האוכלתן במותן : 
! {כ}
<b>מבקר לערב יכתו .</b> ביום אחד הם כתותים ונדכאים : <b>מבלי משים .</b> על לב לשוב לקונם : <b>לנצח יאבדו .</b> וגם זה הוא מקרא חסר : 
! {כא}
<b>הלא נסע יתרם בם .</b> כמו ( ישעיה טו ) יתרה עשה וגו' גאותם וחכמתם תתבלע בגופן והנה נוסעת ונסחת כמו ( שם לג ) בל יסע יתידותיו : 

~ איוב פרק-ה

! {א}
<b>קרא נא .</b> ע''כ דבר הנבואה מכאן ואילך חוזר לתוכחתו צעק הרבה מי יענך : <b>ואל מי מקדושים .</b> הגוזרים זאת עליך : <b>תפנה .</b> להלחם אתו : 
! {ב}
<b>כי לאויל וגו' .</b> לאויל כמותך יהרג כעסו שאלו היית שותק שמ' תשוב מדת הרחמים עליך : <b>קנאה .</b> היא כעס וחימה כמו ( דברים לב ) הם קנאוני : 
! {ד}
<b>ירחקו בניו .</b> זו היא קללה שקיללתיו : <b>וידכאו .</b> בדגש הוא ידוכאו ואם הוא נקוד רפי משמש ידכאו אחרים : 
! {ה}
<b>אשר קצירו .</b> של רשע : <b>רעב יאכל .</b> כי ימות רשע יבואו הרעבים שקיצץ בחייו וישתלמו גזילותיו מנכסיו : <b>ואל מצנים יקחהו .</b> כמו ויקחו אליך ( שמות כה ) כלומר לצורך העני הנושע מצינים וכלי זיין של רשע זה יקחהו השופט הממונה לכך : <b>ושאף .</b> ובלע איש איש מן הצמים שגזל בחייו : <b>חילם .</b> ממונם שגזל זה בחייו : 
! {ו}
<b>כי לא יצא מעפר .</b> מכה הבאה על האדם לא לחנם הוא ולא מן העפר תצמח לו : און הוא לשון שבר : 
! {ז}
<b>כי אדם לעמל יולד .</b> כי אי אפשר שלא יחטא ויקבל עמל בחטאו לקבל פורענותו ואינו כבני רשף שהם מלאכי' ורוחות שהם יגביהו עוף ואינם מן התחתוני' ( יגביהו לעוף ולהיות מן התחתונים סא''א ) לשלוט בם שטן ויצה''ר : <b>רשף .</b> כמו רשפי קשת ( תהלים עו ) כשהחץ מעופפת נקרא רשפי ל' עוף ( סא''א ) : 
! {ח}
<b>אולם אני אדרוש .</b> אם היו באים עלי יסורים הללו הייתי דורש אל הקב''ה בתפילה ובתחינה ואלו הייתי שם דברתי בבקשה : 
! {י}
<b>הנותן מטר על פני ארץ .</b> ארץ ישראל : <b>ושולח מים על פני חוצות .</b> ע''י שליח לשאר ארצות : 
! {יא}
<b>לשום שפלים למרום .</b> הוא נותן את המטר והוא מפר מחשבות ערומים שמדמין להפקיע שערים ולקנות שדות העניים במעט תבואה : ( קודרים . שקומטים פנים ברעב ) : 
! {יב}
<b>ולא תעשינה ידיהם תושיה .</b> אותה עצה שדימו לעשות : 
! {יג}
<b>נמהרה .</b> כל העצה העשויה במהירות היא סכלות : 
! {יד}
<b>יומם יפגשו חושך .</b> שאין עצתם מתקיימת ונעשית ואור חכמתם נהפך לחושך : 
! {טו}
<b>ויושע .</b> את הנכשלים : <b>מחרב .</b> ומאיזה חרב מפיהם שהיו סבורים לבולעם ומתאוים להשמידם : <b>ומיד חזק .</b> יושיע אביון : 
! {טז}
<b>ועולתה .</b> העלילה והרשעה כמו ( תהלים סד ) יחפשו עולות , עלילות : <b>קפצה .</b> סגרה כמו לא תקפוץ : 
! {יז}
<b>הנה אשרי אנוש וגו' .</b> ביסורין וכשם שהוא ( מרפא ) מכאובי' של אלו כך אני אדרו' אליו על יסורין אלו שיבואוני והיה לך לקבלם ולא למאסם : 
! {יט}
<b>בשש צרות .</b> כשיבאו לעולם יצילך על ידי יסורין שנתייסרת בהם וגם בשביעית כשתבא לא יגע בך ועכשיו הוא מונ' אותן : 
! {כא}
<b>בשוט לשון .</b> של שטן תחבא אותך אלו הן השבע צרות רעב וחרב לשון ושוד וגו' וחית הארץ הם ליסטי' : 
! {כג}
<b>אבני השדה .</b> מין אדם הן : <b>וחית השדה .</b> הוא . שנקרא גרוש''ה בלע''ז וזו היא חית השדה ממש ובלשון משנה תורת כהנים נקראים אדני השדה : <b>השלמה לך .</b> יהיה לה שלום עמך : 
! {כד}
<b>וידעת כי שלום אהלך .</b> ואז בכל מקום תהיה בטוח , כי שלום אהלך : <b>ופקדת נוך ולא תחטא .</b> כמו אל השערה ולא יחטיא ( שופטים כ ) וזו היא השביעית שמנינו למעלה : 
! {כו}
<b>תבא בכלח אלי קבר .</b> שתנצל מן הדבר , ובכלח אין סמך במקרא להבין מליצתו ולפי ענין המליצה יפתר וילמד מעניינו שהוא ל' גמר בישול התבואה כשבישלה כל צורכה ורז''ל פירשו בכלח בגימטריא ששים שנה : <b>כעלות גדיש .</b> כאשר נסתלק הגדיש מן הקרקע בעתו :

! {ג}
<b>לעו .</b> מגומגמין כאדם שאין בו כח להוציא דבר מפיו כתיקונו וכן ושתו ולעו ( עובדיה א ) שכן דרך שיכור לגמגם בדבריו :

<b>הינהק פרא וגו' .</b> כלומר וכי לחנם אני צועק והלא בהמה שוטה אינה נוהקת בזמן שיש לה דשא וכן שור לא יגעה אא''כ אין לו בליל תבואה , בפרא נוהג ל' נהק ובשור ל' געיי' :

<b>ותהי עוד נחמתי .</b> ותהי עוד זאת לי לנחמה : <b>ואסלדה בחילה ולא יחמול .</b> ואבקש ממנו סליחה שלא יחמול מלבצעני ואני איני מוצא לו דמיון במקרא זולתי שבלשון משנה הוא לשון ירא ודואג שהיד סולדת בהן מלכוות ברותחין כלומר אתחולל בדאג' ובחילה בקשה זו : <b>כי לא כחדתי .</b> ולא חסרתי אמריו מלקיימן :

<b>לא תשורני .</b> לא יראני עוד עוד עין שתחפוץ לראותי אחר שאמות : <b>עיניך בי ואינני .</b> להקב''ה מדבר למה הוצרכת לפצעני ולדכאני בייסורים בראייה א' שתשים עיניך עלי ואינני בעולם :

! {כא}
<b>ומה לא תשא פשעי .</b> ומה הדבר הזה שאינך נושא לפשעי : <b>ושחרתני .</b> תבקשני ולא תמצאני :

! {יא}
<b>היגאה גומא .</b> זאת יאמרו אין הגומא גדל אלא בעוד שהביצה לחה במימיה ובליחלוחיה , ביצה מריש''ק בלע''ז :

<b>לא יתנני .</b> אינו מניחני להשיב רוחי אלי : 
! {יט}
<b>אם לכח .</b> איך אריב עמו אם לכח הוא בא אמיץ הנה הוא : ואם למשפט מי יוכל להועידני לפניו ולהחזיק דבר משפטי : 
! {כ}
<b>הלא אם אצדק פי ירשיעני .</b> כי יסתמו דברי מיראה ויעקשני פי : 
! {כא}
<b>תם אני .</b> יודע אני בעצמי שאני תם ואעפ''כ לא אדע נפשי במה אמצא נופש : 
! {כב}
<b>אחת היא .</b> בעולם ועליה אמרתי שתם ורשע הוא מכלה בעבורה , ומה היא : 
! {כג}
<b>אם שוט .</b> ל' של שטן ימית פתאום את הצדיק : <b>למסת נקיים .</b> שהמית ילעג : 
! {כד}
<b>ביד רשע .</b> כלפי שטן : <b>פני שופטיה יכסה .</b> מהבחין בדבר אמת : <b>ואם לא איפוא .</b> דבר אמת דבר זה מי הוא המכלה את התמימים : 
! {כה}
<b>וימי .</b> עם כליון שאר התמימי' קלו מני רץ : 
! {כו}
<b>אבה .</b> שם נהר שוטף : <b>יטוש .</b> יפרח : 
! {כז}
<b>שיחי .</b> דברי צערי : <b>פני .</b> חמתי : <b>ואבליגה .</b> אתחזק כמו המבליג שוד על עז ( עמוס ה ) : 
! {כח}
<b>יגורתי כל עצבותי .</b> שלא יניחוני מלצעוק : <b>ידעתי כי לא תנקני .</b> מעוני מלהפרע ממנו : 
! {כט}
<b>למה זה הבל איגע .</b> לבא במשפט : 
! {ל}
<b>בבור .</b> נקיון : 
! {לא}
<b>בשחת תטבלני .</b> להיות צחון ומגואל כלומר אז תמצא לי עון רב : 
! {לב}
<b>במשפט .</b> בדברי תוכחות משפט משמש ג' לשונו' תחלתו הוכחת דברים , אמצעיתו פסק דין וסופו משטר יסורין וגיבוי חוב ופעמים שהמקרא כותבו ע''ש תחלתו ופעמים ע''ש אמצעיתו ופעמים ע''ש אחריתו : 
! {לג}
<b>מוכיח .</b> דרשני''ר לבע''ז מרא' לכל א' חובתו וזכותו : <b>ישת ידו .</b> משטר אימתו וניגושו : <b>על שנינו .</b> שלא יחזק החזק את הרפה : 
! {לה}
<b>כי לא כן אנכי .</b> על כן שאנכי ירא אני מוחזק בי שלא אתחייב בדין : 

~ איוב פרק-י

! {א}
<b>נקטה נפשי .</b> נפשי מתקוטטת על שאני חי והוא לשון ונקוטות' בפניכם ( יחזקאל כ ) : <b>אעזבה עלי שיחי .</b> צרתי לזעוק ולקבול ולא אוכל לשבחו ולהבליגו : 
! {ג}
<b>הטוב .</b> בתמיה כי תעשוק ותגזול מן הצדיק את צדקו ועל עצת רשעים ומקניטים הופעת זיוך להראות להם פנים שוחקו' : 
! {ה}
<b>הכימי אנוש ימיך .</b> להתגרות בו ולרדוף : 
! {ז}
<b>על דעתך .</b> יודע אתה : 
! {ח}
<b>עצבוני .</b> עשאוני , עצב דפוס כמו אין מעצבין את הקטן במסכת שבת ( דף קמו ) ועתה חיילותיך יחד סביב : <b>ותבלעני .</b> הם התולעים : 
! {ט}
<b>תשיבני .</b> לבסוף : 
! {י}
<b>כחלב תתיכני .</b> כטיפת תולדותי : 
! {יב}
<b>ופקודתך .</b> פקודת משמרתך שמניתה לי שומרים : <b>שמרה רוחי .</b> בבטן אמי , ואחר כך : 
! {יג}
<b>ואלה צפנת בלבבך .</b> לבלעני כאילו אין ( אתה ) זוכרם : <b>ידעתי כי זאת עמך .</b> הכל כמפורש ולא שכחת : 
! {יד}
<b>אם חטאתי ושמרתני .</b> נתת עיניך בי מלהעלים ולעבור על מדתיך ( ומעתה ) ומעוני אין אתה מנקני אלא על הכל אתה פורע : 
! {טו}
<b>שבע קלון .</b> אני : <b>וראה עניי .</b> אין זה ציווי אלא שם דבר ראה עניי בדנ''ץ בלע''ז שכל שעה אני רואהו כמו שאתה אומר ושבע ימים ( לקמן בסוף ) כמו ושבע ימים ( בקמץ תחת שי''ן ) כן תאמר וראה כמו ורואה עניי : 
! {טז}
<b>ויגאה כשחל תצודני .</b> ויגדל בעיניך להיות לך למשא ולפח נפש את אשר כשחל תצודני אשר תפרוש לי מצודה כאילו אני גבור כשחל : <b>ותשוב .</b> מיום אל יום : <b>תתפלא בי .</b> לדקדק ולהפליא במשפט ל' כי יפליא ( במדבר ו ) : 
! {יז}
<b>עדיך .</b> להעיד על חליפת החלאים ומכאובים : <b>וצבא עמי .</b> זמנים קצובים ליסורי אלו משמשים ( צבא א' ) וחוזרים ומתחלפים שנויים משונים מאלה ומשמשין זמן אחר : 
! {יח}
<b>אגוע ועין לא תראני .</b> הלואי אגוע מרחם : 
! {יט}
<b>אובל .</b> הלוואי הייתי מובל : 
! {כ}
<b>ושית ממני .</b> תשית עצמך ממני ומעלי כמו ( ישעיה סה ) קרב אליך קרב לעצמך ורחק ממני : 
! {כא}
<b>בטרם אלך .</b> בעודני חי ולא הגיע עת מותי הנח לי וארגיע : 
! {כב}
<b>עיפתה .</b> שם דבר חשיכה כמו שחר ועפה : <b>צלמות .</b> היא ולא סדרים . בה ( שאין שם סדרי אדם כלומר מנהגים שאין שם יישוב ואור שלה כמו אופל הוא ) : <b>ותופע כמו אופל .</b> מקום מופע שבה כמו אופל הוא : 

~ איוב פרק-יא

! {א}
<b>מענה צופר : 
! {ב}
לא יענה .</b> מאחרים מפני רוב דבריו : 
! {ג}
<b>בדיך .</b> שאתה בודה מלבך : <b>מתים יחרישו .</b> בני אדם : 
! {ד}
<b>ותאמר .</b> להקב''ה זך לקחי ובר הייתי בעיניך : 
! {ו}
<b>תעלומות חכמה .</b> ותדע כי כפלים לתושיה יש לתורה שלא קיימת : <b>כי ( השורש ) ישה .</b> כמו ( דברים כד ) כי תשה ברעך ( כלומר ) הרבה יש לו עוד חוב עליך בדברי עוניך : 
! {ז}
<b>החקר אלוה תמצא .</b> שאתה סבור שקיימת הכל : 
! {ח}
<b>גבהי שמים .</b> דבר שהוא גבוה כגובהי שמים מה תפעל עתה ועמוקה היא החכמה משאול : 
! {ט}
<b>מדה .</b> מדוד שלה ( ס''א מד' שלה ) וכאן הזכיר מידוד בל' זכר ( שלא אמר מדתה ) : 
! {י}
<b>אם יחלוף ויסגיר .</b> ביסורין את מי שירצה : <b>ויקהיל .</b> כל פמליאות שלו לווכח על שהסגיר : <b>ומי .</b> בהן ישיבנו . בדברים למצא עול בפעלו ואם מפני שהנידון שליו ושקט ימים רבים , כן דרכו : 
! {יא}
<b>כי הוא ידע מתי שוא וירא און ולא יתבונן .</b> שהם עושים ימים ושנים ודומה כמי שלא יתבונן ( יתבוננו ) כי מאריך אפו : 
! {יב}
<b>ואיש נבוב .</b> חלול בלא לב אשר לא הבין דרכו : <b>ילבב .</b> יקנה לו לב לשוב לבוראו ולפשפש במעשיו או אדם אשר היה כעיר פרא לימוד מדבר קל בלא דעת : <b>אדם יולד .</b> ילמד עצמו להיות אדם חדש כסדר בני אדם ויכוין דרכו : 
! {יג}
<b>אם אתה .</b> היית מכוין לבך אחר יסורין : <b>ופרשת אליו כפיך .</b> בתחנה : 
! {יד}
<b>ותרחק און .</b> מידך ואל תשכן באהליך עולה : 
! {טו}
<b>כי אז .</b> אם אתה היית בטוח שתשא פניך ממום שתגרום לעצמך לשאת פניך מכל מום ( מתוך כך ) : <b>והיית מוצק .</b> חזק כמו מצוקי ארץ : 
! {טז}
<b>עמל תשכח .</b> וכל עמל היית שוכח : <b>וכמים עברו .</b> וחלפו כן יהיה לך כל זכרון עמל : 
! {יז}
<b>ומצהרים יקום חלד .</b> יותר מאור צהרים יקום לך חלד מזל וזמן כמו וחלדי כאין נגדך ( תהלים לט ) ד''א יקום חלד כמו יקום אור חלודה שלך כלומר שיהיה מקום אפילה מאיר יותר מאור צהרים זו שמעתי אבל הראשון לא נראה ( סא''א ) : <b>תעופה .</b> אפילה שלך כבקר תהיה ע''א תעופה לשון עפעפי שחר ( לעיל ג ) שאם הוא לשון אופל היה לו לומר תעופה נקוד בשו''א כמו תנופה תקומה תרומה תנומה : 
! {כ}
<b>ועיני רשעים .</b> שונאיך : <b>תכלנה .</b> בפנותם ברעתך לראות ולא ישיגו ( תאוותם ) וכל כליון עינים שבלשון קודש זהו המצפה לראות ואינו משיג וכמו ועיניך רואות וכלות אליהם וגו' ( דברים כח ) : <b>ומנוס .</b> הוא מבטח : <b>ותקותם .</b> מה שהם מקוים לראות בך למפח נפש יהיה להם : 

~ איוב פרק-יב

! {א}
מענה איוב : 
! {ב}
תמות חכמה .</b> כשתמותו תמות עמכם כל חכמה כי כל חכמה בכם , אבל : 
! {ג}
<b>גם לי לבב כמוכם .</b> ואת מי אין כמו אלה , שהקב''ה שליט לעשות רצונו ולהאריך שלוות רשעים אינכם אלא כמשחקים בי , ואני הוא ( ההוה ) שחוק לרעהו : 
! {ד}
<b>אהיה קורא .</b> להקב''ה ויענהו מצער הקנטתכם ומה יענהו לתת לו שחוק לצדיק ותמים שלם : <b>שחוק צדיק תמים .</b> היא הענייה אשר יענהו ידעתי אני כאשר אמרתם , כי : 
! {ה}
<b>לפיד בוז .</b> אש של גיהנם עומד למי ששאנן בעשתותו לומר שלום יהיה לי : <b>נכון .</b> מזומן הוא העשתות שאנן להיות במועדי רגל שרגלם מועדת : 
! {ו}
<b>ישליו .</b> ידעתי אני כי ימי ישליו אוהלים לבני קדר השוכנים באהלים המדברה , והם שודדים שנאמר פרא אדם ידו בכל ( בראשית טז ) : <b>ובטוחות .</b> שלימות יש למרגיזי אל , לכל אשר הביא לו הצור השלום בידו אם טוב ואם רע : 
! {יא}
<b>הלא אזן מילין תבחן .</b> ולמה לא ידעו זאת השומעים פעלו כאשר החיך יטעם לאכול : 
! {יב}
<b>בישישים חכמה וגו' .</b> לדעת כי עמו חכמה וגו' : 
! {טו}
<b>וישלחם ויהפכו ארץ .</b> בשליחותו כאשר עשה לדור אנוש שהציף שלישו של עולם : 
! {טז}
<b>לו שוגג .</b> האדם השוגג : <b>ומשגה .</b> שטן המשגהו : 
! {יז}
<b>שולל .</b> שם דבר הוא שטות , מוליכם בשטות כשהוא רוצה לערבב ולבלע חכמתם כמו וסר מרע משתולל ( ישעיה נט ) אשתוללו אבירי לב ( תהלים עו ) ותי''ו אשר במשתולל ואשתוללו כן דרך לשון עברית בתיבה שיסודה שי''ן או סמ''ך והוא בא לדבר בלשון נעשה ונפעל או מתפעל נותן תי''ו אחר אות ראשונה של יסוד כמו וישתכחו בעיר ( קהלת ח ) וישתמר חוקות עמרי ( מיכה ו ) מסתולל בעמי ( שמות ט ) ויסתבל החגב ( קהלת יב ) : <b>יהולל .</b> ישוטה לשון הוללות וסכלות : 
! {יח}
<b>מוסר מלכים פתח .</b> להסיר מוסר סבלם מעל אחרים ומעבירם ממלוכה : <b>ויאסור אזור במתניהם .</b> בתחילתם כשהוא חפץ בהם חוגר חלציהם ומזרזם כח למלוכה : 
! {יט}
<b>כהנים .</b> שרים כמו כהן מדין ( שמות ג ) כהן און ( בראשית מא ) : 
! {כ}
<b>לנאמנים .</b> אף לנאמנים כי פעמים שפתיו נבללים לדבר סרה כגון אברהם והאמין בה' וסוף נכשל במה אדע : 
! {כא}
<b>ומזיח אפיקים רפה .</b> וחגור החזקים מרפה ומפתח חגורם להתיש כחם כמו ( תהלים קט ) ולמזח תמיד יחגרה : <b>אפיקים .</b> חזקים כמו אפיקי נחושה ( לקמן מ ) אפיקי מגינים ( שם מא ) : 
! {כג}
<b>משגיא לגוים ויאבדם .</b> יש אומרים כאשר עשה למצרים בהשיגם את ישראל לפני בעל צפון ( שמות יד ) שגו ואמרו תדעו שהסכים עמנו לטבעם בים כאשר גזרנו כל הבן הילוד היאורה תשליכוהו ( שם א ) וקשה יראתו של בעל צפון שלא נעשו בו שפטים ככל אלהי מצרים וגרם המקום להשגותם כדי לטבעם וזה הלשון אינו עיקר דאם כן היה לו לומר בצד ימין משגיא כמו שוגה אבל זה עיקר מגדיל לשון שגיא כח משגיא את הגוים ומצליחם ולבם מתגאה בהצלחת' והוא היה אבדן שלהן מה שהצליחו : <b>שוטח .</b> מרבה אותם להשטיח בארץ ולכסותם : 
! {כד}
<b>בתוהו לא דרך .</b> בתוהו שאינו דרך : 

~ איוב פרק-יג

! {ג}
<b>אולם אני .</b> איני מבקש אלא לדבר אליו ולהתווכח עמו אחפוץ : 
! {ד}
<b>טופלי שקר .</b> מחברי דברי שקר : <b>רופאי אליל כולכם .</b> כל אליל שבמקרא לשון אל ורבותינו פירשו זה גיד הצואר שאין לו רפואה ושמו אלל : 
! {ו}
<b>וריבות .</b> הוכחות : 
! {ז}
<b>הלאל תדברו עולה .</b> אחרי אשר תחת אל באתם לריב לא טוב שתדברו עולה : <b>ולו תדברו רמיה .</b> בעבורו תדברו רמיה כמו ( לקמן לג ) הן אני כפיך לאל וכמו אל הארץ אשר דבר להם ( דברים ט ) לצורכם וכמו ילחם לכם ( שמות יד ) האתם תריבון לבעל ( שופטים ו ) הוקשה בעיני לדמותו כאן והוא דוגמת ביד משה לו על קרח ( במדבר יז ) : 
! {ח}
<b>הפניו תשאון .</b> וכי להחניף לו בשאת פנים באתם : <b>אם לאל .</b> בשבילו תתווכחון על רמיה : 
! {ט}
<b>הטוב כי יחקור אתכם .</b> ותמצאו שקרים : <b>אם כהתל באנוש תהתלו בו .</b> לומר לכבודך עשינו עצמינו שקרים : 
! {י}
<b>הוכח יוכיח אתכם .</b> הלא יודיע לכם דבתכם : <b>אם בסתר .</b> וכי בסתר שתעמדו לפניו בדין פנים תשאו לו לומר שם נשאנו פניך : 
! {יא}
<b>הלא שאתו .</b> גבהו ואימתו תבעת אתכם ויש אומרים שאתו שריפת אשו כמו וישאם דוד ואנשיו ( שמואל ב ה ) וכמו משאת העיר ( שופטי' כ ) : 
! {יב}
<b>זכרוניכם .</b> נמשל משלי אפר . זכרוניכם נמשל לאפר , אתם סבורים להתמשל לאברהם שאמר אנכי עפר ואפר ( בראשית יח ) : <b>לגבי חומר .</b> שוה לעפר יהי גביכם אצלו . גביכם . גובה שלכם כמו ותבני לך גב ( ביחזקאל טו ) : 
! {יג}
<b>ויעבור עלי מה .</b> מה שיעבור עלי בעון צעקתי יעבור עלי ואל תחושו על מה אני צריך לשאת ולאחוז : 
! {יד}
<b>בשרי בשני .</b> לייסרני ולכופני אל השתיקה : <b>ונפשי אשים בכפי .</b> להגיש נפשי למות בייסורי שתיקה : 
! {טו}
<b>הן יקטלני .</b> הלא אם יקטלני איני נפרד ממנו ותמיד לו אייחל על כן אין מרד ופשע בדברי : <b>אך דרכי .</b> ומדותי שנהגתי בהם : <b>אל פניו אוכיח .</b> לדעת מה יפשיעני : 
! {טז}
<b>גם הוא .</b> וכאשר אני שלם עמו כן הוא גם הוא לי לישועה ולא תמצאו אתם חן בעיניו : <b>כי לא לפניו חנף יבוא .</b> וזה שנא' להם ( לקמן מב ) כי לא דברתם אלי נכונה : 
! {יח}
<b>ערכתי משפט .</b> הוכחת דברים סדרתי טענותי בלבי מה להשיב : 
! {יט}
<b>אחריש ואגוע .</b> אם לא אריב אחריש ואגוע : 
! {כ}
<b>אך שתים אל תעש עמדי .</b> דין של מעלה ודין של מטה ואם באחת תבוא למשפט אז מפניך לא אסתר אם בדין של מעלה אני אומר חטאתי מכסה פשעיו לא יצליח וגו' ( משלי כח ) ואם בדין של מטה אף אנכי לא אודה כסוי חטאה ל''ש ( סא''א ) : 
! {כא}
<b>כפך מעלי הרחק .</b> כפייתך כמו שהשיבו אליהוא ואכפי עליך לא יכבד ( לקמן לג ) ויש פותרים כפך כלומר ידך וקשה לי לאומרו כי לא מצינו כף לרעה כי אם להגן כמו ושכותי כפי ( שמות לג ) : 
! {כד}
<b>פניך תסתיר .</b> לראות את דרכי יושרי : 
! {כה}
<b>תערוץ .</b> תפחיד : 
! {כו}
<b>מרורות .</b> סרבניות אשר מריתי פיך אתה כותב ולא הטובות שפעלתי : 
! {כז}
<b>בסד .</b> בלשון ארמי סדנא בסדני' יתיב עץ גדול שנותנין בו רגלי האסורים ובל' יוני אשתו''ק : <b>תתחקה .</b> אתה עצמך מתחקה על פסיעותי כן פסע זה וכה פעל וכמו מחק הארץ ( יחזק' מג ) וחיק האמה ( שם ) זה היסוד אפיקיו''ל בלע''ז הן על כפים חקותיך ( ישעיה מט ) : 
! {כח}
<b>והוא כרקב יבלה .</b> הגוף הזה אשר תרדוף ירקב ויבלה ואין כבודך ברדיפתך : 

~ איוב פרק-יד

! {ג}
<b>אף על זה .</b> ההבל והנרקב : <b>פקחת עיניך .</b> לדקדק בחטאיו : <b>ואותי .</b> אתה מביא במשפט על עוונותי : 
! {ד}
<b>מי יתן טהור .</b> מדבר טיפה סרוחה ושכבת זרע שהוא טמא לא אחד בהם טהור שלא יחטא : 
! {ה}
<b>אם חרוצים ימיו .</b> אם גמול זה הוא משתלם שחרוצים וקצובים ליום קבוע ומיועד למיתה : <b>ומספר חדשיו אתך .</b> קצובים וחוק זה עשית לו ולא יתקיים עוד יותר בעולם די לך בתשלומין הללו : 
! {ו}
<b>שעה מעליו .</b> באותו מיעוט חרץ הימים ויחדל לו ממכאוב עד ירצה בזקנותו ותשות כחו , את יום מותו כשכיר הרוצה להשלמת עבודת יומו כי עבירתו מן העולם חלוטה ונצמתת משאר כל העוברים : 
! {ז}
<b>כי יש לעץ תקוה .</b> שיחלוף : <b>ויונקתו .</b> אשר מגזעו : <b>לא תחדל .</b> מהיות ענף : 
! {ט}
<b>ועשה קציר .</b> היא יונקת כמו ( תהלים פ ) תשלח קציריה עד ים : <b>נטע .</b> שם דבר הוא וטעם הברתו למעלה בנו''ן וננקד כולו קמץ מפני שהוא סוף פסוק אף מגזרת נטע הוא : 
! {יא}
<b>אזלו מים מני ים .</b> ממקום שנהר בא ויוצא ממקורו הבא מאליו מן הים : <b>והנהר .</b> הבא משם יחרב ויבש לעולם , כן איש שכב ולא יקום : 
! {יד}
<b>כל ימי צבאי איחל .</b> לתחייה : 
! {טו}
<b>תקרא .</b> תחינה היא קרא אלי ואנכי אענך להוכיח דברי : <b>תכסוף .</b> תחמוד : 
! {טז}
<b>כי עתה .</b> אתה מריע לי : <b>צעדי .</b> אתה סופר : <b>לא תשמור .</b> אינך ממתין על חטאתי מלהפרע כמו ואביו שמר את הדבר ( בראשית לו ) המתין שתתקיים ובלשון משנה ( גמ' סנהדרין סג ) לא יאמר אדם לחבירו שמור לי בצד עבודת כוכבי' פלונית ל' המתן : 
! {יז}
<b>חתום בצרור .</b> חתום ומשומר בצרור בגד ככסף ומרגליות שלא תאבד : <b>ותטפול על עוני .</b> נתחברת על עוני : 
! {יח}
<b>ואולם הר נופל .</b> הר הנופל גבהו יבול ויעשה עפר ותהא לו תקוה ליהנות ממנו וצור אשר נעתק ממקומו גם הוא , יבול לשון תבואה , או : 
! {יט}
<b>אבנים .</b> אשר שחקו מים . בעברם עליהם תמיד : <b>תשטף .</b> השטיפו ספיחיה של אבן ליהפך לעפ' ולצמוח בו דב' כאשר יצמח ספיח הקציר שעבר כן יתהפך עפר זה ספיחיה לאבן : <b>ותקות אנוש .</b> אינה כן אשר משימות אבדה סלה : 
! {כ}
<b>תתקפהו .</b> תחזק ממנו ותאנסהו בתקפך ויהלך מן העולם : 
! {כא}
<b>יכבדו .</b> יהיו כבידים בכסף ובזהב והוא לא ידע : <b>ויצערו .</b> וימעטו מכל טוב והוא לא יבין : 
! {כב}
<b>אך בשרו עליו יכאב .</b> קשה רימה למת כמחט בבשר החי : 

~ איוב פרק-טו

! {א}
<b>( מענה אליפז )</b> : 
! {ב}
החכם יענה דעת רוח .</b> תמוה הוא וכי מי שהוא חכם יענה וירים קול בדעת של רוח והבל : 
! {ג}
<b>הוכח בדבר לא יסכון .</b> להתווכח בדבר שלא תהא לו מהן הנאה כמו ותהי לו סוכנת ( מלכים א א ) וכמו יסכן בם ( קהלת י ) יהנה בם : 
! {ד}
<b>אף אתה .</b> כל שכן אתה שחכם מאוד אתה תפר יראה במרבית דבריך ותגרע שיחה כמו כי יגרע נטפי מים ( לקמן לו ) וכמו ותגרע אליך חכמה : 
! {ה}
<b>כי יאלף עונך .</b> יצרך הרע את פיך לדבר כדברים האלה והיה לך לבחור לשון ערומים : 
! {ו}
<b>ירשיעך פיך ולא אני .</b> כלו' השיבני על שאלתי ונמצא פיך הרשיעך כי טוב לי שירשיעך פיך ולא אני וזו היא שאלתי : 
! {ז}
<b>הראישון אדם תולד .</b> הבטרם אדם תולד , ה''א הזאת משמשת לשון תימה לפיכך נקודה בחטף פתח ומליצתו הראשון לאדם הנוצר מאדמה נולדת שידעת לקיים כל חכמת מצות הבורא : 
! {ח}
<b>ותגרע .</b> ותרבה מלמעלה למטה לרדת עליך : 
! {י}
<b>כביר .</b> מרובה ימים מאביך : 
! {יא}
<b>המעט ממך תנחומות אל .</b> אשר הטיב לך הצור עד עתה ימים רבים ודברו הטוב אשר לאט עמך בלאט ונחת דשנו : <b>לאט .</b> הנעים ולשון פעל הוא ולא שם דבר ולכן הוא נקוד חציו קמץ וחציו פתח : 
! {יב}
<b>יקחך .</b> ילמדך כמו יערוף כמטר לקחי ( דברים לב ) : <b>ירזמון .</b> כמו ירמזון כמו שמלה שלמה ( שמות כב ) : 
! {יד}
<b>מה אנוש כי יזכה .</b> מה כחו שיוכל להיות זכאי נגד הקב''ה : 
! {טו}
<b>ושמים .</b> צבא השמים : <b>זכו .</b> יצהרו כל ל' זך הוא צוהר : 
! {טז}
<b>אף .</b> כל שכן : <b>ונאלח .</b> נתעב ומעורבב וכן כולו סג יחדיו נאלחו ( תהלים נג ) : 
! {יח}
<b>אשר חכמים יגידו .</b> ויתוודו פשעם ולא כחדו עוונם מאבותם זהו אשר ראיתי והנני אספר לך והיכן ראיתי יהודה הודה ולא בוש במעשה תמר ראובן הודה ולא בוש במעשה בלהה , ומה היה שכרם : 
! {יט}
<b>להם לבדם נתנה הארץ .</b> בשכר זה זכה יהודה למלכות נצח ונטלו שניהם מן הנחלה חלקם ולא גורשו מן נחלתם בעוונם מאחר שהודו : <b>ולא עבר זר בתוכם .</b> כשבא משה לברכם שנא' ( דברים לג ) יחי ראובן ואל ימות וזאת ליהודה , ד''א ולא עבר זר בתוכם בנחלה של עתיד לבא החלוקה בספר ( יחזקאל מח ) : 
! {כ}
<b>כל ימי .</b> הרשע . שאינו שב מחטאו ואינו מודה פשעיו כל ימיו : <b>הוא מתחולל .</b> ומתאונן על הבטחתו שישלם רעתו אשר פעל כל מספר שנים אשר נצפנו לעריץ הוא מתחולל : <b>מתחולל .</b> ל' אנינות ולא שמעתי : 
! {כא}
<b>קול פחדים באזניו .</b> לאמר הגיע עת תשלומי : <b>בשלום .</b> וכשהוא שרוי בשלום אז השודד יבואנו : 
! {כב}
<b>וצפו הוא אלי חרב .</b> ועתיד הוא אלי חרב ועוני לנוד ללחם איה ימצאנו : 
! {כג}
<b>נודד הוא .</b> כלומר מובטח הוא לנוד : <b>ידע .</b> לו באמת : <b>כי נכון .</b> ומזומן לבוא בידו כלומר אליו ועמו יום חשך כמו מן הבא בידו ( בראשית לב ) : 
! {כד}
<b>תתקפהו .</b> תחזק המצוקה ממנו ותאנסהו : <b>כמלך עתיד לכידור .</b> לא מצאתי דמיונו ויש לפותרו באותיות המתחלפות רי''ש בדלית כמו אשכנז וריפת שבתורה ( שם י ) ( ובדברי הימים א א ) אשכנז ודיפת וכן כתים ודודנים ( שם ) אף כאן לכידור כמו לכידוד כאותו המלך שהוא עתיד ליקוד באש של גיהנם והוא סנחריב שנתנבא הנביא עליו ( ישעיה ל ) כי ערוך מאתמול תפתה גם היא למלך הוכן לשרוף חייליו יקד יקוד ודומה לו כידודי אש יתמלטו ( לקמן מא ) אם ( ס''א אש ) לפידים , אם ( ס''א אש ) להבות : 
! {כו}
<b>ירוץ אליו .</b> היה רץ אל הבורא בגובה וזקיפת צואר להכעיסו : <b>בעבי גבי מגיניו .</b> בחזק' כחו , גבי ל' גובה כמו ותבני לך גב ( יחזקאל טז ) : 
! {כז}
<b>פימה עלי כסל .</b> עשה פה על חלצים שחלבו ועובי שומנו כפול על כסליו ונראין כמי פה : <b>פימה .</b> מגזרת והיתה הפצירה פים ( שמואל א יג ) זו היא לימ''א של נפחים שיש לו פיות הרבה ונקראת הפצר פיות : 
! {כח}
<b>וישכון ערים נכחדות .</b> כן דרך בעלי גאוה בונים חרבות לשם : 
! {כט}
<b>ולא יקום חילו .</b> לא יתקיים לימים רבים : <b>ולא יטה .</b> לא ינטה מליצתו כמו ויטו אחרי הבצע ( ש''א ח ) שפי' וינטו הם עצמם אף כאן לא ינטה מן השמים לארץ מנלם מתת תאותם וזימון מחשבתם והוא כשתי תיבות מן להם : <b>מן .</b> הכנתם כמו וימן להם המלך ( דניאל ח ) ודומה לו הפצירה פים בשתי תיבות ( שמואל א יג ) וכן אראלם ת''י אתגלי להון ( ישעיה לג ) זה נראה בעיני ומנחם מסייעני , ד''א מנלם , לשון כנלותך לבגוד ( שם ) והנו''ן שבו יסוד נופל לשון תכלית לא יטה לארץ להיות בטילה גזירת חרוץ : 
! {ל}
<b>ויסור .</b> מן העולם : <b>ברוח פיו .</b> של שליט בעולם וזה מקרא חסר ויש פותרים בשביל רוח פיו שהרע לדבר סרה : 
! {לא}
<b>אל יאמן .</b> אותו אשר בשוא נתעה כי שוא יהיה תמורתו ויגיעו כמו ( רות ד ) ועל התמורה : 
! {לב}
<b>בלא יומו .</b> תעמוד תמורתו על מליאתה מליגדל ומלהצליח עוד כמו ( בראשית נ ) וימלאו לו ארבעים יום פי' ואשלימו : <b>וכפתו .</b> כפיית ענפיו כל ענפי אילן דומין לכיפה : 
! {לה}
<b>הרה עמל וילד און .</b> עשה רעה וישתלמו שכר הלידה לפי ההריון : <b>תכין .</b> תזמין : <b>מרמה .</b> למרמה תהפך להם תאוותם : 

~ איוב פרק-טז

! {א}
<b>( מענה איוב ) : </b>
! {ג}
ימריצך .</b> ישערך פיך לחשוב דברים נמרצים על אופניהם : 
! {ד}
<b>ככם אדברה .</b> ככם אני יודע לקנתר : <b>לו יש נפשכם .</b> ביסורין תחת נפשי גם אני הייתי מחבר עליכם במילין ומניע עליכם ראשי בצעקתכם ומאמץ אתכם במו פי לאמר החרישו התאמצו ועמדו ביסוריכ' כאשר אתם תאמרו אלי : 
! {ה}
<b>וניד שפתי .</b> שאני נד וקובל עתה וצועק אז יחשוך ולא אזעוק ולא אנוד עוד , אבל עתה : 
! {ו}
<b>אם אדברה .</b> וגו' : 
! {ז}
<b>השימות .</b> לשון שתיקה כמו אשתומם וכמו שומו שמים ( ירמיה ב ) שתקו והתמהו : 
! {ח}

<b>אין שריד לאכלו .</b> לא שייר ממאכלו לחלק לעניים : <b>על כן לא יחיל טובו .</b> לא יצליח כמו ( שם ) יחילו דרכיו ובאנשי סדום מדבר שהיה עיניה' צרה באורחי' כי כן העיד ( יחזקאל טז ) על סדום גאון שבעת לחם וגו' וידי עני ואביון לא החזיקו ורבותינו פירשו אין שריד לאכלו לא השרה תלמידי חכמים על שלחנו כמו ובשרידים אשר ה' קורא : 
! {כב}
<b>במלאות שפקו .</b> בהתמלאות תאות לבו די , אז : <b>יצר לו .</b> תבא עליו צרה : <b>ספקו .</b> ל' די כדמתרגמינן כד ספיקו גמליא , ומצא להם היספקון : <b>כל יד עמל תבואנו .</b> תחזק עליו ותצר יד עניים : 
! {כג}
<b>יהי למלא בטנו ישלח בו חרון אפו .</b> שליחות חרון אפו יהי למלא בטנו ואין זה לשון קללה אלא לשון נבואה ועתיד : <b>וימטר עלימו .</b> הקב''ה : <b>בלחומו .</b> בזעף מלחמתו פחים אש וגפרית כענין שנא' ( בראשית יט ) וה' המטיר על סדום וגו' : 
! {כד}
<b>תחלפהו .</b> חצים שתזרוק הקשת יחלפו את גופו לעבור מעבר אל עבר , ל''א תחלפהו , כמו תחליפהו ( תקדמהו ) קשת נחושה : 
! {כה}
<b>שלף .</b> השולף עליו את הנשק : <b>ויצא .</b> הנשק : <b>מגוה .</b> מתוך תערה : <b>וברק .</b> פלנדור''ש בלע''ז כמו ולמען היות לה ברק ( יחזקאל כא ) פורבי''ר : <b>ממרורתו יהלוך .</b> ממרורתו של נשק יהלוך ומשם יהיו אמים ופחד על הרשע : 
! {כו}
<b>לצפוניו .</b> לצורך מצפוני גנוזים הצפונים לו : <b>אש לא נופח .</b> אש של גיהנם : <b>ירע שריד באהלו .</b> תבא רעה לשריד שישאר באהלו , כל הפרשה כולה באנשי סדום : <b>ירע שריד באהלו .</b> אפי' לוט הנמלט מהם באה רעה לאהלו זו אשתו שהיתה נציב מלח כן מצאתי במדרש רבי תנחומא : 
! {כז}
<b>מתקוממה .</b> ל' אויב הקם כנגדו : 
! {כח}
<b>יגל .</b> ל' גלות : <b>נגרות .</b> יהיו אוצרותיו נתונות למשיסה ולבז כמו ( מיכה א ) כמים המוגרים ארצה : 
! {כט}
<b>אמרו מאל .</b> האמור עליו מלפני המקום אמרו לשון זכר מגזרת אמרי האזינה ( תהלים ה ) , אמרתו לשון נקבה מגזרת אמרות טהורות : 

~ איוב פרק-כא

! {א}
<b>( מענה איוב ) : </b>
! {ב}
ותהי זאת תנחומותיכם .</b> את אשר תחרישו ותשמעו אלי זאת תהי לי תנחום מאתכם : 
! {ג}
<b>שאוני .</b> סבלוני : 
! {ד}
<b>האנכי לאדם שיחי .</b> למה תעמדו על צעקתי וכי אני לאדם כמוני משיח שישמע קולי ואם מדוע לא תקצר רוחי , לצעוק מצרה לי : 
! {ה}
<b>פנו אלי .</b> לשמוע : <b>והשמו .</b> התמהו לדברי : <b>ושימו יד על פה .</b> שלא תדעו להשיב : 
! {ו}
<b>ואם זכרתי .</b> דבר שאני רוצה לומר מדוע רשעים יחיו : <b>ונבהלתי .</b> שאני רואה רשעי דור המבול עתקו גם גברו חיל : 
! {ט}
<b>שלום מפחד .</b> שלא היו מזיקין שולטין בהן : 
! {י}
<b>שורו עבר .</b> את הנקבה : <b>ולא יגעיל .</b> לא יורה בה זרע פסולת שיהא חוזר ונפלט מגופה בלא הריון כל לשון הגעלה פליטה הוא וכן בל' משנה מגעילן ברותחין וכן כי שם נגעל מגן גבורים ( שמואל ב א ) נפלט משיחתו והיה כאילו לא נמשח בשמן שהיו רגילין למשוח מגיני עור כדי שיהא חלק ומחליק את הרומח והחנית המכה בו והוא כמו שרים משחו מגן ( ישעי' כא ) : <b>תפלט פרתו .</b> בעת תלד את וולדה : <b>ולא תשכל .</b> בלא עת : 
! {יא}
<b>ישלחו כצאן עויליהם וגו' .</b> בעת לידתו היה הולך ומרקד מיד ואם פוגע במזיק נלחם בו והיתה אומרת לו אמו הבא לי מספריים ואחתוך שערך בהגדת בראשית , לך והדליק לי את הנר בראשית רבה : 
! {יב}
<b>ישאו .</b> היו נושאין קול בשיר בתוף וכנור , עוגב אינו אלא ל' שחוק על שם העגבות אשביימנ''ט בלע''ז , כ''ש : 
! {יג}
<b>וברגע שאול יחתו .</b> כשמגיע יום מותו היה מת לפי שעה בנחת בלא יסורין : <b>יחתו .</b> ירדו ובתרגום נחיתו : 
! {טו}
<b>ומה נועיל .</b> מה הנאה יש לנו כי נפגע בו אין אנו צריכין לו אפי' לטיפת גשמים כי אד יעלה מן הארץ ( בראשית ב ) : 
! {טז}
<b>הן לא בידם טובם .</b> בתמיה וכי אין כל טובם בידם : <b>ועצת רשעים רחקה מני .</b> איוב היה משבח בעצמו שראה אותם ולא נמנה עמהם : 
! {יז}
<b>כמה נר רשעים ידעך .</b> למעלה הוא חוזר למה רשעים יחיו כמה תאריך זמן זה וחזר ומקללם נר רשעים ידעך וגו' : <b>חבלים .</b> גורל הראוי להם : 
! {יט}
<b>אלוה יצפון לבניו אונו .</b> בתמיה וכי מדה היא שכן זה הרשיע כל ימיו ויהא בשלום וימות בשלום ואונו ורשעו יצפון הקב''ה לבניו ללקות להם : <b>ישלם אליו וידע .</b> שבח הוא להקב''ה שישלם לרשע עצמו גמולו ויבין שהרשיע לעשות : 
! {כ}
<b>יראו .</b> עיניו כידו . שברו , ואיני יכול למצוא לו דמיון בתורה כי אם לפי ענין הפסוק נראה שבר וכן הרב לא מצא לו דמיון : ( כידו . לשון אידו א' מתחלף בכ''ף ) : 
! {כא}
<b>כי מה חפצו בביתו אחריו ומספר חדשיו חצצו .</b> כי מה הוא חושש שקלירי''ר בלע''ז וחפץ בביתו לאח' מותו לדאג בחייו בפורענו' העתידה לבא עליהם אחרי שמספר חדשיו נקבצו ויכלו לפני הרעה שהבטיחם המקום והיו ימיו מאה ועשרים שנה ( שם ו ) זאת היא שאמרתי לכם פנו אלי והשמו : 
! {כב}
<b>הלאל ילמד דעת .</b> היש אחד מכם אשר הוא תחת האל שילמד דעת מה היא המדה הזאת : <b>והוא רמים ישפוט .</b> כגון אני והצדיקים והחשובים הוא מייסר ודן ונפרע מהם כמו ( לקמן כד ) ולא יפנה דרך כרמים האמור בנח ובמתושלח באגדת חלק , ל''א הלאל ילמד דעת כלומר האני צריך ללמדו דעת לשפוט אמת הוא עצמו נודע שכן אבל הוא ישפוט דינים רמים בגבהו ונגדלו ואינו חש לדקדק , זה שמעתי ולא הראשון : 
! {כג}
<b>זה .</b> הרשע : <b>ימות .</b> נענש בעצם תומו במראה שלימתו לא חסר : <b>שלאנן .</b> כמו שאנן : 
! {כד}
<b>עטיניו מלאו חלב .</b> ל' משנה הוא ( מנחות פו ע''א ) עוטנו בבית הבד צוברים הזיתי' יחד ושומנן מתעגל ומתאסף לתוכו כדי שיהא מוכן כשעוצרו בבית הבד ואותו כלי שצוברין אותו כדי להאסף שמנו לתוכו נקרא מעטן אף כאן חלבו ולחלוחו ושומנו נקרא עטיניו : 
! {כה}
<b>וזה ימות בנפש מרה .</b> הרמים והקדושים אליו יפרע מיד מהם : 
! {כו}
<b>יחד על עפר ישכבו .</b> ולאחר מותן אין ניכר לבריות מי הרע מי הטוב כי כזה כן זה יחד על עפר ישכבו : 
! {כז}
<b>הן ידעתי מחשבותיכם .</b> והמזמות אשר עלי תחמוסו . תריעו לחשוב חמס : 
! {כח}
<b>כי תאמרו וגו' .</b> בשביל הרעה אשר באתני אתם אומרים ראו איה ביתו של זה שהיה נדיב ואיה אהל ומשכנות שאר רשעים כלומר כמוהו כמוהם ברשעו אבד ביתו , ( ולמה אתם בוטחים בשלותכם ) : 
! {כט}
<b>הלא שאלתם וגו' .</b> כל זה אתם מושלים עלי משלות : <b>ואותתם .</b> אשר הודיעו אתכם לא תנכרום משום על לב כמו ויתנכר אליהם ( בראשית מב ) : 
! {ל}
<b>כי ליום .</b> אמרו לכם אשר ליום איד יחשך רע , הרשע מונע ונחשך מרעה ושמור ליום הקבוע לאידו כמו אשר חשכתי לעת צר ( לקמן לח ) הצנעתים וחשכתים להודיעם עד יום קרב ומלחמה : 
! {לא}
<b>מי יגיד על פניו דרכו .</b> מי הוא המלך הגדול אשר יגיד על פניו דרכו ולא יראו ממנו כמה הוא נשגב זה הקב''ה שעשה : <b>ומי ישלם לו .</b> וגמול רשעו מי ישלם לו : 
! {לב}
<b>והוא לקברות יובל .</b> סופו יובל לקבר : <b>ועל גדיש ישקוד .</b> כשהוא נקבר בשדות אצל הגדישים : ( ועל גדיש ישקוד . ידונו אותו על שלא קיים המצות השייכים בשדה לקט שכחה ופאה ושאר מצות הנוהגים בעומר ובגדיש ולכן שקדו וימהרו עליו הפורענות הג''ה מר''ע ) : 
! {לג}
<b>רגבי .</b> בלשט''ז בלע''ז : <b>ולפניו אין מספר .</b> לנמשכים לאותו דרך כל אלה מזימות תחמסו עלי : 
! {לד}
<b>ואיך תנחמוני הבל .</b> מאחר שתשובותיכם נשאר מעל נשכחה ואבדה מכם תשובות החכמה ותשובותיכם לא נשארה בידכם אלא תשובת מעל , מעל שם דבר הוא לכך ננקד כולו קמץ וטעמו מלעיל כ''ש : 

~ איוב פרק-כב

! {א}
<b>( מענה אליפז )</b> : 
! {ב}
<B>הלאל יסכן גבר .</b> הלצורך ולהנאת היוצר יועיל גבר ויהנה כאשר ילמד דעת אל הבריות וחכמה , יסכן הראשון לשון הנייה כמו ותהי לו סוכנת ( מלכים א א ) והשני לשון למוד כמו ההסכן הסכנתי ( במדבר כב ) וגם הראשון יתכן להיות נפתר לשון למוד הלטובת היוצר ולצרכו ילמד גבר כאשר ילמד השכל : 
! {ג}
<b>החפץ לשדי .</b> כמו כי מה חפצו בביתו אחריו ( לעיל כא ) ההנאה וחשש לו אם תצדיק מעשיך בהתווכח לפניו שיבא עמך להתווכח בדבריך : <b>ואם בצע .</b> ממון לו אם תתם דרכיך : 
! {ד}
<b>המיראתך יוכיחך .</b> או אם מיראה שיירא מפניך יבא ויתווכח עמך : 
! {ה}
<b>הלא .</b> יודע הוא כי רעתך רבה : 
! {ח}
<b>ואיש זרוע לו הארץ .</b> בתמיה וכי מחמת שהיית גדול תירש את הארץ כלום הוא ראוי להתקיים : <b>ישב בה .</b> יאריך בה כמו ותשבו בקדש ( דברי' יא ) : 
! {יא}
<b>או חשך לא תראה .</b> כלומר לא תתן עיניך ובטחונך בכח לאמר : 
! {יב}
<b>הלא אלוה גובה שמים .</b> ולא ישפיל לראות : <b>וראה ראש כוכבים כי רמו .</b> כמו וראה עניי שבסוף מענה החמישי ( לעיל י ) כלומר ונותן אתה עין בראש כוכבים כי רמו ובשביל כך אמרת : 
! {יג}
<b>מה ידע אל .</b> במעשה החשך : <b>הבעד ערפל ישפוט .</b> הכנגד החשך הזה יראה וישפוט , הנה : 
! {יד}
<b>עבים סתר .</b> לפניו ולא יוכל לראות : <b>וחוג שמים יתהלך .</b> ולא ידע את אשר בארץ : <b>חוג .</b> עוגל מחוגת שמים כמו ובמחוגה יתארהו ( ישעיה מד ) קומפ''ש בלע''ז : 
! {טו}
<b>האורח עולם .</b> הדרך הראשונים אשר מאז : <b>תשמור .</b> שמרת בלבך לזכור מה נהייתה בהם : 
! {טז}
<b>קומטו ולא עת .</b> נגזרו בלא עתם : <b>נהר וגו' .</b> ונהר של מבול או גפרית ואש של סדום הוצק ביסודם : 
! {כ}
<b>אם לא נכחד קימנו .</b> התשמור וראית אם לא נכחד יקומם : <b>קימנו .</b> כמו קומתו של אותו הדור כמו ( הושע יב ) בית אל ימצאנו ושם ידבר עמנו כמו עמו : <b>ויתרם .</b> גאותם : <b>אכלה אש .</b> ברותחין נדונו ר' תנחומא דרש שהיו בעלי קומה ולא מתו בתהומות ובאו עליהם רותחין מלמעלה וזה יתרם קימה יתרה שלהם , כ''ש : 
! {כא}
<b>הסכן .</b> למוד להיות עמו שלם : <b>ושלם .</b> ותהיה שלם ולא תחסר בהם בדברים האלה : <b>תבואתך טובה .</b> תבואך טובה : 
! {כד}
<b>ושית על עפר בצר .</b> על אשר תשוב אליו תשית על הארץ חוסן ומבצרים : <b>ובצור נחלים .</b> הסלעים גבוהים שבנחלים עמוקים יהיה לך אוסף צבירת אופיר היא סגולת מלכים שהיו מסגלים ואוצרים זהב הבא מאופיר : 
! {כה}
<b>והיה שדי בצריך .</b> אם שדי זה קדש תמהתי על פתרונו שמשמעו קללה שימנה עם אויביו כמו ואת היית בעוכרי ( שופטים יא ) עם עוכרי וע''י צורך נדוננו מן המקראות חסרים ( כמו והיה שדי שופט בצריך , ואם נוכל לאמרו חול לשון דיין וחוסן זה פתרונו ) והיה חוסן מבצריך כמו ושית על עפר בצר כלומר הון עתק תהיה נבצר ולפי הענין כן פתרונו שהרי סופו מוכיחו וכסף תועפות לך ל' חוסן כמו ותועפות ראם לו ( במדבר כג ) : 
! {כו}
<b>ותשא אל אלוה פניך .</b> לא תבא להרים פניך אלא לשאול , צרכיך הימנו : 
! {כז}
<b>ונדריך תשלם .</b> מובטח אתה שירצו קרבנות תשלומי נדריך : 
! {כח}
<b>ויקם לך .</b> כמו ויקום לך : 
! {כט}
<b>כי השפילו ותאמר גוה .</b> ואם רואך דורך שפל תאמר בהבטחה להגביהו ויגבה : <b>ושח עינים .</b> בצרה הבאה ( אליו ) בעונו : <b>יושיע .</b> אותו אלוה בידך ובתפילתך : 
! {ל}
<b>ימלט אי נקי .</b> אין נקי כמו אי כבוד ( ש''א ד ) אין כבוד וכן ( משלי לא ) ולרוזנים אי שכר אין שכר אין נאה להם שכרות שלשתן חברם מנחם במחלוקת אחת ואע''פ שהן שתי תיבות מפרשין כן ונמלט את אי נקי בבור כפיך וכן שמעתי , ל' אחר שמעתי שהיו אומרים מפי ר' יעקב שהוא לשון אדם השרוי בצער ובאנקה והיו''ד יתירה בתיבה כמו היושבי בשמים ( תלים קכג ) אבל לא שמעתי מפי הרב :

! {ב}

! {ג}
<b>ואמצאהו .</b> לשופטי : <b>תכונתו .</b> מכון כסאו : 
! {ד}
<b>משפט .</b> הוכחת דברים : 
! {ה}
<b>( אדעה מלים יענני .</b> וגם אם יסתום אותי מלדבר לפניו שמא ידבר הוא עמי ואדעה מלים יענני רבי תנחומא ) : 
! {ו}
<b>הברב כח .</b> דברים ותוכחות של חטאיו יריב עמדי : <b>לא אך .</b> מכה וכפיה לא ישים בי אכפו ואימתו ( ועוד יפתר ) לא אך הוא ישים בי כי לא ישים בי אך הריב של תוכחת פשעי ולא תשומות עלילות : 
! {ז}
<b>שם .</b> במקום ריבנו : <b>ישר .</b> כושר מפעלותי יהא נוכח ונגלה ונראה עמו לפניו : <b>נוכח .</b> מוכח ושם אפלטה לנצח כי לא ימצא בי עון : 
! {ח}
<b>הן קדם .</b> למעלה הוא חוזר למי יתן ידעתי וגו' : <b>קדם .</b> מזרח שנאמר קדמה מזרחה ( שמות כא ) : <b>אחור .</b> מערב הואיל והמזרח קדמת עולם נמצא המערב אחוריו של עולם : 
! {ט}
<b>שמאל .</b> צפון : <b>בעשותו .</b> כשבראו לא עשה מקום כסאו שם שאראנו שם : <b>אחז .</b> תרגום של אראה ותדע שהטעם תחת האל''ף ואם לשון אחיזה הוא היה טעמו מלרע תחת החי''ת . <b>יעטוף ימין</b> . כיסה פני הדרום בפניו למען לא אראנו :

<b>כי ידע דרך וגו' .</b> אשר עמדי מהו לפיכך אינו בא לריב עמדי : <b>בחנני וגו' .</b> אשר כזהב אצא אם יריב עמדי : 
! {יא}
<b>באשורו .</b> במעגלו כמו אם תטה אשורי ( לקמן לא ) כונן אשוריו ( תלים מ ) : <b>ולא אט .</b> איני רגיל לנטות : 
! {יב}
<b>מחוקי צפנתי אמרי פיו .</b> יותר מחוקי מזונותי הייתי זריז לצפון אמרי פיו כמו ( משלי ל ) לחם חוקי מזונותי : 
! {יג}
<b>והוא .</b> לפי שהוא יחיד בעולם ויודע דעתן ומחשבתן של בריות ומה יש להם לדבר לפניו לפיכך תשובות מוכנות לפניו : <b>ומי ישיבנו .</b> לפיכך נפשו אותה ויעש : 
! {יד}
<b>כי ישלים חקי .</b> ידעתי כי לא ישיב ידו עד אשר ישלים חוק גזירותיו פורעניות אשר גזר עלי : <b>וכהנה רבות עמו .</b> כי כהנה רבות עשה : 
! {טו}
<b>על כן .</b> שאינו גומל לאדם כפי דרכיו : <b>מפניו אבהל .</b> וכשאתבונן בדרכיו אפחד ממנו : 
! {יז}
<b>כי לא נצמתי מפני חשך .</b> בזאת אני נבהל כי לא נצמתי לחלוטין וצמיתות קבר מפני החשך הזה הבא עלי וכן לא כסה מפני את האופל הזה עד מותי : <b>ומפני כסה אופל .</b> על כי לא שבראש הפסוק חוזר : 

~ איוב פרק-כד

! {א}
<b>מדוע משדי לא נצפנו עתים .</b> קבלה הוא זה למה לא נצפנו עתי ימי האדם מן הקב''ה שלא ידע יום מות האדם שמא לא ימהר פורענות ויקדמנו יום המיתה : <b>ויודעיו לא חזו ימיו .</b> מיום האדם כמו חדש ימים ( ומתרגמינן עידן בעידן סא''א ) , ד''א מדוע יודעיו ואוהביו של הקב''ה אינן יכולין לראות סוף דרכיו ומדותיו : <b>ימיו .</b> זו היא דרך שנהג בימיו כמו לא יחרף לבבי מימי ( לקמן כז ) ומה הן המדות התמוהות : 
! {ב}
<b>גבולות ישיגו .</b> אותן שדרכן להסיג גבולות שהן קונין שדות כנגד שדות אחרים ובלילה הם מרבים גבוליהם שהיו עושים גדריהם בלילה בתוך תחום שכיניהם ( וזהו משיג דמתרגמינן משגה שהיו משגים הגבול סא''א ) : <b>עדר גזלו וירעו .</b> לעצמן : 
! {ג}
<b>חמור יתומים ינהגו .</b> ועושין כל רעות האמורות בענין , סוף דבר ואלוה לא ישים תפלה להודיעם עוונם ולהמציאם גמולם : 
! {ד}
<b>יחד חובאו .</b> עניי ארץ . מפני שיראים מהם : 
! {ה}
<b>הן פראים .</b> הפראים שהם לימודי מדבר לארוב את העוברים : <b>יצאו בפעלם .</b> יצאו למלאכתן ומה היא הפעולה משחרי לטרף מבקשים לטרוף : <b>ערבה לו לחם לנערים .</b> למשרתיו ונעריו של רשע הערבה והמדבר הווה לו לחם לצרכם שמשם הוא חוטף ואוכל וגוזל מה שהם אוכלים , לנערים הם ההולכים אתו לגזול כמו רק אשר אכלו הנערים ( בראשית יד ) : 
! {ו}
<b>בשדה .</b> של אחרים : <b>בלילו .</b> יבולו : <b>יקצורו .</b> לגזול : <b>וכרם רשע ילקשו .</b> וכרמים של אחרים הרשעים האלו נוטלים פריים , ילקשו אשפריימנ''ט נוטלים לקש שלהם , לקש לשון צמח פרי כמו בתחלת עלות הלקש ( עמוס ז ) תירגם יונתן צימוח וכן יסד ר''א ( בהושענא אדמה מארר ) לקש מארבה . ( ד''א וכרם רשע ילקשו . ילקשו אותה מהשחיתה כי את של העניים ויתומים הם אוכלים , ילקשו כמו ופיהם פערו למלקוש זמן ארוך וכמו לקישיא ללבן ) : 
! {ז}
<b>ערום ילינו .</b> הם מלינים את העניים : <b>מבלי לבוש .</b> שנטלו כסוין : 
! {ח}
<b>מזרם .</b> היורד מן ההרים ירטבו ויתלחלחו אותן הערומים לפי שמבלי כיסוי אין מחסה להחסות בו : <b>חבקו צור .</b> נחבאים בנקיקי הסלעים וגם שם הזרם יורד ומקלח עליהם ורוטבם , רטוב תרגום של לח וכל מי שאינו מנוגב קרוי לח : <b>ירטבו .</b> מויישט''א בלע''ז : 
! {ט}
<b>משד יתום .</b> יניקת היתום מקום מוצא מאכלו ומזונותיו כמו ושוד מלכים תינקי ( ישעיה ס ) : <b>יחבלו .</b> ל' אם חבול תחבול ( שמות כב ) וכמו חבול לא יחבול ( יחזקאל יח ) : 
! {י}
<b>הלכו .</b> הם הלכוהו בלי לבוש : <b>ורעבים נשאו עומר .</b> ומרעבים נטלו העומרים של לקט ועוד יפתר ורעבים אותם שהיו נושאים עומר לפי שאלו גוזלים אותם מהם והיו נשארים רעבים : 
! {יא}
<b>בין שורותם יצהירו .</b> ויעשו שמן , בעלי זתים : <b>ויקבים דרכו .</b> בעלי כרמים : <b>ויצמאו .</b> לאחר זמן לפי שאלו הרשעים גוזלים אותם מהם : <b>בין שורותם .</b> כמדומה לי שעושין הזתים שורות שורות בבית הבד והיצהר זב ביניהם : 
! {יב}
<b>מעיר מתים ינאקו .</b> מתיה וטוביה של עיר ינאקו מחמת אותן האונסין : <b>לא ישים תפלה .</b> להקפיד עליהם וליקח נקמה : 
! {יג}
<b>המה .</b> הרשעים האלה הם דור המבול שמרדו באור , בשביל המטר שהיה עולה מן הארץ ולא היו צריכים לכלום : 
! {יד}
<b>לאור יקום .</b> אם היו עומדים ביום היו הורגים עוברי דרכים : <b>ובלילה יהי כגנב .</b> לחתור בתים : 
! {טו}
<b>ועין נואף .</b> והנואף שבהם היה שומר וממתין עד נשף לאמר וגו' : <b>וסתר .</b> ובסתר ישים פניו לעבור עבירה , שעל כל אלה היו חשודין דור המבול על הגזל שנאמר ותמלא הארץ חמס ( בראשית ו' ) ועל העריות שנאמר כי השחית כל בשר ( שם ) : 
! {טז}
<b>יומם חתמו למו .</b> אותם שהיו חלשים מהם מעשו' לסטיו' ביום היו חותרים בבתים בלילה וביום היו חתומים ומסוגרים בבתיהם : <b>לא ידעו אור .</b> לא היו מכירין את האור ולא היה עת מהלכם : 
! {יז}
<b>כי יחדו בקר למו צלמות .</b> שביום היו נסגרים ובלילה היו הולכים למחתרתם ולא היו יראים מן המזיקים כלל : <b>כי יכיר בלהות צלמות .</b> מכיר היה כל א' שדים המבהילים ולא היה ירא מהם , ורבותינו פירשו יומם חתמו למו ביום היו עושים סימנים היכן מרתפו של בית והיכן גנזיו של עשיר שהיו מפקידין אפרסמון אצל בעלי בתים עשירים ויודעים הם שיניחנו בבית גנזיו ובלילה מריחין והולכים שם וחותרים שם : 
! {יח}
<b>קל הוא על פני מים .</b> כשהיו הצדיקים נח ומתושלח מוכיחים אותם ואומרים להם שעתיד המבול לבא עליהם היו אומרים קלים אנו לשיט על המים ויש לנו גומא ודברים קלים לעשות מהם אניות ולשוט : <b>תקולל חלקתם בארץ לא יפנה דרך כרמים .</b> היו מקללים את חלקם בארץ לאמר לא יפנה דרך הטוב בדרכי הצדיקים הרמים ושוכנים ברומו של עולם , ל''א כרמים הם הצדיקים ופרנסי הדור כמו ונתתי לה את כרמיה משם ( הושע ב ) ת''י ית פרנסהא היו מקללים את חלקם בארץ לאמר לא יפנו דרך הטוב נשבעים שלא יפנו את דרכם , ל''א תקולל חלקתם מתקלקלת היתה חלקתם שהיו משחיתים זרעם לבטלה כמו כי השחית כל בשר את דרכו ( ראשית ו ) תקולל לשון קלקול כמו בעון אשר ידע כי מקללים להם בניו ( שמואל א ג ) ל''א תקולל חלקתם הדורות שאחריהם מקללים בהם מי שפרע מדור המבול , כ''ש , ל''א תקולל חלקתם לא יפנה דרך כרמים , לא היו מקבלים תוכחה מן הטובים , לשון אחר היו קלים לשוט בספינות להרוג ולחמוס ממון ודרך יישוב בני אדם ונטיעת כרמים אין חושש לילך : 
! {יט}
<b>ציה גם חום יגזלו .</b> מפני רעתם גרמו להם שנה אחת להשתנות עולם מששה סדריו זרע וקציר וקור וחום וקיץ וחורף : <b>מימי שלג .</b> כמו ומימי שלג גם אותם גזלו מן העולם לפי ששאול חטאו יש תיבות חסירות ווי''ן כמו ( ירמיה יא ) ואני ככבש אלוף והוא נפתר ככבש ואלוף עוד הרבה במחברת מנחם בן סרוק : 
! {כ}
<b>ישכחהו רחם .</b> אמו , דרש רבי תנחום שהיו דור המבול נוטלים את בניהם ונותנים אותם תחתיהם לסתום ארובות התהום ואפילו אמו לא היתה מרחמת עליו : <b>מתקו רמה .</b> למות ולאכול רמה בשרם : <b>עוד לא יזכר .</b> בשנה אחת נעקרו ועל ארכא שניתן להם מתחילה היה איוב קובל שהרבה מתו ולא ראו בפורענות זה : 
! {כא}
<b>רועה עקרה לא תלד .</b> כך היה מנהגם נושא שתי נשים אחת לתשמיש ואחת לבנים זו שעומדת לתשמיש משקה כוס עיקרין כדי שלא תלד עולמית ומאכילה תפנוקים ומרחיצה ומקשטה כדי שתהא נאה וזו העומדת לבנים מלבישה בגדי אלמנות ומרעיבה וזהו שאומר העקרה שלא תלד היה רועה ומטיב ואת האלמנה שעומדת לבנים לא ייטיב : 
! {כב}
<b>ומשך .</b> הקב''ה אותן אבירים בכחו לפורענות ולמחייה מתוך ארכא שהאריך להם . יקום . יום א' : <b>ולא יאמין בחיין .</b> כמו בחיים ביום הזה נבקעו כל וגו' ( בראשית ז ) : 
! {כג}
<b>יתן לו לבטח .</b> עד יום אידם ונשענו : 
! {כד}
<b>רומו מעט .</b> נסתלקו מן העולם כמעט רגע : <b>והומכו .</b> נדכאו ונתמסכנו : <b>ככל יקפצון .</b> כשאר כל הנקפצים ונמלטים בעת קציר ובציר ומסיק שנקפצים יחד כן תמו כרגע : 
! {כה}
<b>ואם לא איפו .</b> כאשר אמרתי יבא אחד מכם ויכזיבני : <b>וישם לאל מלתי .</b> שאני מהרהר אחריו לדעת מדותיו שמאריך אף לרשעים , ואת אוהביו כמוני הוא מייסר ואינו ממיתן כמו שקובל ( למעלה כג ) כי לא נצמתי מפני חושך ואלו שגבולות ישיגו וכל אלה עשו לא הוכו בייסורין : 

~ איוב פרק-כה

! {א}
<b>( מענה בלדד )</b> : 
! {ב}
המשל ופחד עמו .</b> כלפי שאמר איוב אערכה לפניו משפט אמר לו המשל ופחד עמו , המשל זה מיכאל ופחד זה גבריאל כלום אתה יכול להשיב את אחד מהם : <b>עושה שלום במרומיו .</b> כשהמזלות עולים כל א' סובר אני עולה ראשון לפי שאינו רואה מה שלפניו לפיכך אינו מתקנא , כ''ש , ל''א עושה שלום במרומיו אש ומים בלולין ואין מים מכבין את האש , כ''ש : 
! {ד}
<b>ומה יצדק וגו' .</b> ילוד אשה . שאמרת בחנני כזהב אצא ( לעיל כג ) : 
! {ה}
<b>עד ירח .</b> כמו ולא עדה עליו שחל ( לקמן כח ) : <b>ולא יאהיל .</b> כמו לא יהלו אורם ( ישעיה יג ) ואל''ף זו באה כמו ולאדיב את נפשך ( שמואל א ב ) אל''ף יתירה , ל''א לא יאהיל לא יהיה לו צל כשאינו מאיר אין לו צל והראשון נראה ושניהם שמעתי : 

~ איוב פרק-כו

! {א}
<b>( מענה איוב ) : </b>
! {ד}
ונשמת מי יצאה ממך .</b> מי שם בך הרוח הזה שיצא מפיך מי לא ידע זאת : 
! {ה}
<b>הרפאים .</b> גיהנם המרפה את הבריות : <b>יחוללו .</b> שבעה חללים שלו : <b>מתחת מים ושוכניהם .</b> בעמקי עמקים , וגם הוא :

<b>ערום שאול נגדו .</b> לדעת ולראות את כל מה שבתוכו : 
! {ז}
<b>על בלימה .</b> אין כלום ביסוד כי הם עומדים באויר על חוזק זרועותיו של הקב''ה ודונש פי' בלימה מישור כמו ( תהלים לב ) במתג ורסן עדיו לבלום ( כך ראיתי סא''א ) : 
! {ח}
<b>ולא נבקע ענן .</b> מעול' שיפלו מימיו ביחד : <b>תחתם .</b> תחת המים : 
! {ט}
<b>מאחז .</b> במחיצות פני כסא הכבוד כולו כגון ישת חשך סתרו ( שם יח ) : <b>ופרשז .</b> על הכסא עננו כמו ודמות על ראשי החיות רקיע ( יחזקאל א ) : 
! {י}
<b>חוק חג .</b> קומפ''ש בלע''ז לשון ובמחוג' יתארהו ( ישעי' מד ) סיבבו בחול להיות לו ( החול ) חוק גבול עולם ולא יעבור הים אותו חוג עד שיכלה אור וחשך : 
! {יא}
<b>ירופפו .</b> כשבראם והקרישם היו רופפים וגער בהם ועמדו ביובש ובחוזק : <b>ויתמהו .</b> תוהה ועולה ( ס''א כאדם תוהה ועומד ) במקום אחד : 
! {יב}
<b>רגע הים .</b> כמו עורי רגע ( לעיל ז ) . נקמט קמטים קמטים נעשו ( כך כשאמר יקוו המים נאספו מפשיטתן ונקמטו קמטים קמטים ) עד שנקוו למקום אחד המוכן להם : <b>מחץ רהב .</b> מצריים קרויין רהבים : 
! {יג}
<b>ברוחו שמים שפרה .</b> בדבריו וברוח פיו איהל כמו ונטה אהל שפרירו ( ירמיה מב ) : <b>חוללה ידו .</b> את פרעה הנקרא נחש בריח ( ישעיה כז ) לשון חיל וצרה וכן במקום אחר הלא זאת היא המחצבת רהב מחוללת תנין ( שם נא ) , ל''א יסדה ידו את לויתן כמו גבעות חוללתי ( משלי ח ) : 
! {יד}
<b>קצות דרכיו .</b> קלות ומועטות לפי שאר גדולות שבו : <b>ומה שמץ דבר נשמע בו .</b> מה רמז דבר אדם יכול להבין מפעלותיו ד''א שמץ כמו לשמצה לגנות : 

~ איוב פרק-כז

! {א}
<b>ויוסף .</b> עמדו ריעיו מענות כאשר הוכיחם אליהוא כי עמדו ולא ענו עוד ( לקמן לב ) : 
! {ב}
<b>חי אל .</b> שבועה היא כי באמת הסיר את משפטי ומכאן דרש ר' יהושע שמאהבה עבד איוב את המקום אין אדם נודר בחיי המלך אלא א''כ אוהב את המלך : 
! {ה}
<b>אם אצדיק אתכם .</b> אם אומר הדין עמכם לחייבני : <b>לא אסיר תומתי .</b> לא אודה לדבריכם לומר שאיני תם : 
! {ו}
<b>בצדקתי החזקתי .</b> לומר שאני צדיק : <b>לא יחרף לבבי .</b> אותי מימי מכל דרך ומדה שנהגתי בימי ועל כרחו יהי כרשע אויבי : 
! {ח}
<b>כי מה תקות חנף .</b> כי למה אהיה רשע וגזלן מה סופו של רשע הגוזל ממון : <b>כי ישל .</b> כי ישליך המקום את נפשו מעליו לשון של נעלך ( שמות ג ) ונשל הברזל ( דברים יט ) : 
! {י}
<b>אם על שדי יתענג .</b> כפל התמיהה היא ככל התמיהות הכפולות שכן דרך הראשונה בה''א והשנייה באם כמו הירוצון בסלע סוסים אם יחרוש בבקרים ( עמוס ו ) , היש בלשוני עולה ואם חכי לא יבין הוות ( לעיל ו ) אף כאן הצעקתו ישמע אל אם על שדי יתענג , שמא או מי שהוא רשע חנף יכול הוא להתענג ולסמוך על ישועת הקב''ה ולקרא אליו בעת צרתו בתמיה : 
! {יא}
<b>אורה אתכם .</b> מה המדה שביד אל ודרך אשר עם שדי לא אכחד , כמו כי ידע דרך עמדי ( לעיל כג ) : 
! {יב}
<b>כולכם חזיתם .</b> שזה חלק אדם רשע : <b>ולמה זה הבל תהבלו .</b> להיות רשעים וחניפים : 
! {יד}
<b>אם ירבו בניו למו חרב .</b> ירבו וימותו ביסורים ותחלואים עד כי ישמחו אלמנותיו במיתתן ולא תבכינה : 
! {יז}
<b>( וכסף נקי .</b> כספו של רשע לנקי יחלקנו הקב''ה ) : 
! {יח}
<b>בנה כעש .</b> כתולע הבונה חור שאינו מתקיים וכסוכה שעשה נוצר התאנה והכרם שאינה של קיימא ( ע''א נוצר הכרם שאינו שומרה בכל עת זה שמעתי סא''א ) : 
! {יט}
<b>עשיר ישכב .</b> ימות בעשרו ופעמים שלא יאסף לקבורה : 
! {כ}
<b>בלהות .</b> שדים : <b>גנבתו סופה .</b> ותשליכהו לדראונים : 
! {כב}
<b>וישלך עליו .</b> המשליך עליו רעה ולא יחמול : <b>מידו .</b> של רשע ומכחו וממקומו : <b>ברוח יברח .</b> בן עצתו שהיה מסייעו בתחלה יברח מעזור ידו בשעת כשלונו כמו שעשה חרבונה להמן ( מגילה ז ) : 
! {כג}
<b>ישפוק עלימו כפימו .</b> היודעים ( אותו ) לפנים ועתה רואים בשברו : 

~ איוב פרק-כח

! {א}
<b>כי יש לכסף מוצא .</b> גם זה טעם אחר לדברו הראשון שאמר בצדקתי החזקתי כי למה אהיה רשע אם בשביל כסף וזהב לכל יש מוצא וסוף אבל החכמה מאין תבוא זה סוף הפרשה היא יקרה מכל , על כן כל ימי נתתי לבי עליה ללמוד : <b>ומקום .</b> יש לזהב שמשם יזוקו ויתכו אותו : 
! {ב}
<b>ואבן יצוק נחושה .</b> מן ההר יצק האומן את הנחושה הרי תחילה וסוף יש לכל דבר שכל העולם יכלה וכל אשר בו : 
! {ג}
<b>קץ שם .</b> המקום מתי תהיה החושך שהשמים נמלחו והארץ תבלה ולכל תכלית דבר הוא חוקר לו : <b>אבן .</b> אשר משם יצא לו חשך וצלמות אבן מקום מוצא פורענו' לכן נקרא אבן במקום פורענות כאדם הנוגף את רגליו באבן שיש לו צער , כ''ש , כמו קו תוהו ואבני בוהו , ( ישעיה לד ) דמן טורא אתגזרת אבן וגומר ( דניאל ד ) : 
! {ד}
<b>פרץ נחל .</b> על סדום ועמורה נחלי אש וגפרית : <b>מעם גר .</b> ממקום שהוא נובע ונוגר : <b>הנשכחים מני רגל .</b> אלו אנשי סדום ששיכחו תורת רגלי אורחים מארצם שהיו משכיבים האורח במטה אם היה ארוך יותר מקטעין רגליו ואם היה קצר מאריכין אותו כך מפורש באגדה : <b>דלו .</b> ע''י פריצת נחל זה ומאנוש נעו : 
! {ה}
<b>ארץ .</b> אשר ממנה יצא לחם . נהפך מקומה כמו אש : 
! {ו}
<b>מקום ספיר .</b> היו אבניה : 
! {ז}
<b>לא ידעו עיט .</b> לא עבר עליהן חיל של ליסטין : <b>ולא שזפתו עין איה .</b> לפי שהיא רואה יותר משאר עופות ולכך נקראה ראה שהיא עומדת בבבל ורואה נבילות בארץ ישראל כך מפורש באגדת ( חולין סג ע''א ) כלומר לא חפרוה מרגלים : 
! {ח}
<b>לא עדה עליו .</b> עבר כדמתרגמינן אשר עבר בין הבתרים ( בראשית טו ) ( צ''ל הגזרים וע' בתרגום ) דעדא : <b>שחץ ושחל .</b> משבעה שמות האריה הם ( בסנהדרין צה ) ששה שמות יש לאריה , והקב''ה מה עשה : 
! {ט}
<b>בחלמיש שלח ידו .</b> והפך שרשי ההרים מלמעלה למטה : 
! {י}
<b>בצורות יאורים בקע .</b> חוזר לתחילת ענינו שאמר כי לכל דבר יש מוצא , ובצורות בקע למוצא היאורים וכן : <b>כל יקר ראתה עינו .</b> מהיכן יצא לבריות ( כמו שעשה לישראל הך סלע ויזובו מימיו ) : 
! {יא}
<b>מבכי נהרות חבש .</b> בבריאת העולם כשבכו מים התחתונים , בכי כמו נבכי ים ( איוב לט ) : <b>חבש .</b> כמו ויחבש את חמורו ( בראשית כב ) כ''ש , ( ס''א מבכי וגו' מפני בכייתן של ישראל שהיו בוכין לאמר מה נשתה חבש ותיקן להם נהרי מים מן הסלע ויחבש להם מתרגם ואתקין ) : <b>ותעלומה יוציא אור .</b> חלון יש ברקיע ושמו תעלומה משם תולדות השמש לתקופתו ( בפרקי ר' אליעזר ) כלומר לכל דבר יש מוצא וסוף רק לחכמה לא מוצא לה אלא מפיו ולא סוף עולמות : 
! {יב}
<b>והחכמה .</b> התורה : 
! {יג}
<b>ולא תמצא בארץ החיים .</b> במי שמחיה אלא במי שממית עצמו עליה ביגיעה וברעבון : 
! {יד}
<b>תהום אמר לא בי היא .</b> אם תשאלנה ליורדי תהום העוסקים במוצא מרגליות ובמוצא זהב וכסף ממקום תהומות הארץ יאמרו לך לא בי היא , איני בקי בהוראה ( שבת פט ע''א , ע''ש היטב וצ''ע ) : <b>וים אמר .</b> ויורדי הים לסחורה יאמרו לך לא עמדי חכמה למה לפי שאינם יכולים לקנותה כשאר פרקמטיא לתת מעושרם בדמיה , ובאגדה הוא אומר שבשעת מתן תורה בא השטן לפני הקב''ה ואמר לו היכן תורה א''ל אצל בן עמרם , בא לו אצל משה אמר לו היכן תורה אצל ים בא לו אצל ים אמר לו היכן תורה לא בי היא לך אצל בן עמרם בא לו אצל בן עמרם אמר לו היכן תורה אצל תהום בא לו אצל תהום אמר לו היכן תורה א''ל אין עמדי לך אצל בן עמרם בא לו אצל בן עמרם אמר לו היכן תורה אצל הקב''ה וזהו שאמר תהום אמר לא בי היא וים אמר אין עמדי : 
! {טו}
<b>לא יותן סגור .</b> זהב סגור זהו זהב טוב שבשעה שנפתח כל חנויות של זהב נסגרות : 
! {טז}
<b>לא תסולה .</b> לא תשתבח לשון סולו לרוכב בערבות ( תלים סח ) המשבחה לומר חשובה היא ככתם אופיר וכשוהם יקר לא שבח הוא לה וכן המסולאים בפז ( איכה ד ) : 
! {יז}
<b>זהב .</b> טהור מנהיר כמרגלית : <b>ותמורתה וגו' .</b> ותמורתה לא בכלי פז : 
! {יח}
<b>ראמות וגביש .</b> שמות אבנים טובות שבים הם : <b>לא יזכר .</b> אצלה : 
! {יט}
<b>פטדת כוש .</b> אבן טובה היא כמו אודם פטדה , כתם היא קבוצת עדי לתכשיטי נשים ונקראת הקבוצה כתם בלשון עברי ובלשון ערבי אלחלו והוא נזם זהב וחלי כתם ( משלי כה ) וכן ותעד נזמה וחלייתה ( הושע ב ) ויש פותרים כתם כתר ולא נכון כי לא מצינו פעולה בלשון זה כאשר מצינו בלשון כתר מכתיר כותרות : 
! {כא}
<b>ומעוף השמים .</b> מלאכים המעופפים : 
! {כב}
<b>אבדון ומות אמרו .</b> המאבדין וממיתין עצמן עליה אמרו באזנינו שמענו שמעה שהמיגע בה מתקיימת בו , וי''א כאשר באגדה אשר פירשתי למעל' שבא השטן לפני המקום אמר לו היכן תורה א''ל וכו' והיינו דכתיב אבדון ומות אמרו באזנינו שמענו שמעה כשנתנה לישראל , אבל : 
! {כג}
<b>אלהים הבין דרכה והוא ידע את מקומה .</b> שיודע הוא היכן שורה וגם ישבחוה ויאמרו לה כי אלהים הבין דרכה נסתכל בה וברא את העולם באותיותי' כסדרם ומשקלן יצר כל היצורים כאשר כתוב בסוד ספר יצירה : 
! {כד}
<b>כי הוא לקצות הארץ יביט .</b> בה היאך יבנם ויבראם ותחת כל השמים צרכי ברייתן ראה בה : 
! {כה}
<b>לעשות וגו' .</b> בה משקל לרוח , לכל ארץ לפי כחה וכן המים לפי המדה לפי מה שהיתה ארץ צריכה להשקותה יש ארץ נגובה וצריכה הרבה גשמים ויש ארץ שאינה צריכה כל כך : 
! {כו}
<b>חק .</b> גזירה קצובה לפי כל ארצות ומעיין ופלג להשקות כל ארץ וארץ ובעשותו דרך לכל החזיזים שלא יצאו שני קולות דרך נקב אחד פן יחריב את העולם : 
! {כז}
<b>אז ראה .</b> ראה אותה נסתכל בה ובעצתה יעשה הכל : <b>ויספרה .</b> ספר אותיותיה כפולו' ופשוטות ראשונ' ואמצעית ואחרונית היא אמת חותמו של הקב''ה וכן בשאר סדרים ברא כל דבר ודבר באותיות הללו והכל מפורש בסוד ספר יצירה : <b>הכינה .</b> זימנה ליצירה : 
! {כח}
<b>הן יראת אד' היא חכמה .</b> זו צריכה לזו ואין חכמה יפה בלא יראה : 

~ איוב פרק-כט

! {א}
<b>( מענה איוב ) : </b>
! {ב}
מי יתנני כירחי קדם .</b> כימים קדמונים שהייתי בגדולתי : 
! {ג}
<b>בהלו נרו .</b> לשון אור ושמחה : <b>לאורו .</b> הייתי הולך בכל חשך וצרה שהיתה באה לעולם לי היה אור : 
! {ד}
<b>בימי חרפי .</b> בימי קדמותי לשון ארמי הוא חרפי ואפלי : <b>בסוד אלוה עלי אהלי .</b> כשהיו כשרי הדור באים לאהלי להווסד בדברי הקב''ה בתורה ובגדר סייג לשעה : 
! {ה}
<b>נערי .</b> משרתי : 
! {ו}
<b>הליכי .</b> רגלי : <b>בחמה .</b> כמו בחמאה : <b>וצור יצוק עמדי פלגי שמן .</b> כלומר נחת רוח וכל תאות לב הנוחים כנחלי שמן : <b>יצוק .</b> כמו ויצקת : 
! {ז}
<b>קרת .</b> תקרה גבוה מוכנת ככסא לשבת לזקני העיר : <b>ברחוב וגו' .</b> וכאשר ברחוב הייתי מכין מושבי : 
! {ח}
<b>היו נערים .</b> אים אותי ונחבאים וישישים עמדו מפני : 
! {ט}
<b>שרים עצרו במילים .</b> ומצפים שאפתח אני תחלה : 
! {יא}
<b>כי אזן .</b> אשר שמעה מלתי ותאשרני : <b>ועין ראתה .</b> בכל מקום בגודלי וכושר מידתי : 
! {יג}
<b>ברכת אובד .</b> אדם מת בעונו ונטרד : <b>עלי תבא .</b> כי הייתי זן את בניו ואת אשתו ורבותינו אמרו שהיה גוזל שדה יתומים ומשביחה ומחזירה להם ופשטיה דקרא אינו נראה כך שאין קרוי אובד אלא דבר שהוא עצמו אובד כמו ככלי אובד ( תהלים לא ) וכן תעיתי כשה אובד ( שם קיט ) צאן אובדות ( ירמיה כ ) : <b>ולב אלמנה ארנין .</b> שהיה מטיל שמו על אלמנות לאמר קרובתי היא או אני אשיאנה כדי שיקפצו עליה תובעים : 
! {יד}
<b>צדק לבשתי וילבשני .</b> רדפתי אחרי הצדק והוא נמצא לי כמעיל וצניף שהם תכשיטים נאים כן היה משפטי נאה וטהור : 
! {יח}
<b>ואומר עם קני אגוע .</b> בשביל כושר מדתי הייתי אומר בלבי כשאמות יהיה קני וביתי נכון עמי : <b>וכחול ארבה ימים .</b> עוף ושמו חול ולא נקנסה עליו מיתה שלא טעם מעץ הדעת ולבסוף אלף שנה מתחד' וחוזר לנערותו : 
! {יט}
<b>שרשי פתוח וגו' .</b> כל זה הייתי אומר בלבי שלא יבש שרשי ולא ימות קצירי כמו ועשה קציר ( לעיל יד ) והן יונקות ענפים רכים של אילן , ואמרתי : 
! {כ}
<b>שכבודי .</b> יתחדש עמדי . תמיד : <b>וקשתי .</b> ואיתני תחליף תמיד לטובה : 
! {כא}
<b>לי שמעו ויחלו .</b> השומעים דברי היו מיחלים שיתקיימו והיו מתקיימות כי מסורת חכמה היה בידי : 
! {כב}
<b>תטוף מלתי .</b> ל' נבואה וכן לא תטיף על בית ישחק ( עמוס ז ) כלומר כנבואה היה דברי : 
! {כג}
<b>למלקוש .</b> לזמן ארוך היו מדברים כי רוב שעותיהם שתיקה שהיו תאיבים לשמוע דברי : 
! {כד}
<b>לא יאמינו .</b> שאני משחק עמהם מפני חשיבותי : <b>ואור פני לא יפילון .</b> היו יראים להתקרב אלי ולהגיס לבם בי : 
! {כה}
<b>אבחר דרכם .</b> כל דרכיהם ועצתם אני הייתי בוחר להם שהיו שואלים לי איזה דרך נלך ומה נעשה : <b>ואשב ראש .</b> אם היו מסובים אני הייתי מסב בראש המסובים : 

~ איוב פרק-ל

! {ב}
<b>גם כח ידיהם למה .</b> היה לי באותן הימים רשעים היו ולא תועלת בהם כי פורענות בא על ידם : <b>עלימו אבד כלח .</b> זקנה : 
! {ג}
<b>בחסר ובכפן .</b> היו גלמוד : <b>העורקים ציה .</b> שמבריחין אותן בימיו מקרב בני אדם לילך בחושך : <b>אמש שואה ומשואה .</b> ברואינ''א בלע''ז : 
! {ד}
<b>הקוטפים מלוח עלי שיח .</b> בהיותם במדברות היו קוטפים להם מלוחים על אילנות היערים ואכולי' , מלוח שם עשב הוא ובל' ארמי נקרא קקו''לין ובל' משנה מלוחים מצו''ש בלע''ז כאותה ששנינו ( קדושין סו ע''א ) העלו מלוחים על שלחנות של זהב : <b>שיח .</b> אילן כמו אחד השיחים ( בראשית כא ) : 
! {ה}
<b>מן גו .</b> מן קרב עיר היו נגרשין : 
! {ו}
<b>בערוץ נחלים .</b> יש ארץ מלוחה מתבקעת ששם היו שוכבים בחורי העפר וחורי נקיקי הסלעים : <b>וכפים .</b> תרגום של סלעים . ( ד''א בערוץ נחלים . הוא הנקיק שעושה הנחל סביבותיו ) : 
! {ז}
<b>חרול יסופחו .</b> אורשיא''ש בלע''ז היו נאספים כמו ונספחו על בית יעקב ( ישעיה יד ) : 
! {ח}
<b>נכאו .</b> בימי מן הארץ שהייתי משפילם ל' נכה רוח : 
! {יא}
<b>כי יתרי .</b> פתח . יתר קשתי פתח ( כמו שאמרתי למעלה וקשתי תחליף ) כלומר כחי התיש מי שבידו לעשות : <b>ורסן מפני שלחו .</b> רסני שהיו מרוסנים בו שלחוהו מפני כי אינם יראים מעתה ממני : 
! {יב}
<b>על ימין .</b> על ימיני עומדים פרחים קטנים כשאר הדיוטים שאיני חשוב בעיניהם להכנע מפני : <b>רגלי שלחו .</b> אם המקום צר להם דוחפים אותי הלאה : <b>ויסולו עלי אורחות אידם .</b> כל דרך נבלה שיש להם לעשות עושים מסילתן לפני ואצלי ולא ייראו : 
! {יג}
<b>נתסו נתיבתי .</b> הטוב' ואין חוששין לה , נתסו כמו נתצו וסמ''ך מתחלפ' בצד''י : <b>להוותי יועילו .</b> להקניטני מוסיפין : 
! {יד}
<b>כפרץ רחב .</b> שבגדר שהכל באים ועוברים שם כן יאתיו יחד עלי להכלימני : <b>תחת שואה התגלגלו .</b> כלומר מתגלגלים ובאין בסתר עד שמגיעין אצלי להקניטני כדי שלא אשמט מהם : 
! {טו}
<b>ההפך עלי .</b> נהפך עלי : <b>בלהות .</b> שדים : <b>תרדוף כרוח .</b> רעה : <b>את נדיבתי .</b> רוח נדיבה שהיה שורה עלי מתחלה : 
! {טז}
<b>ועתה .</b> אל הצרה הזאת תשתפך נפשי כאדם שתתפעם רוחו כי יאחזוני וכופין אותי להשתפל נפשי : <b>ימי עוני .</b> שאני בהם : 
! {יז}
<b>ניקר .</b> כמו העיני האנשי' ההם תנקר ( מדבר טז ) ניקר פוריי''ר בלע''ז התולעים מנקרין בשרי מעל עצמי : <b>ועורקי .</b> גידי אין להם מנוח' גיד בלשון ערבי עורק , כך פי' דונש לשון אחר עורקי רודפי המבריחין אותי : 
! {יח}
<b>יתחפש לבושי .</b> משתנה לבושי גלד אחר גלד כשאדם פושט את בגדיו ולובש בגדים אחרים משונים מראשונים קורא אותם בגדי חופש כמו ( מלכים א כב ) התחפש ובא במלחמ' ומתורגם אישתני : <b>כפי כתנתי יאזרני .</b> שאוזר ומעניק צוארי כן יאזרני אותי לבושי כל סביבי : 
! {יט}
<b>הורני לחומר .</b> הורני ילמדני השחין הזה לישב כמו בחומר , כמו שאמרנו ( למעלה ב ) והוא יושב בתוך האפר ד''א הורני לחומר פי' השליכני לשון ירה בים ( שמות טו ) : <b>ואתמשל .</b> נדמיתי ובמדרש רבי תנחומא אמר ר' ברכיה סרס את המקרא אני נמשלתי לאברהם בצדקתי שקרא עצמו עפר ואפר ( בראשית יא ) והוא דנני כרשעי דור הפלגה ( שמרדו בו בבנין המגדל ) שכתוב בהם והחמר היה להם לחומר ( שם יא ) : 
! {כ}
<b>עמדתי .</b> לראות לשתוק ולהבין כמו ובפתחו עמדו כל העם ולא ענו עוד ( נחמיה ח ) : <b>ותתבונן בי .</b> מחשב עלי לשנות את נגעי : 
! {כב}
<b>תשאני אל רוח .</b> שדים : <b>תרכיבני ותמוגגני תושיה .</b> חלשות ותשות כח היה ממוגגני : <b>ותמוגגני תושיה .</b> לשון נקבה ועל תושיה עומד ואין תי''ו זו כתי''ו של תשאני : 
! {כג}
<b>כי ידעתי .</b> כי יודע הייתי אשר למות תשיבני , מקום שהוא מועד לכל חי בקבר : 
! {כד}
<b>אך לא בעי .</b> ובהשחת' היה רגיל השופט לשלוח יד כמו לעי השדה : <b>אם בפידו .</b> אם היה שולח פיד שהוא שבר על בריותיו : <b>להן שוע .</b> היה משעשען תנחומין במקצת וכן פירשו רבותינו אין הקב''ה מכה אומה ומושיב' צדו אלא מביא פיד לזו ומשעשע' בחברתה שלקתה כבר ומתנחמת בה הביא פיד לאשור ושעשע במצרי' שנא' התיטבי מנא אמון ( נחום ג ) הביא פיד למצרי' ושעשע באשור שנא' הנה אשור בלבנון ( יחזק' לא ) ועוד יפתר להן שוע תפלה ותחינה מועלת בהם : 
! {כה}
<b>אם לא בכיתי לקשה יום .</b> יודע ומכיר אם לא הייתי רחמני ובוכה על עניים קשי יום ועגמה נפשי לאביונים כי על כל אלה באתני זאת : 
! {כו}
<b>כי טוב קויתי וגו' .</b> כלומר כי הגמול הזה ראוי לי : 
! {כז}
<b>ולא דמו .</b> אין יכולין לידום ולהחריש : 
! {כח}
<b>בלא חמה .</b> לא שזפתני השמש והנני קודר : 
! {כט}
<b>אח הייתי לתנים וגו' .</b> תנים ובנות יענה עסוקים בבכי כל ימיה' : 
! {ל}
<b>חרה .</b> לשון דבר הנחרה ומתייבש מפני חום האש כמו נחר מפוח ( ירמיה ו ) ותוכו ניחר ( יחזקאל טו ) : 

~ איוב פרק-לא

! {א}
<b>ברית כרתי לעיני .</b> שלא להסתכל באשת איש : <b>ומה אתבונן על בתולה .</b> מה לי להתבונן בה כך היתה חסידותי שלא לתת עין אפילו בפנויה שמא לאחר ימים תנשא ונמצאתי כרוך אחריה כך מפורש באבות דרבי נתן : 
! {ב}
<b>ומה חלק וגו' .</b> שגמלני כך : 
! {ג}
<b>הלא איד לעול .</b> לו ראוי ולא לי : <b>ונכר .</b> דוגמת ואל תרא ביום אחיך ביום נכרו ( עובדיה א ) : 
! {ד}
<b>וכל צעדי יספור .</b> ורואה אם הלכתי עם שוא : 
! {ה}
<b>ותחש .</b> ותמהר כמו אחישנה ( ישעיה ס ) כי גז חיש ( תלים צ ) : 
! {ז}
<b>מאום .</b> הוא מום דבר שמץ ואל''ף זו כתובה ולא קריא במסורת הגדולה והוא מגזירת לא תאסיפון אם זו עשיתי הייתי ראוי לבא עלי קלל' זו : 
! {ח}
<b>אזרעה ואחר יאכל וצאצאי ישורשו .</b> יעקרו השרשין : 
! {ט}
<b>ועל פתח רעי ארבתי .</b> לאשתו ישיב לי גמולי : 
! {י}
<b>תטחן לאחר אשתי .</b> רבותינו פירשוהו לשון תשמיש כמו ויהי טוחן בבית האסורין ( שופטים ז ) : 
! {יא}
<b>כי היא .</b> זימה . וירא הייתי אם עשיתי שהוא עון פלילי , עון שמשפט יסורין ראויין לבא על ידו : 
! {יב}
<b>ובכל תבואתי תשרש .</b> האש הזאת לכן נמנעתי מעשות כשהוא מדבר בל' עיקור אומר תשרש ובל' שרש אומר תשרש וכן ישורשו ל' עיקור ישרשו ל' שרש : 
! {יד}
<b>ומה אעשה וגו' .</b> זאת נתתי אל לבי וכי יפקוד את דרכי מה אשיבנו : 
! {טו}
<b>ויכוננו ברחם .</b> אני בבטן אמי והוא בבטן אמו הכיננו יוצר א' : 
! {טז}
<b>אכלה .</b> שלא מלאתי תאותה שהוא כליון עינים המתאוה ואינו מגיע קרוי כליון עינים : 
! {יח}
<b>כי מנעורי גדלני .</b> הכשרון הזה כאב המגדלני והמלמדני כן למדנו יושר לבי מנעורי : <b>ומבטן אמי אנחנה .</b> למדה זו : 
! {כב}
<b>משכמה .</b> הוא רחב הכתף : <b>מקנה .</b> הוא עצם העליון העגול כקנה ועל שהוא עגול כקנה קורא אותו כן : 
! {כג}
<b>כי פחד אלי וגו' .</b> פחד היה בעיני אידו שהוא מביא על הרשעים לכן חדלתי מעשות : <b>ומשאתו .</b> ומכובדו : 
! {כד}
<b>כסלי .</b> כמו אשר יקוט כסלו ( לעיל ח ) תקותי ומחשבתי וע''י שכליות על הכסלי' הם והכליות יועצות נקראת המחשבה כסל : <b>ולכתם .</b> קבוצות זהב ואוצרות : <b>אמרתי מבטחי .</b> להתחזק בהם על העניים : 
! {כה}
<b>אם אשמח כי רב חילי .</b> לא הייתי נוהג שמחה בפני העניים בעשרי שלא להקניאם בי : 
! {כו}
<b>אם אראה אור כי יהל .</b> חמה בתקופת גבורתה והילה : <b>וירח יקר הולך .</b> מגיה והולך כמו אור יקרות ( זכריה יד ) כיקר כרים ( תלים לז ) : 
! {כז}
<b>ויפת בסתר לבי .</b> לשמש ולירח לאמר אלוהות הם כקצת עובדי כוכבים הנדחים ( ועובדים ) לכל צבא השמים : 
! {ל}
<b>ולא נתתי לחטוא חכי .</b> להקניט שונאי ולתובעו ממון שאין לי עליו כדי להשביעו ולשאול באלה נפשו של שונאי : 
! {לא}
<b>לא נשבע .</b> מתוך שנאה שהיו שונאים אותי על שהייתי מטריחם בהכנסת אורחים : 
! {לב}
<b>לארח .</b> לא היה דלתות ביתי פנימיים ומוקפין בית שער מבחוץ או מוסבין בקרני זוית להשבית את האורחים מלבא אלא פתוחין לאם דרך לבוא כל הבא : 
! {לג}
<b>בחובי .</b> בנחבואי : 
! {לד}
<b>כי אערוץ .</b> מתחילה הייתי מכניע המון רבה של רשעים ועתה אף בין כשדיים זה העם לא היה ( ישעיה כג ) : <b>יחיתני .</b> מיראני : 
! {לה}
<b>הן תוי שדי יענני .</b> ולואי שיעיד עלי המקום בדין כמ''ש עלי כי אין כמוהו בארץ תם וישר וירא אלהים ( לעיל א ) גם משה שכתב ספרו וספרי יעיד עלי , ( ד''א ) כן היה איוב אומר תנה לתוות אותי ואת כושרי : <b>שדי יענני .</b> לבקשה זאת : <b>וספר כתב איש ריבי .</b> יכתוב גם הוא ספרו להתווכח עמי : 
! {לו}
<b>אשאנו .</b> לכתוב את ספרי : <b>אענדנו .</b> אקשרנו לספר עטרות לי : 
! {לז}
<b>מספר צעדי אגידנו .</b> לאותו האיש : 
! {לח}
<b>אם עלי אדמתי תזעק .</b> על לקט שכחה ופאה ומעשרות שלא הוצאתי מעשרות כראוי : <b>ויחד תלמיה יבכיון .</b> שחרשתי בשור ובחמור יחדיו : 
! {לט}
<b>בלי כסף .</b> לשכור שכירים ולעשקם ד''א זה מעשר שני דכתיב וצרת הכסף בידך ( דברים יד ) : <b>ונפש בעליה .</b> אריס העובד בה : 
! {מ}
<b>באשה .</b> ( דבר מבאיש ) כמו באושים ( ישעיה ה ) ואם לא עשיתי כן תחת חטה יצא חוח ( תנא רבי אושעיא למדתך תורה דרך ארץ שדה שמעלה חוחים יפה לזורעה חטים , שמעלה באושים יפה לזורעה שעורים כ''ה בילקוט ) ומה טעם תחת חטה יצא חוח : <b>תמו וגו' .</b> עד כאן הוא חוזר ומתנבא כמה נבואות ואת אומר תמו דברי איוב אלא כך אומר איוב אם לא עשיתי כן יתמו דברי איוב ואל יהי פתחון פה לומר לפניך בערתי הקדש ( דברים כ ) כך ראיתי בפסיקת' בפרשת עשר תעשר ( שם יד ) מפורש כן מן הפסוקים האלו : 

~ איוב פרק-לב

! {א}
<b>( מענה אליהוא ) : </b>
! {ב}
ממשפחת רם .</b> אברהם שנאמר האדם הגדול בענקים ( יהושע יד ) זה אברהם : <b>מאלהים .</b> יותר מן המקום : 
! {ג}
<b>וירשיעו את איוב .</b> זה אחד מן המקראות שתקנו סופרים את לשון הכתוב וירשיעו כלפי המקום בשתיקותם היה לו לכתוב אלא שכינה הכתוב וכן וימירו את כבודם בתבנית שור כבודי היה לו לכתוב אלא שכינה הכתוב וכן ואל אראה ברעתי ( במדבר יא ) ברעת' היה לו לכתוב אלא שכינה הכתוב וכן הרבה מקומות בספרי ובמסורת הגדולה : 
! {ד}
<b>חכה .</b> המתין : <b>כי זקנים המה ממנו לימים .</b> האחרים לכן חדל לו עד ששתקו הם , ( מענה אליהו ) : 
! {ו}
<b>זחלתי .</b> יראתי : 
! {ח}
<b>אכן רוח היא באנוש .</b> החכמה ולא ע''י ימים וזקנה אלא רוח המקום הוא : 
! {יב}
<b>ועדיכם אתבונן .</b> התבוננתי בכם וראיתי כי אין מוכיח לאיוב : 
! {יג}
<b>פן תאמרו מצאנו חכמה .</b> בשתיקותינו שלא להקניטו עוד . כי אל ידפנו ולא איש . לא היה לכם למעט כבוד המקום בכך : 
! {יד}
<b>ולא ערך אלי מלין .</b> קבלה היא או שלא ערך אלי מלין : <b>ובאמריכם לא אשיבנו .</b> לא באותן אמרים שהשיבותם לו הייתי משיבו : 
! {טו}
<b>העתיקו .</b> סרו מהן מלין : 
! {יט}
<b>הנה בטני .</b> מלא רוח , דברים להשיב בתאוה : <b>כיין לא יפתח .</b> כיין חדש שאינו נפתח והוא נופח ואף בתוך אובות חדשים ומבקעם : <b>אובות .</b> נודות עשויין כקומת אדם גבוהים : 
! {כא}
<b>לא אכנה .</b> לא אשנה דברי להחליפו לכבודו כמו שאמרתי למעלה וירשיעו את איוב זהו כינוי שמשנה את הדיבור בשביל כבודו ואף זאת לא אעשה כי בפה מלא אוכיחנו : 
! {כב}
<b>כמעט ישאני .</b> יסלקני מן העולם ויש פותרים ישאני כמו ותשא הארץ מפניו ( נחום א ) : 

~ איוב פרק-לג

! {ד}
<b>רוח אל עשתני .</b> ואיני חזק יותר ממך : 
! {ו}
<b>הן אני כפיך לאל .</b> כמו ששאל פיך להתווכח למי שלא יבעתך הן אני במקומו של הקב''ה ובשבילו לדבר דבריו : 
! {ז}
<b>ואכפי .</b> כפייתי כמו אכף עליו פיהו ( משלי יז ) : 
! {ט}
<b>חף .</b> טהור ומשופשף כמו נזיר לא יחוף ראשו וכמו בכל חפין את הכלים : 
! {י}
<b>תנואות .</b> עלילות ל' תואנה ( שופטים יד ) לשוב מן המקום : 
! {יב}
<b>כי ירבה אלוה מאנוש .</b> אין המקום צריך כל כך לדקדק אחר האדם כי הרבה יש להקב''ה צדקות יותר מאנוש : 
! {טז}
<b>אז יגלה אוזן אנשים .</b> כמו שעשה לאבימלך בחלום הלילה : <b>ובמוסרם יחתום .</b> חותמן ואוסרן ביסורין על עוונם : 
! {יז}
<b>להסיר אדם .</b> בחליו : <b>מעשה .</b> שהיה בדעתו לעשות : <b>וגוה מגבר יכסה .</b> ביסוריו למען הכניעו ולהשפילו לשוב כדי שיחשך נפשו ממיתה גמורה : <b>גוה .</b> כמו גאוה : 
! {יח}
<b>וחיתו מעבור בשלח .</b> בשלחו וחרבו של מלאך המות כלומר לטובתך הביא עליך יסורין : <b>שלח .</b> כלי זיין וכן ( נחמיה ד ) ידו אחת מחזקת השלח : 
! {יט}
<b>והוכח .</b> ל' תוכחה שישוב מעונו : <b>ורוב עצמיו .</b> שהוא איתן מוכיחן במכאוב : 
! {כ}
<b>וזהמתו חיתו לחם .</b> ממאסתו נפשו וחייתו של חולה כל מאכל : <b>ונפשו .</b> מזהמת כל מאכל תאוה : 
! {כא}
<b>יכל בשרו מרואי .</b> יחשך בשרו ממראה כמו כלו בדמעות עיני ( איכה ב ) מכלות עינים ( ויקרא כו ) : <b>ושופו עצמותיו .</b> מתנתקין ממקומו , ושופו כמו דשף מדוכתיה ( חולין מב ) : <b>לא רואו .</b> אין להם מראה ותואר : 
! {כג}
<b>אחד מני אלף .</b> במגידי פשעו והוא יליץ ישרו : 
! {כד}
<b>ויחוננו .</b> המקום : <b>פדעהו .</b> כמו פדאהו אל''ף מתחלפ' בעיי''ן והענין מוכיח מצאתי כופר והוא פדיון : 
! {כה}
<b>רטפש בשרו .</b> רוטפש אשקוש''א בלע''ז : <b>מנוער .</b> נתעורר בשרו מן הנוער שהוא מנוער ומשונק ( בייסורי ) חליו ואין לתיבה זו דמיון במקרא ולפי העניינים הוצרכנו לפרשו כמשמעות הענין והפותרו בלשון רפש ואומר הטי''ת נתוספ' בו כמו ( בראשית מד ) נצטדק טועה הוא מאוד כי אין טי''ת מתוספת בתיבה כי אם בלשון מתפעל או נתפעל ובתיבה שיסודה צד''י ולא בשאר תיבות כמו צדק ונצטדק וכמו צרוע ( ויקרא יג ) נצטרע , ציר נאמן , וילכו ויצטיירו , צדה הצטיידנו ( יהושע ט ) : <b>ישוב .</b> בכחו וברפואה לימי בחרותו : 
! {כו}
<b>בתרועה .</b> בתפילה : 
! {כז}
<b>ישור על אנשים .</b> יעשה שורות של אנשים כשניצול מחליו ויתודה ליוצרו : <b>ולא שוה לי .</b> לא נהיה לי שכר בדבריו : 
! {כט}
<b>פעמים שלש .</b> מייסרו על עונותיו בחליו שלא להשחיתו ואם יותר יכעיס לפניו ידאג מן גיהנם ומן המיתה שנאמר ועל ארבעה לא אשיבנו ( עמוס א ) : 

~ איוב פרק-לד

! {א}
<b>( מענה אליהו ) : </b>
! {ג}
כי אוזן מלין תבחן .</b> כאשר החיך יטעום האוכל : 
! {ה}
<b>משפטי .</b> הוכחת דברים : 
! {ו}
<b>אכזב .</b> אכזיב שופטי שלא דנני אמת : <b>אנוש .</b> חולה מכתי כמו ויאנש ( שמואל ב יב ) : <b>חצי .</b> נגעי : 
! {ט}
<b>לא יסכן גבר .</b> לא יהנה אדם אם מתמים דרכיו : 
! {יג}
<b>מי פקד עליו ארצה .</b> מי מושל בו לאמר למה כן עשית ולא צויתיך כן כי מי צוהו בממשלתו ולמה ימצא תואנות לבריות אינו צריך להשיב דבר למושל בו לאמר בשביל זאת הרעותי לו : 
! {יד}
<b>אם ישים אליו לבו .</b> על אדם להשחיתו למה יבא בעלילה עליו הרי מיד יאסוף אליו רוחו ונשמתו רוח ונשמה שלו שהיא ( נפוחה בו ) ואין מוחה בידו ויגוע כל בשר יחד : 
! {טז}
<b>ואם ביגה .</b> ואם אתה רוצה הבין שמעה זאת , בינה זו אינה שם דבר אלא ל' צווי כמו בינה הגיגי ( תהלים ה ) לפיכך שתים אלו טעמן למעלה בבי''ת וכל שאר בינה טעמן למטה בנו''ן : 
! {יז}
<b>האף שונא משפט .</b> משפטו של שופ' צדק : <b>יחבוש .</b> המקום וא''ת אינו משפט כי לא כדין עשה אפשר זאת מי שהוא צדיק מאוד תרשיע : <b>כביר .</b> ל' מאוד : 
! {יח}
<b>האמור למלך בליעל .</b> וכי הגון וראוי לאמר למלך בליעל ולאמר רשע אל נדיבים : 
! {יט}
<b>אשר לא נשא פני שרים .</b> כלומר למלך עולם אשר אין משוא פנים לפניו : <b>שוע .</b> שר כמו ולכילי לא יאמר שוע ( ישעיה לב ) : 
! {כ}
<b>רגע ימותו .</b> כשירצה : <b>וחצות לילה .</b> כמו שעשה למצרים : <b>יגועשו עם .</b> כלומר ברגע יעבורו : <b>ויסירו אביר .</b> כח שלהם : <b>לא ביד .</b> לא ישנו בידם ואין להם כח : 
! {כג}
<b>כי לא על איש ישים עוד .</b> כלומר אמרת לא אך ישים בי ( לעיל כג ) רק חטאי לבדי ואני משיבך לא ישים הקב''ה על בריותיו עלילות תוספת על פשעם , עוד ל' תוספת : <b>להלוך אל אל במשפט .</b> כשהוא בא על בריותיו להתווכח עמהם : 
! {כד}
<b>ירוע .</b> ירוצץ כמו רוע התרועעה ( ישעיה כד ) : <b>ויעמד אחרים .</b> במקומם : 
! {כה}
<b>לכן יכיר מעבדיהם .</b> לפי שהוא מכיר מעבדיהם הטוב והרע ואין צורך לו להתווכח : <b>והפך לילה .</b> עת חשך וצרה להם וידכאו : 
! {כו}
<b>תחת רשעים .</b> במקומם : <b>ספקם .</b> לאנשי סדום : <b>במקום רואים .</b> לעין כל : 
! {כז}
<b>אשר על כן סרו מאחריו .</b> כמו ( מדבר י ) על כן ידעת חנותינו במדבר כמו מפני אשר סרו מאחריו והביאו לפניו ועליו צעקת דלים : 
! {כט}
<b>והוא ישקיט .</b> משקיט את הדלים מאותן העושקים ומי ירשיע עוד ובעוד שהסתיר פנים מי ישורנו : <b>ועל גוי ועל אדם יחד .</b> כיחידים כמרובים שוה לפניו להשקיט ולהסתיר פנים : 
! {ל}
<b>( וישקוט ) ממלוך אדם חנף ממוקשי עם .</b> על דלים שמלך : <b>ממוקשי עם .</b> עון , ולאלוה כזה לא היה לך לומר בליעל ורשע : 
! {לא}
<b>כי אל אל האמר .</b> להאמר אליו נכון וראוי מאת כל סובלי יסורים להאמר נשאתי ומקבל וסובל אני משפטיך ולא אחבול בעצמי ועוד צריך הנידן לומר : 
! {לב}
<b>בלעדי אחזה .</b> לבד מה שאני יודע לראות ולהבין בדבריך אתה הורני ואם און פעלתי לא אוסיף : 
! {לג}
<b>המעמך ישלמנה כי מאסת וגו' .</b> כך אמר אליהו לאיוב המעמך היה לו להקב''ה להימלך בפירעון תשלומין ממך : <b>כי מאסת .</b> לאמר מאסתי לא לעולם אחיה ( לעיל ג ) יתר ידו ויבצעני ( לעיל ו ) סבור אתה שיגבה ממך לפי דעתך ורצונך ואתה תבחר התשלומין ולא אני במקום הקב''ה דבר אליהו בתמיה : 
! {לד}
<b>אנשי לבב יאמרו לי .</b> כי איוב לא בדעת ידבר : 
! {לו}
<b>אבי יבחן .</b> אבי אתה הקב''ה אב לכל הלואי יבחן איוב עד נצח למען תהיה תשובה לאנשי און ואף לבני אדם דבר הכתוב כן כמו אבי דבר גדול וגו' ( מלכים ב ה ) ( ד''א אבי כמו ותפר האביונה ( קהלת יב ) ל' רצון כך חברו מנחם וכן פתרונו אבי יבחן איוב עד נצח רצוני ומאויי שיבחן איוב עד נצח סא''א ) : 
! {לז}
<b>יספוק .</b> ל' רוב דברים והרמת קול : <b>בינינו יספוק .</b> הוא אומר מה לנו ולדבריו אף אמנם שגיתי אתי תלין ( לעיל יט ) ואינו יודע שאנו נענשים ומספיק לנו עונות על שאנו שומעים כך מפורש בויקרא רבה : 

~ איוב פרק-לה

! {א}
<b>( מענה אליהוא ) : </b>
! {ב}
הזאת חשבת למשפט .</b> שהוא משפט הבריות כנגד יוצרם אשר אמרת צדקי משל הבורא : 
! {ג}
<b>כי תאמר מה יסכן לך .</b> דרכי יושר : <b>מה אועיל .</b> בצדקתי יותר מחטאתי : 
! {ד}
<b>ואת רעיך עמך .</b> ששתקו על דבריך : 
! {ה}
<b>הבט שמים .</b> ואחרי שהוא גבוה ואתה נמוך ואין לו תועלת ברשעך וצדקך למה תתפאר אליו בצדקך : 
! {ח}
<b>לאיש .</b> אשר כמוך . יוכל ויועיל רשעך וצדקתך , ראה רשעים רבים אשר : 
! {ט}
<b>מרוב עשוקים .</b> שהם עושקים יזעיקו את הבריות לפניו וישועו עניים מזרוע רבים העושקים אותם : 
! {י}
<b>ולא אמר .</b> הרשע : <b>איה אלוה עושי .</b> לירא מפניו : <b>נותן זמירות .</b> כמו כרמך לא תזמור ( ויקרא כה ) שמכרית את הרשעים בלילה כגון אמרפל וחביריו ומצרים וסנחרב : 
! {יא}
<b>מלפנו .</b> מלמדנו חכמה יותר מבהמו' כלו' חשבנו וגדלנו מבהמו' ומעוף : 
! {יב}
<b>שם יצעקו .</b> וראה אשר שם צועקים העניים מפני גאון העושקים אותם והוא אינו עונה : 
! {יג}
<b>אך שוא לא ישמע .</b> מיד : <b>ושדי לא ישורנה .</b> להנקם מיד אלא מאריך אפו : 
! {יד}
<b>אף כי תאמר .</b> כ''ש אתה שאין צעקתך באה כי אם על ידו ואתה תאמר ותזעק על אשר לא תשורנו להתווכח עמו כלומר אותם הקובלים וצועקים אינו ממהר לענות' ואתה תצעק שלא תשורנו : <b>דין לפניו .</b> ( דברך סס''א ) בכל מקום שהוא : <b>ותחולל .</b> וקוה לתנחומין : 
! {טו}
<b>ועתה .</b> דע לך כי אין היא פקידת ( אפו ) זאת עליך אינה כלום לפי רוב עוונותך : <b>ולא ידע בפש מאד .</b> ברוב חטאך התנהג הבור' כאילו לא ידעם : <b>בפש .</b> ל' רוב כמו ופשתם ( מלאכי ג ) וכמו כי תפושו וגו' ( ירמיה ג ) כי פשה הנגע ( ויקרא יג ) : 
! {טז}
<b>יכביר .</b> ל' מאד : 

~ איוב פרק-לו

! {א}
<b>ויוסף אליהוא .</b> ראיתי בטעמי רבי משה הדרשן שלפיכך אמר ג' מענות כנגד ג' ריעי איוב וזה רביעי לפיכך נקרא תוספת : 
! {ב}
<b>כתר לי זעיר ואחוך .</b> כולו ל' ארמי הוא הוחל לי מעט ואגידך , כתר ל' הוחל כאשר תירגם יונתן ולתורתו איים ייחלו ( ישעיה מב ) יכתרון : <b>לאלוה .</b> לבאים במקומו : 
! {ד}
<b>תמים דעות עמך .</b> הנה אתחיל עתה לדבר עמך שאתה סבור שדעתך שלימה : 
! {ה}
<b>הן אל כביר .</b> בחכמה וברחמים : <b>ולא ימאס .</b> עני : <b>כביר כח לב .</b> הוא להתנקם לפיכך לא יחיה רשע : 
! {ז}
<b>לא יגרע מצדיק עיניו .</b> עד אשר עם מלכים יושיבם : 
! {ח}
<b>ואם אסורים .</b> ואם תראה צדיקים שיהו אסורים בזיקים של ייסורין וחליים , או ילכדו בחבלי אוני , בל' גורל ומיתרים נקרא חבלי ובל' דאגה נקרא חבלי יולדה . בחבלי עוני . קודר''ש בלע''ז אינו אלא שחטאו לפניו ובא ליפרע מהם לטובתם לנקותם ולהזהירם לשוב אליו : 
! {ט}
<b>ויגד להם פעלם .</b> בייסורין הללו מודיעם שחטאו לפניו וכל דברי אליהוא תנחומין שלימים היו ולא קינתורין כלומר אל תדאג על יסורין אם צדקת כי לטובה הן לך : 
! {י}
<b>ויאמר .</b> כי בייסורין ישובון מאון : 
! {יא}
<b>אם ישמעו .</b> ויעבדוהו : 
! {יג}
<b>וחנפי לב .</b> שהם רשעים . הם ישימו אף . בבא ייסורין עליהם מחרפים ומגדפים : <b>ולא ישועו .</b> לפניו בתחינה : <b>כי אסרם .</b> כשהוא אוסרן בייסורין לפיכך לא ירפאם : 
! {יד}
<b>תמות בנוער .</b> ובשינוק שעליהם נפשם : <b>יחלץ עני .</b> את העני השם עצמו רש בדבר תחנונים : 
! {טו}
<b>יחלץ .</b> מגיהנם בשביל עניו שהביא עליו : <b>ויגל בלחץ אזנם .</b> בלחץ שהביא עליו מגלה את אזנו לאמר שוב אלי : 
! {טז}
<b>ואף .</b> זאת עוד גדולה עשה לך ביסורין הללו הסיתך והדיחך מגיהנם שפיה צר ותחתיה רחב : <b>לא מוצק .</b> כדי שיהא עשנה צבור לתוכה ולא עוד אלא שתזכה לעולם הבא ונחת שלחנך מלא דשן : 
! {יז}
<b>ודין רשע מלאת .</b> ואם נתמלאת ביסורי הרשעים הדין והמשפט הם הייסורין הם יתמכו לך לעתיד מגיהנם ויש לך לקבלם באהבה : 
! {יח}
<b>כי חמה .</b> גדולה תהיה : <b>פן יסיתך .</b> יצרך לדבר דברים : <b>בשפק .</b> בשפק זה אשר תשפק ותרב אמריך לאל ומאחר שתעלה החמה רב ממון אל יטך ממנה : 
! {יט}
<b>היערוך שועך לא בצר .</b> וכי באתה לערוך צעקה שלא תבואך שום צרה ושום מאמצי כח לאונסך אם כן במה יגבה את חובו : 
! {כ}
<b>אל תשאף הלילה .</b> כלומר ואין לך לבחור בייסורים אחרים אל תשאף ותתאוה הלילה שבא לאמרפל למצרים ולסנחריב לסלק עמים מן העולם כשהם תחתם במקומם בשלוה כי הם אבדו בעולם הזה ובעולם הבא : <b>לעלות .</b> כמו אל תעלני בחצי ימי ( תהלים קב ) כעלות גדיש בעתו : 
! {כא}
<b>השמר אל תפן אל און .</b> לדון ולומר היה לו לשפטני ביסורי עניות : כי על זה בחרת מעני כי על הייסורין הללו יש לבחור יותר מעניות אדם שנהגו בו כבוד ולמוד בעושר כל ימיו ויעמוד על המקח ואין לו דמים ובוש ושב לביתו בפחי נפש וא''ת איוב עני היה עם הייסורין לא מצינו שאיבד אלא בהמותיו אבל לא כסף וזהב ואפילו איבד בהמותיו עם כסף וזהב טוב לו להיות עני וחולה מהיות תמים והולך בשווקים ואין בידו כלום וכן מפורש במדרש רבי תנחומא מן המקרא הזה כי נוחין היו יסורי איוב מיסורי עניות וכן באגדות הגמרא : 
! {כב}
<b>הן אל ישגיב בכחו .</b> מתוך שכחו גדול ואין להנצל מפניו . לפיכך אין כמוהו מורה . כשמורה לחוטא לשוב אליו מתרה קודם מכה לפי שהוא יודע שאין יכול להשמר ממכתו כמו שהיה מתרה בפרעה על כל מכה ומכה הנני ממטיר כעת מחר ברד ( שמות ט ) והנני מביא מחר ארבה וכן כולם ובשר ודם כשרוצה להנקם מאויביו בא עליו פתאום שאם יתרה בו יחזור וינצל ממנו : 
! {כג}
<b>מי פקד עליו דרכו .</b> יצוה עליו מעולם את הדרך הזה תלך : 
! {כד}
<b>זכור .</b> דרכיו ובהזכירך תשגיא מאליך שבח פועל מדותיו : <b>אשר שררו אנשים .</b> אשר ראו אנשים כמו ( במדבר כד ) אשורנו ולא קרוב : 
! {כה}
<b>חזו בו .</b> בפעלו : <b>מרחוק .</b> מיצירת עולם עד עכשיו : 
! {כו}
<b>מספר שניו ולא חקר .</b> לפיכך צריך לאנוש להביט מרחוק לשאול מה שאירע בראשונים וילמד מהם : 
! {כז}
<b>כי יגרע .</b> כי ירבה כמו ותגרע שיחה ( לעיל טו ) ויש פותרים יגרעם מן השמים להורידם ארץ : <b>ויזקו .</b> הרקיעים את המטר : <b>לאידו .</b> לעננו : 
! {כט}
<b>אף אם יבין .</b> או אם יבין אדם מה יש במפרשי עב , בתמיה : <b>תשואות .</b> לשון ערפל סוכתו : 
! {ל}
<b>הן פרש .</b> ( על העב את אורו ) ושרשי הים כסה בו : 
! {לא}
<b>כי בם .</b> בעבים ידין עמים ייסרם בעצירת גשמי' ברעב וכל מכה , חטאו דור המבול וארובות השמים נפתחו ( בראשית ז ) אנשי סדום גפרית ומלח שריפה כל ארצה ( דברים כט ) סיסרא מן השמים נלחמו עם סיסרא ( שופטים ה ) : <b>יתן אוכל למכביר .</b> משם באים מזונות הנני ממטיר לכם לחם מן השמים ( שמות טו ) משל לנחתום העומד לפני הכבשן נכנס שונאו חות' גחלים וזורק בו נכנ' אוהבו מוציא פת חמה ונותן לו , למכביר למי שטפליו מרובים וכבירים וצריך למזונות : 
! {לב}
<b>על כפים .</b> על העננים וכן נשא לבבינו אל כפים ( איכה ג ) על שם עב קטנה ככף איש ( מלכים א יח ) ורבותינו אמרו על ( חמס ) ועון כפי' כסה אור , המטר נעצר : <b>ויצו עליה .</b> לבא ע''י מפגיע בתפלה אוהבו של דור וגדול וחכם שבו יגיד עליו לפני הקב''ה צרת הדור וצרכיו : 
! {לג}
<b>יגיד עליו רעו .</b> ידרוש להם לבדוק להם במעשיה' ולשוב אליו : <b>מקנה .</b> וזהו קנין שנקנין בו הגשמים : <b>אף על עולה .</b> גם למעלה למעלה ( עולה כך ) כמו ישובו אל על ( הושע ז ) או הקם אל ( שמואל ב כג ) ורבותינו פירשוהו קניין אף וחמה על המגיס דעתו אבל הניקוד שנקד מקנה פתח מוכיח שהוא אינו דבוק לאף שאילו הי' דבוק הי' נקוד קמץ ולא הי' בו פיסוק הטעם שהוא נטעם ברביעי : 

~ איוב פרק-לז

! {א}
<b>ויתר .</b> ויקפוץ כמו לנתר בהם על הארץ ( ויקרא יא ) וזו היא שלבו תוהה : 
! {ב}
<b>שמעו שמוע ברוגז קולו .</b> הרעמים שהוא מרעי' ברקיע כהגה היוצא מפיו : 
! {ג}
<b>תחת כל השמים ישרהו .</b> ראיתיו כמו אשורנו ( במדבר כד ) והם אותם אורות שהברקים מאירים : 
! {ד}
<b>אחריו .</b> של אותו אור , ישאג הרעם ומרעים בקול גאונו , והם ברקים ורעמים המביאים את המטר : <b>ולא יעקבם .</b> המקום : <b>כי ישמע .</b> לפניו קולו של מפגיע : <b>יעקבם .</b> יכריתם , ולשון משנה שנינו ( סוטה טז ע''א ) בשלש מקומות הלכה עוקבת מקרה : 
! {ו}
<b>כי לשלג יאמר .</b> ע''י הקולות הללו : <b>הוא ארץ וגשם מטר וגשם מטרות עזו .</b> כלומר מד' רוחות יבא : 
! {ז}
<b>ביד כל אדם יחתום .</b> כשאדם חוטא לפניו האדם עצמו חותם כתב ידו ביום מותו על עבירות שעבר הכתובות לפניו : <b>לדעת כל אנשי מעשהו .</b> להודיע על מה הוא נידון : 
! {ח}
<b>ותבוא חיה במו ארב .</b> כלומר ועוד שלוחים הרבה יש למקום ליפרע פורענותיו שילוח חיות רעות וסופה וקרה : 
! {ט}
<b>וממזרים .</b> אוצר יש ושמו מזרים : 
! {י}
<b>מנשמת .</b> מרוחו : <b>ורוחב מים במוצק .</b> ביציקה לפורענות : 
! {יא}
<b>אף ברי .</b> שם מלאך הממונה על העננים והוא יפיץ ענן מטרו של המקום : <b>ענן אורו .</b> ענן מטרו : 
! {יב}
<b>והוא .</b> הממונה : <b>מסיבות מתהפך בתחבולותיו .</b> של מקום : <b>לפעלם .</b> של גשמים לפועל המוטל עליהם לעשות אם טוב ואם רע מתהפך במסיבות הרבה כאותו ששנינו הרי שהיו ישראל צדיקים גמורים בר''ה ופסקו להם גשמים לסוף סרחו לפחות מהם אי אפשר שכבר נגזרה גזרה אלא הקב''ה מורידם שלא בזמנם ועל ארץ שאינה צריכה להם ועתים לטובה כיצד כו' : <b>כל אשר יצום .</b> לאותן גשמים לעשות לארץ . ( ד''א והוא מסיבות מתהפך בתחבולותיו . מקיף עננים ורוחות סביבות ההרים שיהיו שותים מכל רוחותיהם , באגדת והיה עקב ) : 
! {יג}
<b>אם לשבט .</b> אם לפורעניות בני אדם מורידם בהרים וגבעות שאין מקום זריעה : <b>אם לארצו .</b> להורידם כאשר נגזר בתחילת דין לא לשבט ולא לפנים מן השורה אלא בבינונית : <b>אם לחסד .</b> שנעשו הדור עכשיו יותר צדיקים משהיו בראש השנ' ואותן גשמים שנגזרו בבינונית מתהפך בתחבולות מסיבותיהם לירד בחסד בזמנם ועל הארץ הצריכה להם שלא תרד הטפה מהם לאיבוד : <b>ימציאהו .</b> למטר לאחת מן המדות הללו : 
! {טו}
<b>בשום אלוה .</b> חק ברית עליהם היאך תלוים במאמר : 
! {טז}
<b>מפלשי עב .</b> גילוי עב ל' מבוי מפולש כלומר לארכן ולרחבן של עבים : <b>התדע .</b> עליהם מפלאות תמים דעים : 
! {יז}
<b>אשר בגדיך חמים .</b> במה בגדיך חמים בהשקיטי ארץ מרוח דרומית שאינה הורגת את העולם בקרה שרוח מזרחית מחממת שנאמר רוח קדים חרישי' ( יונה ד ) שמשתקת כל הרוחות מפניה : 
! {יח}
<b>חזקים .</b> הרקיעים : <b>כראי מוצק .</b> כראי שהנשים רואות בו , ( מוצק טרישטייר והוא חזק ביותר ) : 
! {יט}
<b>הודיענו מה נאמר לו .</b> להקב''ה בריבנו עמו , לא נוכל לערוך אליו מפני חשך אשר סתרו וסביבותיו : 
! {כ}
<b>היסופר לו כי אדבר .</b> וכי כמדת הבריות הוא שצריך שיספרו אליו דברים היוצאים מפינו : <b>אם אמר איש .</b> אומר מפיו : <b>כי יבולע .</b> יגלה אליו מיד , כמו פן יבולע למלך ( ש''ב יז ) יגלה , ורבותינו פירשו היסופר כל שבחו לו : <b>כי אדבר .</b> כי נדבר בו ובגבורתו , אם אמר איש , לספרם כולם : <b>כי יבולע .</b> ממקומו : 
! {כא}
<b>ועתה לא ראו אור .</b> בסוף דבריו אל אליהוא חזר לקנתר אותן שלשת ריעי איוב על לא מצאו מענה ואמר ועתה שלשת ריעיך אלה שהיה להם להשיבך כל דברים אלו שאמרתי לך , לא ראו אור נעשו המה כנפלים אשר לא ראו חכמה : <b>בהיר הוא בשחקים .</b> דומה בהירים הנראים בשחקים להוריד גשם : <b>ורוח עברה .</b> ורוח בא ומטהרתן ומעבירתן ואין מטר יורד כך הוא דוגמא שלהם התחילו להשיבך וסוף עמדו ולא ענו : 
! {כב}
<b>מצפון זהב יאתה .</b> הרי הוא כחוזר על הראשונות ( ועתה לא ראו אור ) ד''א מדרש חכמים כשהרקיע מלא בהרות עננים עננים להוריד גשם רוח עברה ותטהרם ואין גשם יורד זה מדרש חכמים ואינה מתיישב' על סדר המקראות : <b>מצפון זהב יאתה .</b> מנשב רוח צפונית ומעביר הגשם והחמה זורחת כמו המריקי' מעליהם הזהב ( זכריה ד ) : <b>על אלוה .</b> מלפניו . ( מצפון זהב יאתה . נצפן הזהב הטוב לאותן שיאתיו את עצמן למצות הקב''ה ומאמינים באלוה הנורא מאד ) : 
! {כג}
<b>שדי לא מצאנוהו שגיא כח .</b> לעורר משפטיו על הבריות בגודל כחו כ''א ברחמי' וכפרת הבריות שם בדבר מועט לפי היכולת קומץ מנחה מחצית השקל תורים ובני יונה בקר וצאן ולא טרח לבק' ראם ויחמור : <b>ומשפט .</b> יסורין : <b>לא יענה .</b> הבריות יותר מדאי וכן לא יענה הצדיק יותר מדאי במה שא''א כי אם איש כמתנת ידו : 
! {כד}
<b>לכן יראוהו אנשים .</b> מלהתווכח אליו . ( כי ) לא יראה כל חכמי לב . המתחכמים אליו כי אין חכמתם בעיניו כלום ורבותינו פירשוהו על איוב שאמר לו אליהוא לפי שהיית מטיל אימה יתירה על דורך חנם לפיכך לא זכית להתקיים לך בנים חכמי לב : 

~ איוב פרק-לח

! {א}
<b>( מענה מן הסערה ) :  מן הסערה .</b> מנה שערות שלך ואשיבך דברים כנגדן , מן הסערה שחרפתני אשר בסערה ישופני ( לעיל ט ) ממנה אשיב לך כאשר תשמע בסוף הענין מי פלג לשטף תעלה גומא לכל שער ד''א בסערה הכהו בסערה רפאהו כדמפרש בספרי : 
! {ב}
<b>מחשיך עצה .</b> עצה חשיכה וסכלה , ד''א מי זה מחשיך עצה בלי דעת , אמר הקב''ה מי זה הוא מחשיך עצה מקלקל במילים וברוב דברים שכתבתי איש תם וישר בתחילת הספר כדי לחול שמי עליו והוא בא והחשיך וקילקל מה שיעצתי ברוב דבריו ואיוב ענהו מי זה מעלים עצה ( לקמן מב ) אילו הייתי יודע עצתך לא הייתי מרבה בדברים והקב''ה השיבו והלא אברהם לא היה יודע ועמד בעשר כך מצאתי : 
! {ג}
<b>אזר נא כגבר חלציך .</b> רמז לו שיתעורר מן חלייו ויסוריו : 
! {ד}
<b>איפה .</b> אי פה : 
! {ה}
<b>ממדיה .</b> מדותיה : 
! {ו}
<b>ירה אבן פנתה .</b> באמצע הים ומשם נשתת כל העולם : 
! {ז}
<b>ברן יחד .</b> מתחלה כוכבי אור : 
! {ח}
<b>בדלתים ים .</b> חול גבול לו : <b>בגיחו .</b> בהמשכתו יוצא מן התהום כמו גוחי מבטן מושכי : 
! {ט}
<b>בשומי ענן .</b> סביבות ים אוקיינוס שהוא היקף העולם ועננים סובבים לו כלבוש : <b>חתולתו .</b> תכריכו כמו והחתל לא חתלת ( יחזקאל טז ) : 
! {י}
<b>ואשבור .</b> עשיתי לו שברים סביב לעכבו בתוכם והם לו חוק אשר לא יעברם כמו וירדפם לפני השער עד השברים ( יהושע ז ) והם תועלות סביב לעיר : 
! {יב}
<b>המימיך .</b> מיום שנולדת : 
! {יג}
<b>לאחוז בכנפות הארץ .</b> ועוד המימיך צוית לאחוז בכנפי הארץ כאדם שאוחז בכנפי הטלי' ומנערה כן אני עתיד לאחוז בכנפותיה ולנער את הרשעים וכן הוא אומר וה' יטה את ידו וכשל עוזר ( ישעיה לא ) כאדם האוחז בידו דבר ומטה ידו והוא נופל : 
! {יד}
<b>תתהפך .</b> בציורין : <b>כחומר חותם .</b> מטבע צורת האדם במותן ובתחיית המתים יחיו ויעמדו בלבושיהם : 
! {טו}
<b>וימנע .</b> אז מרשעים אורם וכן הוא אומר וזרוע רמה תשבר : 
! {טז}
<b>נבכי ים .</b> מסגרי ים כמו נבוכים הם בארץ ( שמות יד ) : 
! {יז}
<b>שערי מות .</b> זה גיהנם : 
! {כא}
<b>ידעת .</b> כל זאת : <b>כי אז תוולד .</b> כשבראתים : 
! {כג}
<b>אשר חשכתי .</b> המנעתי : <b>לעת צר .</b> לחמשת מלכים בגבעון : <b>ליום קרב .</b> מלחמת גוג ומגוג : 
! {כד}
<b>יחלק אור .</b> תימור של חמה מפציע לכאן ולכאן כקרני איל : <b>קדים .</b> שמש המזרח : 
! {כה}
<b>מי פלג לשטף תעלה .</b> בערביא קורין לשערה שיטפא כל שיער ושיער שבראש פלגתי לו תעלה גומא לינק ממנה שאילו שנים יונקות מגומא אחת מחשיכות מאור עיניו של אדם והם לא נתחלפו לי בין איוב לאויב נתחלף לי וזהו מן הסערה ד''א לשטף טיפת גשם שוטף פילגתי לכל טיפה גומא בעבים שאילו יורדות ב' טפות מגומא אחת מטשטשות את הארץ ונעשי' טיט : <b>ודרך לחזיז קולות .</b> מי פלג כל קול וקול יש לו שביל ואם שנים הולכים בשביל אחד אין הבריות סובלות את הקול : 
! {כז}
<b>שואה .</b> ארץ שהיא חשיכה ברעב : 
! {כח}
<b>אגלי טל .</b> גלי טל והאל''ף יתירה כמו ואחותי באזניכם ( לעיל יג ) אל''ף יתירה בראש התיבה להיותו שם דבר וכן אבחת חרב ( יחזקאל כח ) כמו ניבוח חרב צעקת חרב , ל''א אגלי כמו אגני טל כמין אגנות פלגים שהטל נתון בהם למ''ד מתחלף בנו''ן וכן בספר נחמיה נשכה כמו לשכה ( נחמיה יג ) : 
! {כט}
<b>כפור .</b> גריס''א : 
! {ל}
<b>כאבן מים יתחבאו .</b> ע''י הקרח שמגליד טיפה שעל פניהם : <b>ופני תהום יתלכדו .</b> בו ויתאחזו כאחד : 
! {לא}
<b>מעדנות כימה .</b> קישור כימ' שלא תצא כל קורותיו ויחריב את העולם בצינה : <b>או מושכות כסיל תפתח .</b> להוציא חמה להפיג צינתה של כימה : 
! {לב}
<b>מזרות .</b> כל שאר מזלות : <b>ועיש .</b> כוכב גדול שבכימה והרבה כוכבים קבועים בו ונטל שני כוכבים ממנה לפתוח ארובות המבול ונבקעו במזל טלה ועתיד הקב''ה להחזירם לה תנחומא : 
! {לג}
<b>משטרו .</b> שטירת המזלות הממונים להביא על הארץ קור וחום קיץ וחורף : 
! {לד}
<b>התרים לעב קולך .</b> לצוות עליו להתקשר בחשכת מים להיות שפעת מים מתכסך והוא קדירת העבים כמו או חשך לא תראה ( לעיל כב ) . או שפעת מים . על פיך יכסך מן השמים : 
! {לה}
<b>ויאמרו לך .</b> במקום שנשתלחו שם הננו עשינו שליחותך שאין צריכין לחזור למקומן ולהשיב שלוחן שהשכינ' בכל מקום : 
! {לו}
<b>בטוחות .</b> אלו הכליות : <b>לשכוי .</b> זה תרנגול לשון חכמים וי''א זה הלב שהוא שוכה וצופה ומחשב את הנולדות : 
! {לז}
<b>יספר .</b> ידבר ויזהיר תפקיד' : <b>ונבלי .</b> עשויין כנודות והגשמים כנוסים בו שם , כמו נבל יין : 
! {לח}
<b>בצקת עפר .</b> ביום שיצקתי עפר : <b>למוצק .</b> ליסוד עולם באמצעיתו : <b>ורגבים ידובקו .</b> סביבותיו לצדדין עד כי נתמלא ארכו ורחבו : 
! {מ}
<b>ישוחו .</b> עומדי' צפופי' : <b>למו ארב .</b> לארוב כן דרך האורבי' לשוח ולכוף עצמן כדי שלא יכירו בהם : 
! {מא}
<b>יתעו לבלי אכל .</b> שאביהם שונאם הקב''ה מזמן להם יתושין נבראים מצואתן ונכנסין לתוך פיהם : 

~ איוב פרק-לט

! {א}
<b>יעלי סלע .</b> אשטנבו''ק ושונאה את ולדה ועולה לראש סלע גבוה כשכורעת לילד כדי שיפול לארץ וימות והקב''ה מזמן לו נשר ומקבלו בכנפיו : <b>חולל אילות תשמור .</b> לשון חיל יולדה ( ירמיהו ) האילה רחמה צר ואין הוולד יכול לצאת ובשעת לידתה אני מזמן לה דרקון ומכישה ברחמה ונפתח ואם מקדים ומאחר רגע מיד מתה , ובין אותם רגעים איני מחליף ובין איוב לאויב נתחלף לי : 
! {ג}
<b>תפלחנה .</b> מתבקע רחם שלהם ומוציא הולדות וחבלי לידה שלהם : <b>תשלחנה .</b> מעליהם בלדתן לרגע שאני מזמין להם : 
! {ד}
<b>יחלמו .</b> יחזקו : <b>ולא שבו למו .</b> משירבו קצת יגדלו בבר בזרעים ובדשאים ואינם צריכים לגידול אביהם ואמם : 
! {ה}
<b>חפשי .</b> שאין אדם יכול לגדל פרא ללמדו עבודת בהמות : <b>ערוד .</b> חמור הבר : <b>מי פתח .</b> מוסרותיו , שאין עליו עול אדם : 
! {ח}
<b>יתור הרים מרעהו .</b> הוא עצמו תר לו מקום מרעה : 
! {ט}
<b>רים .</b> כמו ראם : <b>עבדך .</b> לעבדך : 
! {י}
<b>בתלם עבותו .</b> ( או קשרת ראם בעבור תלם של שורך ) עבות חבל שקושרין ( שוורי' ) לתלמי המחריש' : <b>ישדד .</b> יתקן השדה : 
! {יא}
<b>התבטח בו .</b> לאסוף תבואתך בשביל שכחו רב ויכול לסבול הרבה : <b>ותעזוב אליו יגיעך .</b> לאסוף אל הבית : 
! {יג}
<b>כנף רננים נעלסה .</b> כן שם עוף גדול רננים ולשון משנה קרוי בר יוכני : <b>אברה .</b> כל עוף קרוי אברה על שם שפורח כלומר או עוף ששמו חסידה ועוף ששמו נוצה ראה מה מנהגה : 
! {יד}
<b>תעזוב לארץ ביציה .</b> והולכת ומתחממת למרחוק על עפר : 
! {טו}
<b>ותשכח כי רגל תזוריה .</b> לביצתה תבקענה : 
! {טז}
<b>הקשיח .</b> הקשה בניה מלבה כאילו אינם שלה : <b>לריק יגיעה .</b> ואינה יראה שתאבדם : 
! {יח}
<b>כעת במרום תמריא .</b> כל לשון המראה כאדם שזונה לבו לצאת ממידתו ומגידולו ומארצו לשוטט לו בארצות ולנסות לו מדות אחרות וכן בן סורר ומורה ( דברים כא ) ובל' גמרא אימראי כלתיה דרב זביד אימראי ונפקא ( כתובו' ס''ג ושם איתא אימרד' ועיין ברש''י בחולין נח ומציעא עו ד''ה ואימרו ) : 
! {יט}
<b>רעמה .</b> לשון רעם ובעתה כמו רעמו פנים : 
! {כ}
<b>כארבה .</b> מדלג ומקפץ כארבה ומרעיש סביביו : <b>נחרו .</b> כשהוא נופח בנחיריו בקול אימה הוא : 
! {כא}
<b>יחפרו בעמק .</b> כמו ויחפרו לנו את הארץ ( שם א ) שדרך הפרשים אורבי' בעמקי' ובנחלים : <b>וישיש בכח .</b> והסוס שמח ויוצא לקרא' נשק : 
! {כב}
<b>ולא יחת .</b> ולא יירא : 
! {כג}
<b>תרנה אשפה .</b> מלאה חצים ומקשקשות זו בזו ונשמע קולם : <b>להב חנית .</b> כלי של ברזל ( וחנית של ברזל ) קרוי להב כמו ויבא גם הנצב אחר הלהב ( שופטים ד ) : 
! {כד}
<b>יגמא ארץ .</b> עושה גומות ברגליו , ע''א יגמא כמו הגמיאני נא ( בראשית כד ) : <b>ולא יאמין .</b> מרוב שמחה שהוא מתאווה למלחמה : <b>כי קול שופר .</b> של מלחמה : 
! {כה}
<b>בדי שופר .</b> ל' די השיב לו בהרבה שופר : <b>יאמר האח .</b> ל' שאומרים ( העוברים ) כשהן שמחין כמו שאומרים אהה מחמת צער : 
! {כו}
<b>יאבר נץ .</b> יגדל הנץ כנף ומלאך הוא דומה לדמות נץ ומעמיד קושי הרוח דרומית בפרישות כנפיו שלא תחריב את העולם : 
! {כז}
<b>אם על פיך יגביה נשר .</b> השכינה שנאמ' כנשר יעיר קינו ( דברים לב ) היה מסלק פמליא שלו שלא יזיקו לאהרן בבואו אל הקדש ביו''ה : 
! {כח}
<b>סלע ישכן ויתלונן .</b> בטוח היה אהרן במחתת הקטורת כשנתנ' על אבן שתיה : 
! {כט}
<b>משם חפר אכל .</b> מתפלל על צרכי כל השנה ואחר כל הכבוד הזה ראה במיתת שני אפרוחיו נדב ואביהוא שמתו להתקדש שמי בהם שנפלה אימתי על השאר : 
! {ל}
<b>יעלעו .</b> ואינן אלא כפי מדת בלעו דם ומים ונראה כבולע הרבה כאומר על על וכן בלשון חכמים בהשקאת סוטה ( סוטה כ ע''א ) מערערין אותה ומשקין אותה על כרחה וכן זעקת שבר יעוערו ( ישעיה טו ) ל' קריאת גרון דומה לכך ולפי הדרך דבר הנביא . ובאשר חללים שם הוא . לאכול ורבותינו דימו כל ענין זה באהרן בפסיקת' דאחרי מות : 

~ איוב פרק-מ

! {א}
<b>( מענה ה' ) : </b>
! {ב}
הרוב עם שדי וסור .</b> העם שדי אדם ישתרר : <b>מוכיח אלוה יעננה .</b> הבא להתווכח יענה דבריו . <b>( מענה איוב ) : </b>
! {ה}
<b>אחת דברתי .</b> כלומר מעט אמרתי ולא אוסיף עוד , ויש פותרים אחת דברתי , אשר אמרתי ( לעיל ט ) תם ורשע הוא מכלה לא אענה עוד : <b>ושתים .</b> שאמרתי ( לעיל יג ) אל תעש עמדי לא אוסיף : 
! {ו}
<b>( מענה ה' מן הסערה ) : </b>
! {ח}
האף תפר משפטי .</b> האף תוכיח שדבריך להפר משפטי ולהראות שהוא מעוקל : 
! {י}
<b>עדה .</b> לשון עדי ותכשיט : 
! {יב}
<b>והדוך .</b> ל' דכו במדוכה ( במדבר טו ) : 
! {טו}
<b>בהמות .</b> מוכן לעתיד : 
! {טז}
<b>כחו במתניו .</b> ביציו מעוכים ונסתרים ולא נתקו לגמרי : <b>בשרירי .</b> כמו שררך ( שיר ז ) : 
! {יז}
<b>יחפוץ .</b> כמו יחפוז מתקשה וכן בלשון חכמים דחפיזא כי אופתא ( חולין מז ) קשה כעץ : <b>פחדיו .</b> ביציו : <b>ישורגו .</b> גדילים כמו וישתרגו ( איכה א ) או שריגי מטה בל' משנה והוא ל' גדיל : 
! {יח}
<b>אפיקי .</b> חוזק : <b>גרמיו .</b> עצמיו : <b>כמטיל .</b> כמשאוי של ברזל : 
! {יט}
<b>ראשית דרכי אל .</b> מבראשית נברא ראש לבהמות : <b>העושו .</b> הקב''ה בעצמו ישחטנו : 
! {כ}
<b>כי בול הרים ישאו לו .</b> ובספר תהלים כתוב בהמות בהררי אלף , אלף הרים נשאו לו יבול ובכל יום רועה את כולם , ( ע''כ יסד רש''י מכאן ואילך אינו ל' רש''י ) : 
! {כא}
<b>צאלים .</b> צל אילנות : 
! {כב}
<b>יסוכהו צאלים צללו .</b> אותו צל של סתר קנה וביצה מסובך ומסוכך באילנות ( הרבה ) : 
! {כג}
<b>הן יעשוק נהר ולא יחפוז .</b> פרת שמימיו פרים ורבים והוא נהר גדול עושקו וגוזלו מן הבריות בשתותו : <b>ולא יחפוז .</b> לא מתקשה בטנו על שתייתו כמו יחפוז זנבו : <b>יבטח כי יגיח ירדן .</b> בטוח הוא למשוך כל הירדן לפיהו אילו היה שותה שם כמו גוחי מבטן ( תלים כב ) : 
! {כד}
<b>בעיניו יקחנו .</b> המקום יקחנו בהביטו בו ונרעד מאימתו : <b>במוקשים .</b> כלי אומנו' של ציידים ששולים בהם דגים : 
! {כה}
<b>תמשך לויתן וגו' .</b> כאשר תמשכנו בחכה ותשים אגמון באפו הידבר אליך תחנונים כלו' לא תוכל ( להתקרב אליו ) בכלי מצודה : <b>תשקיע לשונו .</b> תשקיע חבל בלשונו להוציאו ממקומו : 
! {כו}
<b>אגמון .</b> הוא כפוף כמו כיפה ואגמון ( ישעיה ט' ) ועשוי של ברזל לצוד בו דגים : <b>התשים .</b> אם תשים אגמון באפו להוציאו מן הים : <b>ובחוח .</b> כלי של ציידי דגים : <b>תקוב .</b> כמו תנקוב : 
! {כז}
<b>הירבה אליך תחנונים .</b> אם ירבה לך תחנונים ויבקש על נפשו שתניחהו : <b>אם ידבר .</b> כפל מלה : <b>רכות .</b> דברים של תחנונים : 
! {כח}
<b>היכרות ברית עמך .</b> אם יכרות ברית לעבדך שתקחנו להיות לך עבד עולם : 
! {כט}
<b>התשחק .</b> אם תשחק ותטייל בו כאילו הוא צפור קטן : <b>ואם תקשרנו .</b> לצוארי ( ס''א לצורך ) נעריך הקטנים שיטיילו בו כצפרים קטנים שנותנין לתינוקות לטייל ולפייסו : 
! {ל}
<b>יכרו עליו חברים .</b> אם יכרו ויחכמו עליו חברים ומכשפים לצוד אותו להוציאו מן הים ע''י מכשפות שיחצוהו ויחלקוהו בין תגרים כעניין שעושים ציידי הים שצדין דגים גדולים בחכמתם ובאים התגרים וקונים מהם : <b>יכרו .</b> ל' כורה שוחה ( ירמיה יח ) : <b>חברים .</b> ל' וחובר חבר : <b>כנענים .</b> תגרים : 
! {לא}
<b>התמלא .</b> אם תמלא תאותך ותעשה סוכות מעורו : <b>ובצלצל דגים ראשו .</b> ואם תעשה מראשו צלצל דגים כפל לשון : <b>צלצל .</b> סוכת דגים ולסוכה עשויה מראשו קורא אותה צלצל דגים לפי שעשויה ממנו : <b>צלצל .</b> אמבוריץ בלע''ז ל' צל הוא והוא מתיבות כפולות כמו ירקרק אדמדם ( ויקרא יד ) פתלתל ( דברי' לב ) קעקע ( ויקרא יט ) : 
! {לב}
<b>שים עליו כפך .</b> התגבר לערוך לפניו שתשים עליו כפיך להכותו : <b>זכור מלחמה אל תוסף .</b> ואל תוסיף זכור גבורתך להלחם בו במלחמה : <b>אל תוסף .</b> מן יסף כמו ולא יסף עוד לדעתה ( בראשית לח ) ובלע''ז רקרייטר''ש , אל תוסף סמ''ך נפתחת מפני סוף פסוק כמו אל תותר ( בראשית מט ) ותותר ( רות ב ) וכן הרבה כמוהו : 

~ איוב פרק-מא

! {א}
<b>הן תוחלתו נכזבה .</b> הן אם תוחלתו נכזבה שלא יהא בו כח להלחם ולזוז ממקומו : <b>הגם אל מראיו יוטל .</b> אם גם אל מראית פניו יוטל ויושלך אדם לעמוד לפניו להסתכל בו מפני אימתו : 
! {ב}
<b>לא אכזר כי יעורנו .</b> אין אכזר גבור ועריץ אשר יעירנו משנתו ויעוררנו ממקומו להקימו : <b>ומי הוא וגו' .</b> אשר יתיצב ויעמוד לפני אשר אני עושה את כל אלה : 
! {ג}
<b>מי הקדימני .</b> ומי הוא אשר הקדים פני בהקבלתו לעבדני ולשמור מצותי ואשלם לו שכרו : <b>תחת כל השמים לי הוא .</b> כל אשר בשמים לי הוא ובידי לפרוע לו שכרו ופעולתו ומעבדו : 
! {ד}
<b>לא .</b> אחריש בדיו . לא אחריש על בניו של צדיק שהלך לפני בתמים ולבניו ישתלמו שכר טוב אביהם וימצאו טובה בזכותו : <b>ודבר גבורות וחין ערכו .</b> ודבר גבורותיו אשר נתגבר לעשות הטוב והישר ותחינת סדורו של צדיק אשלם שכרו גם לבניו אחריו : <b>בדיו .</b> ענפיו והם בניו כמו ( יחזקאל יז ) ותעש בדים ותשלח פארות : 
! {ה}
<b>מי גלה פני לבושו .</b> חוזר על הראשונות לדבר על לויתן מי יוכל לגלות קשקשות שהן בכפל רסנו , בכפל שפתיו כשהן פתוחות מי הוא אשר יכנס שם , בכפל הן השפתים שהן כפולות אחת למעלה ואחת למטה שאין בריה רשאה ליכנס שם מפני אימתו : 
! {ו}
<b>דלתי פניו .</b> הן השפתים כשהוא סוגר פיו מי יוכל לפותחן : <b>סביבות שניו אימה .</b> אשר מפני פחדו ואימתו אין בריה יכולה להתקר' לשיניו : 
! {ז}
<b>גאוה אפיקי מגנים .</b> גאוה גדולה יש לו בחוזק מגיניו הן קשקשים שעליו המגינין עליו כמגן : <b>סגור .</b> הוא חותם שריונו שהו' צר וסגור ועצום שכל קשקש' וקשקשת דבוקה בו ואדוקה לחברתה שאין בריה יכולה להכותו בין הדבקים וזהו שנא' אחד באחד יגשו שכולן מוגשות ותכופות זו לזו : 
! {ט}
<b>יתלכדו ולא יתפרדו .</b> עדיין כופל לשונו שנלכדות ונאחדות כל אחת ואחת בחברתה שלא תהיינה מתפרדות זו מזו : 
! {י}
<b>עטישתיו תהל אור .</b> בכל עטישה ועטישה שהוא מתעטש יהל ותזרח אור : <b>תהל .</b> כמו בהלו נרו ( לעיל כט ) : <b>ועיניו כעפעפי שחר .</b> עיניו זורחות ומאירות כדמדומי חמה כמו לעת בקר שהשמש מתאדמת : 
! {יא}
<b>מפיו .</b> מלהבת היוצאת ממנו : <b>מפיו לפידים יהלוכו .</b> לפידים יוצאות מפיו : <b>כידודי אש .</b> כמו לפידים : <b>יתמלטו .</b> פתרונו לפי עניינו שוינ''ט אישקמוצי''ר בלע''ז : 
! {יב}
<b>מנחיריו יצא עשן .</b> כסיר נפוח על האש שמעלה עשן : <b>ואגמון .</b> פתרונו לפי עניינו לשון כלי חרש הוא כמו דוד נפוח כענין שנא' ( ירמיה א ) סיר נפוח אני רואה : 
! {יג}
<b>נפשו גחלים תלהט .</b> גחלים בוערות ולוהטות ולהבות דולקות יוצאות מפיו : 
! {יד}
<b>בצוארו ילין עוז .</b> גבורה : <b>ולפניו תדוץ דאבה .</b> שאין דאבון ועצבון לפניו שאינו מתירא משום בריה , תדוץ ל' דיצה ושמחה : 
! {טו}
<b>מפלי בשרו דבקו .</b> נדבקות ותכופות זו בזו שהדגים הן נעשין חתיכות חתיכות בבשרו ומדובקות זו בזו ובדג גדול כשהוא מבושל נראות חתיכות כשהן שוכבות זו על זו וזו אצל זו וכן הולך סדור המלה תחלה דבר בקשקשותיו של לויתן ואחר כן בחתיכות בשרו וחוזק לבו , מפלי ל' כפילות חתיכות וי''ל כמו פלאי פלויי שבל' גמ' ( נדה כא ) לשון ביקוע ביקועי חתיכותיו , דבקו בי''ת קמוצה בקמץ קטן כמו ולשונם לחכם דבקה וכן הרבה באתנחתא וסוף פסוק : <b>יצוק עליו .</b> כל מפל ומפל יצוק זה על זה אשר לא ימוט ולא יתפרד זה מעל זה , יצוק לשון מוצק והוא ל' עשוי כמו ירוד מן ירד יצוק מן יצק ידוע מן ידע : 
! {טז}
<b>לבו יצוק כמו אבן .</b> לבו בקרבו חזק ומיוסד כאבן הקבוע' ונתונה ביסוד הבנין בחוזק : <b>יצוק כפלח תחתית .</b> כופל מלתו שמיוסד לבו וקבוע כפלח של ריחיים שנתון בתחתיתו של פלח רכב שפלח רכב שלמעלה הוא סובב ומתגלגל וטוחן ושל מטה אינו זז ממקומו כי קבוע הוא במקומו אשר לא ימוט הוא נקרא פלח שנאמר ( שופטים ט ) ותשלך אשה אחת פלח רכב על ראש אבימלך ואת שלמטה קרוי פלח תחתית : 
! {יז}
<b>משתו יגורו אלים .</b> מפחדו מתיראים גבורים וחזקים בין דגים בין שאר בריות : <b>משברים יתחטאו .</b> גלי הים שקרוין משברים הן חסרין לפניו שהוא שוברתן בפריחתו כאשר הוא פורח וצף בגלי הים : <b>יתחטאו .</b> ל' חסרון כמו ( מ''א א ) והייתי אני ובני שלמה חטאים : 
! {יח}
<b>משיגהו חרב בלי תקום .</b> גבור המשיגהו להלחם בחרב לא יתקיים חרבו כי הוא ישברנה : <b>חנית מסע ושריה .</b> וכן כל כלי מלחמה הללו שביד המשיג לא יתקיימו ולא יעמדו לפניו של לויתן , מסע י''ל שהוא א' מכלי מלחמה כמו חרב וחנית ושריה וי''ל מסע דוגמת אבן שלימה מסע וגו' ( שם יג ) וזהו פתרונו חנית מסע שהוא גדול ואינו נוח לטלטל כל כך מפני שהוא גדול וחזק מאוד כענין שנא' בגלית הפלשתי וחץ חניתו כמנור אורגים ולהבת חניתו שש מאות וגו' : <b>ושריה .</b> שריון : 
! {יט}
<b>יחשוב לתבן ברזל .</b> כלי זיין של ברזל חשובים לפניו כתבן וכלי זיין של נחושת הם לפניו כרקבות עץ : 
! {כ}
<b>לא יבריחנו בן קשת .</b> לא יברח מפני בעל קשת וחצי' ואבני בלסתראו' הרי הן לפניו כקש : 
! {כא}
<b>תותח .</b> הוא כלי זיין ופתרונו לפי עניינו : <b>וישחק לרעש כידון .</b> לרעש מהומת בעלי מלחמות נוטלי כידון הוא משחק ומטייל שאינו מתיירא מפניהם : 
! {כב}
<b>תחתיו חדודי חרש .</b> במקום חניתו יש זהרורי שמש כי סנפירים שלו אשר תחתיו בבטנו מאירות ומזריחות כשמש : <b>חדודי .</b> כמו חרב חדה שהיא שנונה צחה ומלובנה וע''ש כך קורא לזהרורי חדודי : <b>חרש .</b> שמש כמו האומר לחרס לא יזרח וגו' ( לעיל ט ) בטרם תבא החרס' ( שופטים יד ) : <b>ירפד חרוץ עלי טיט .</b> מלה כפולה ירפד חרוץ מוסב על חדודי חרס עלי טיט מוסב על תחתיתו במקום שהוא שם על החול זה טיט ריפודו מזריח כזהב חרוץ כעניין שנא' רפידתו זהב ( שיר ג ) חרוץ מין זהב כעניין שנא' מחרוץ תבואתה ( משלי ג ) טוב פריה מחרוץ ומפז ( שם ח ) , ירפד ל' מצע ומשכב כמו רפידתו זהב ( שיר ג ) : 
! {כג}
<b>ירתיח כסיר מצולה .</b> כסיר קטן נפוח שמעל' מהרה רתיחו' תכופות ורצופות כן הוא מרתיח את המצולה שהיא גדול' ועמוק' : <b>ים ישים כמרקחה .</b> כמרקחת ריקוח בשמים שהיא מעורה ומעורבת כמו כן ישים ויערב את הים כשהוא זז ממקומו : 
! {כד}
<b>אחריו יאיר נתיב .</b> כשהוא פורח ושט בים אחריו מאיר נתיב התהום שלפי שעה שהוא פורח ואין שם מים מחמת מרוצת פריחתו : <b>יחשוב תהום לשיבה .</b> תהום חשוב לפניו בחלשות ותשות : <b>לשיבה .</b> לחלשות של זקנה ושיבה : 
! {כה}
<b>אין על עפר משלו .</b> מ''ם חטף קמץ ופתרונו ממשלתו שאין בריה מושלת עליו והוא שם דבר מושל ושלטון וע''כ המ''ם מנוקדת בחטף קמ''ץ : <b>העשו .</b> כמו העשוי ולא ידעתי לו ריע ופתרונו העשוי בלי פחד ומחיתה שאינו מתיירא מן הבריות כמו שנאמר בסוס ישחק לפחד ולא ייחת ( לעיל לט ) : 
! {כו}
<b>את כל גבוה יראה .</b> כל הגבוהים שבעולם הוא רואה תחתיו וכן הוא מנהג העול' אשר אותו העומד למעלה רואה את התחתון יותר ממה שרואה התחתון ואין לתחתון גבורה ונצחון נגד העליון ולויתן זה רואה את כל הגבורים לומר שאין להם גבורה כנגדו : <b>הוא מלך על כל בני שחץ .</b> מלה כפולה על כל גאוה יראה והוא מלך על כל בני שחץ בני גסות הרוח שבראתי בעולמי שאין להם כנגדו ממשלה ושרר' והאיך נשאך לבך לשאת ראש לפני להתוכח נגדי כי הנני עושה את כל הנפלאות האלה : 

~ איוב פרק-מב

! {א}
<b>( מענה איוב ) : </b>
! {ב}
ידעתי כי כל תוכל .</b> כי הכל תוכל לעשות כאשר ישר בעיניך : <b>ולא יבצר ממך מזמה .</b> כופל מלתו שאין מזמה ומחשבה נבצרת ונסתרת ממך אשר כל מחשבותיך תוכל לגמור ולמלאות כי בידך כח וגבורה ובידך לעשות כל הטוב בעיניך ומקרא זה דוגמת ועתה לא יבצר מהם כל אשר יזמו לעשות ( בראשית יא ) : 
! {ג}
<b>מי זה מעלים .</b> אשר העלים וכסה עצתו ונפלאותיו של הקב''ה בבלי דעת , ידעתי כי הכל בידך לעשות וע''כ הגדתי קצת מגבורותיו כאשר היו עם לבבי כאש' ידעתי : <b>ולא אבין .</b> אבל לא הבנתי כ''כ כאשר הודעתני כי כמה נפלאות נסתרות ונעלמות ממני אשר אינני יודע : 
! {ה}
<b>לשמע אוזן שמעתיך .</b> כמה וכמה פעמים שמעתי שמעך ועתה עיני ראתה את שנינתך על כן אשר זכיתי לחזות שכינתך מואס אני בחיי ואהיה מתנחם אילו הייתי שוכן בקבר לשוב לעפר ואפר אשר לוקחתי משם : 
! {ז}
<b>ובשני רעיך .</b> בלדד השוחי וצופר הנעמתי : <b>כי לא דברתם .</b> שהרי לא דברתם אלי טענה נכונה כעבדי איוב שהרי הוא לא פשע בי כי אם על אשר אמר ( לעיל ט ) תם ורשע הוא מכלה וע''י השטן המקטרג את העולם שנאמר ( שם ) אם שוט ימית פתאום וגו' ואם הוסיף לדבר מפני קושי יסורין אשר כבדו וחזקו עליו דבר אבל אתם פשעתם על אשר הרשעתם אותו לאמר ( לעיל ד ) הלא יראתך כסלתך והתחזקתם אותו בחזקת רשע ולבסוף הייתם משותקים ומנוצחים לפניו והיה לכם לנחמו כאשר עשה אליהוא ולא די לאיוב בצרתו ויסוריו כי גם הוספתם על חטאתיכם פשע להקניטו : 
! {ח}
<b>והעליתם עולה בעדכם .</b> למען שתתרצו ותתפייסו לפני ובלבד אשר עבדי איוב יתפלל עליכם אשר אסלח לחטאתיכם ולפשעיכם לא אזכר עוד : <b>כי אם פניו אשא .</b> שהרי פניו אשא לקבל תפלתו הרי אם כמו אשר וכן עד אם כלו לשתות ( בראשית כד ) אם כופר יושת עליו ( שמות כא ) וכן הרבה : <b>כי לא .</b> שהרי לא : 
! {ט}
<b>וישא ה' את פני איוב .</b> וישא כמו נשאתי פניך גם לדבר הזה ( ראשית יט ) שקבל תפלתו והסביר לו פנים : 
! {י}
<b>שב את שבות .</b> איוב בהתפללו . שב נופל לומר בענין זה והרי הוא כאילו כתיב והשיב את שבות איוב : <b>בהתפללו בעד רעהו .</b> בזכות שהתפלל על כל ריע וריע שלו : <b>למשנה .</b> על אחד שנים ( נראה דקאי גם על מ''ש למעלה שהיה לו לומר להשיב מהר''ז ) ולא דקדק המקרא במלתו : 
! {יא}
<b>לפנים .</b> כל אשר היו אוהביו ומכיריו בטרם הגיעו אליו היסורים : <b>וינודו לו .</b> לשון קונפליינט''א בלע''ז : <b>קשיטה .</b> מעה : <b>ואיש .</b> וכל אחד ואחד ( ר''ל שלא תאמר שהכונה שיש שנתנו לו קשיטה ויש שנתן לו נזם לכך כתב רש''י ואיש פירושו כל אחד ואחד מהר''ז ) : 
! {יב}
<b>מראשיתו .</b> יותר מראשית עושרו : 
! {יג}
<b>שבענה .</b> הם פעמים שבע שתי שביעיות כמו ( ויקרא יב ) שבועים שהם שתי שבועות : 
! {יד}
<b>ויקרא שם האחת ימימה .</b> ע''ש יופיתן נקראו שמותיהן : <b>ימימה .</b> צחה ומלובנה כיום : <b>קציעה .</b> ריחה נודף ערב ומבושם כקציעה של בשם : <b>קרן הפוך .</b> על שם הקרן שנותנין בו כחול ובורית שנאמר ותקרעי בפוך עיניך ( ירמיה ד ) : 
! {טו}
<b>ולא נמצא נשים יפות כבנות איוב וגו' .</b> ולא נמצא מעשה של נשים יפות כמעשה יופי של בנות איוב ומקרא זה דוגמת ויהי אנשים אשר היו טמאים ( במדבר כ ) וכן הרבה : <b>ויתן להם אביהם נחלה בתוך אחיהם .</b> מתוך חשיבותן ויפיין נתן להם נחלה עם האחים : 
! {יז}
<b>ושבע ימים .</b> מן שבע יאמר שבע בדבקות :
